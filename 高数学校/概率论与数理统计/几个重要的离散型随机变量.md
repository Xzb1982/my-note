
离散均匀分布
	数学表达式为：$$\begin{align*}
P(X=k)= \frac{1}{n},k=1,2,……,n
\end{align*}$$
	也就是：①等可能（每一个基本事件发生的概率都一样）②有限多个值（有限多个基本事件）
	实际上就是[[古典概型的计算与归纳]] 
	再例如投骰子，X=哪个点数正面向上，那么X∈[1,6],那么{X=k}={k点向上的概率}
	P(X=k)= 1/6
二项分布(binomial distribution)
	在一次试验中，事件A发生的概率为p，将这个试验==**独立重复地**==做n次
		注意：==**事件之间要独立重复**==[[事件的独立性]]
		独立就是有两个事件A,B，事件A的发生与否，和B无关
	X=n次中事件A发生的次数，记：
	$$\begin{align*}
X∈[0,n]；P(X=k)= p_{n,k}= C^k_{n}(p)^n(1-p)^{n-k},k=0,1,……,n\\
满足这样的条件，则称随机变量X服从二项分布，记作：X \sim B(n,p)
\end{align*}$$
	这实际上就是==**n重伯努利概型，或者说是伯努利定理**==
	n就是我做了多少次试验，p就是每次实验中，事件A发生的概率
	就例如：
		Ai：第i次试验中，A发生
		$$\begin{align*}
P(A_{1}A_{2}A_{3}……A_{k}\overline{A_{k+1}}……\overline{A_{n}})= C^k_{n}p^{k}(1-p)^{n-k}
\end{align*}$$
		那么根据[[事件的独立性]]中的计算，我们可以推导一下：
		$$\begin{align*}
P(A_{1}A_{2}A_{3}……A_{k}\overline{A_{k+1}}……\overline{A_{n}})= \prod^k_{i=1}A_{i}* \prod^n_{j=k+1}\overline{A_{j}}=p^k*(1-p)^{n-k}
\end{align*}$$
		这还没完，上面的式子只是其中的一种情况
		回顾一下，{X=k}= {n次试验中，事件A恰好发生k次}
		此时就产生了一个问题：*这个恰好发生，是发生在n次试验中的哪几个试验中间呢？*
		那么，我们就要从n次试验中，挑出k次来让事件A发生，那么肯定要用组合数
		同时，我们还要考虑一点，我们要将其他的可能性相加起来，才能会得到C^k_{n}p^{k}(1-p)^{n-k}
		既然要加起来，那么可能性事件之间不能相交[[全概率公式]]：![[Pasted image 20250426212449.png]]
		例如$$\begin{align*}
P(A_{1}A_{2}A_{3}……A_{k}\overline{A_{k+1}}……\overline{A_{n}})\cap P(A_{1}\bar{A_{2}}A_{3}A_{4}……\overline{A_{k}}A_{k+1}……A_{n})\\
我们以此为例，考察这两者是否有交集，答案是没有\\
因为既然B_{i} \cap B_{j}=\emptyset,那么AB_{i}\cap AB_{j}=\emptyset
\end{align*}$$
		例题：假设某产品的一级品率为20％，现在打算抽检，我从一大堆产品里面抽了20个
		那么问这20个产品里面，至少有2个一级品的概率有多大
		设X＝20个产品里面的一级品个数，我们反过来，用至多来算
		$$\begin{align*}
X=0,1,2,3,4……20;设事件A表示检验产品为1级品\\
由题意，该随机变量服从二项分布，X \sim B(20,0.2)\\
P(X\geq 2)=1-P(X=0)-P(X=1)=1-C^0_{20} (0.2)^0(0.8)^{20}-C^1_{20}(0.2)^1(0.8)^{19}
\end{align*}$$
		怎么算(0.8)的20次方，取对数即可，或者用计算器
		二项分布的补充
			零一分布/成败分布(二项分布就是很多个01分布相加)
$$\begin{align*}
若n=2,X=0,1;那么设P(X=0)=p,P(X=1)=1-p\\
P(X=1)=P(A)=p;P(X=2)=P(\bar{A})= 1-p\\
引入记号: X= I_{A} 为示性函数 Indicate Function\\
X= I_{A}= \begin{cases}
p,A发生 \\
1-p,\bar{A}发生
\end{cases}
\end{align*}
$$
				就例如盒子里面有a个红球，b个白球，只摸一次，求摸到红球的概率
				{X=1}={摸到红球}；{X=0}={摸到白球}；P(X=1)= a/a+b;P(X=0)=  b/a+b
			二项分布与二项式定理
				$$\begin{align*}
P(X=x_{k})=p_{k},k=0,1,2……n(有限多个，无穷多个后面就是……)\\
p_{k}\geq 0,证明：\sum p_{k}=1\\
\because P(X=x_{k})=C^k_{n}(p)^n(1-p)^{n-k}\\
实际上就要证明：\sum^n_{k=0}C^k_{n}(p)^n(1-p)^{n-k}=1\\
那么根据二项式定理(a+b)^n,a=p,b=1-p,该式子\sum p_{k}=1成立
\end{align*}$$
			二项分布的最值情况
				求最大值：我们求最大值的做法有两种，要么作差，要么作商，这里作商更简单
				$$\begin{align*}
\begin{cases}
\frac{b_{k}}{b_{k+1}}\geq 1 \\
\frac{b_{k}}{b_{k-1}}\geq 1
\end{cases}
\end{align*}$$
			应用
				例如关于保险，①我们可以假设你出不出事，和他出不出事，这两个事件之间相互独立；
				②在同一个保险产品中（同一产品），每个人出险赔付的概率都一样（因为不同人群的工作危险程度不一样，因此设置的保费金额也不一样）
				==**在一段时间内出险赔付的次数，因为可以算作独立重复，因之可以用二项分布**==
				例如，一对同卵双胞胎A,B，他们在不同房间里，A所在的房间里有8张牌，4红4黑，抽牌不放回
				A正在看一张牌，B要猜是A看的是红的还是黑的，B猜中了6张以上
				设X=B猜中牌的个数，这里不是二项分布而是超几何分布
				我们可以知道，猜中6次，只能有3红3黑的情况有可能猜错
				猜中7次，是不可能猜错的
				猜中8次，只有一种
泊松分布
	$$\begin{align*}
P(X=k)= e^{－\lambda} \frac{\lambda^k}{k!},其中\lambda>0,k=1,2,……(可数多个数字)\\
其中\lambda 是泊松分布的强度
\end{align*}$$
	我们先判断一下该分布符不符合概率的定义和性质：
	也就是满不满足：①非负性②可数可加性
	首先该概率肯定非负数，接下来验证可数可加性：[[五个函数的麦克劳林展开]]
	$$\begin{align*}
\sum^\infty_{k=0} e^{－\lambda} \frac{\lambda^k}{k!}=1\\
e^{-\lambda}可视为常数,e^{－\lambda} \sum^\infty_{k=0} \frac{\lambda^k}{k!}=1\\
即： \sum^\infty_{k=0} \frac{\lambda^k}{k!}= e^{\lambda}\\
\frac{\lambda^0}{1}+ \frac{\lambda}{1}+\frac{\lambda^2}{2!}+ \frac{\lambda^3}{3!}+……= e^{\lambda}
\end{align*}$$
	泊松分布是用来==**描述稀有事件的概率的**==，例如一年之内，十字路口发生事故的概率
	一个单位电话总机，五年之内能收到多少电话；例如医院血常规，将一滴血液分在一个个小方格，就数每个小方格的红血球，白血球数目，最后平均一下大概就是血常规结果（粗略的过程）
	又例如食堂阿姨的一勺，中间的韭菜炒肉丝，肉丝有10根的概率有多大
	泊松分布与二项分布
		泊松分布可以作为二项分布的近似
		$$\begin{align*}
设X \sim B(n,p),其中n很大，np很小\\
b_{n,k}= C^k_{n} (p)^n(1-p)^{n-k} \sim e^{－\lambda} \frac{\lambda^k}{k!}\\
其中 \lambda= np
\end{align*}$$
		该近似也被称为泊松逼近定理
		证明：
		$$\begin{align*}
b_{n,k}= \frac{n!}{k!(n-k)!}* p^k* (1-p)^{n-k},因为n大，np小，因而令p\to 0,n\to \infty,np= \lambda\\
\lim_{ p \to 0,n\to \infty,np= \lambda }  \frac{n!}{k!(n-k)!}* p^k* (1-p)^{n-k}根据等价无穷小，1-x\sim e^{-x}\\
原式=\lim_{ p \to 0 ,n\to \infty,np= \lambda} \frac{n!}{k!(n-k)!}* p^k* e^{-p(n-k)}\\
若k与n相比很小,则 原式=\lim_{ p \to 0,n\to \infty,np= \lambda } \frac{n!}{k!(n-k)!}* p^k* e^{-np}\\
\because \lambda=np,\therefore  原式=\lim_{ p \to 0 ,n\to \infty,np= \lambda} \frac{n!}{k!(n-k)!}* p^k* e^{-\lambda}\\
\because \frac{n!}{k!(n-k)!}= \frac{n^k}{k!},\therefore \lim_{ n\to \infty,np= \lambda,p \to 0 } \frac{(np)^k}{k!} e^{-\lambda}=  e^{－\lambda} \frac{\lambda^k}{k!}
\end{align*}$$
	条件，n趋于∞，p趋于0，np＝λ，k很小或者k很接近n
	一般来说，n>30,np<=5即可进行近似计算
	例如：100人，射击一架飞机，一个人打中飞机的概率是0.02,假定飞机中两枪及其以上被击落
	假如100个人同时射击，飞机被击落的概率是多少
		首先，100个人同时射击就是100个独立重复试验，那么np= λ＝2：
		$$\begin{align*}
设有X＝击中飞机的次数，
P(X\geq 2)= 1- P(X=1)-P(X=0)= 1- \frac{3}{e^2}
\end{align*}$$

	问需要配备多少个工人，才能使得发生故障但得不到维修的概率<1%
