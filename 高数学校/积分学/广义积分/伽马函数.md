综合运用[[第一类广义积分]]和[[第二类广义积分（瑕积分）]]，形如：$$\begin{align*}
\int^{+\infty}_{0} x^{s-1} e^{-x} dx,我们称为\Gamma函数\\
这是一个混合型的广义积分，因为s=1时，x^{s-1}是无意义的\\
因此，我们可以通过区间可加性分开出来:\int^{+\infty}_{0} x^{s-1} e^{-x} dx = \int^1_{0} x^{s-1} e^{-x} dx+ \int^{+\infty}_{1} x^{s-1} e^{-x} dx\\
结果就是（其证明不要求了解）：当 s>0时，右边两个广义积分均收敛\\
那么\int^{+\infty}_{0} x^{s-1} e^{-x} dx收敛，此时每一个s>0都有唯一一个广义积分值与之对应\\
（因为广义积分收敛得到的是一个极限值，两个极限值加起来还是一个值）\\
那么\int^{+\infty}_{0} x^{s-1} e^{-x} dx是一个在(0,+\infty)上关于s的函数，记作\Gamma(s)
\end{align*}$$
也就是说：$$ \Gamma(s) = \int^{+\infty}_{0} x^{s-1} e^{-x} dx ,s∈ (0,+\infty)$$
伽马函数和泊松分布的联系是相当密切的

伽马函数的性质[[定积分的换元积分和分部积分]]
	①递推关系式
	$$\begin{align*}
当 s>0时， \Gamma(s+1) = \int^{+\infty}_{0} x^{s} e^{-x} dx,也就是说当括号内的值>1时，一定可以递推到[0,1]这一区间\\
运用分部积分法，我们可以得到：原式=\int^{+\infty}_{0} x^s d( -e^{-x}) = -(x^se^{-x}|^{+\infty}_{0}) + \int^{+\infty}_{0} e^{-x} d(x^s) \\
\lim_{ x \to +\infty } \frac{x^s}{e^x} \left( \frac{\infty}{\infty}型 \right)如果用洛必达法则，就要分情况\\
0<s<1和s=1只需用一次洛必达,s>1时，s在两个自然数之间 ，然后利用这个性质将整个函数放大\\
得到 \leq \leq 式子之后再用洛必达，左边m次，右边m+1次，极限都为0，用夹逼定理 
\end{align*}$$
	但是，我们可以利用恒等变形的方式：$$\begin{align*}
\lim_{ x \to +\infty } \frac{x^s}{e^x} = \lim_{ x \to +\infty } \left[ \frac{x}{\left( e^{\frac{1}{s}} \right)^x} \right]^s,此时 e^{ \frac{1}{s} } >1,因为e的正数次幂>1\\
那么我们将 e^{ \frac{1}{s} }记作b,b>1,原式 = \lim_{ x \to +\infty } \left[ \frac{x}{b^x} \right]^s \\
此时是 \frac{\infty}{\infty}型,由洛必达得：\lim_{ x \to +\infty } \frac{1}{b^x\ln b} = 0\\
代入原式：0^s = 0
\end{align*}$$
	那么对于原式：$$\begin{align*}
原式=\int^{+\infty}_{0} x^s d( -e^{-x}) = -(x^se^{-x}|^{+\infty}_{0}) + \int^{+\infty}_{0} e^{-x} d(x^s) = \int^{+\infty}_{0} e^{-x} d(x^s)\\
原式 = s \int^{+\infty}_{0} x^{s-1}e^{-x} dx = s *\Gamma(s)\\
如果此时s还是>1的，那么s *\Gamma(s) = s* \Gamma((s-1)+1) = s(s-1) \Gamma(s-1)
\end{align*}$$
	特别地：当n∈N（自然数集）注意，这个结论只能在s取自然数的情况下才可以用
	$$\begin{align*}
\Gamma(n+1) = n\Gamma(n) = n(n-1)(n-2)····3*2*1* \Gamma(1)\\
那么\Gamma(1) = \int^{+\infty}_{0} x^0 e^{-x}dx = -(e^{-x}|^{+\infty}_{0}) =1\\
∴ \Gamma(n+1) = n\Gamma(n) = n(n-1)(n-2)····3*2*1* \Gamma(1) = n!\\
则：\Gamma(n+1) = n!
\end{align*}$$
