在[[定积分的引入和定义]]中，我们明确
	①区间[a,b]是有限的且为闭区间
	②f(x)在[a,b]上有界
	而在实际中，我们有一些问题需要好我们突破这两条限制求出来：$$\begin{align*}
求曲线 y = \frac{1}{x^2} ,与x=1,x轴所围成的平面图形的面积
\end{align*}$$
	![[Pasted image 20250710165745.png]]
	在求面积的时候（[[定积分的几何意义]]）,我们需要用到定积分，但是，定积分要求有界
	但是这里在x -> +∞时，是无界的
	此时，取 x=t,t>1,$$\begin{align*}
在[1,t]上平面图形的面积： \int^t_{1} \frac{1}{x^2} dx \\
我这样写：\lim_{ t \to +\infty } \int^t_{1} \frac{1}{x^2} dx 而且可记成： \int^{+\infty}_{1} \frac{1}{x^2}dx 
\end{align*}$$
	有了这个例子，我们定义：$$\begin{align*}
设 f(x)在 [a, +\infty)上连续(a为常数)，\\
那么 f(x)在[a,t]上连续(t>a)（大区间上连续，小区间一定连续）;\\
那么f(x)在[a,t]上可积，\int^t_{a} f(x)dx存在\\
那么对于 \lim_{ t \to \infty }  \int^t_{a} f(x)dx \\
则称这个极限为f(x)在[a, +\infty)上的第一类广义积分(或称无穷区间上的广义积分)\\
同时记作： \int^{+\infty}_{a} f(x)dx
\end{align*}$$
	但是这个极限存不存在，会不会影响广义积分呢？答案是不会：$$\begin{align*}
\int^{+\infty}_{a} f(x)dx = \lim_{ t \to \infty }  \int^t_{a} f(x)dx = \begin{cases}
A（常数）, 当该极限存在 \\
该极限不存在 
\end{cases}\\
当\lim_{ t \to \infty }  \int^t_{a} f(x)dx不存在时，称第一类广义积分发散\\
当\lim_{ t \to \infty }  \int^t_{a} f(x)dx = A(常数)时，称第一类广义积分收敛
\end{align*}$$
	即：$$\begin{align*}
要能写出\int_a^{+\infty}f(x)，必须先保证 f在每个有限区间 [a,t]上都可积；\\  
要收敛，还必须保证上面那条定积分对 t\to \infty 的极限存在。\\
更简单地：可积性决定了广义积分能否被定义，而极限是否存在决定了广义积分是否收敛／发散
\end{align*}$$
		注：**连续** 在这里，只是一个“帮我们一次性解决所有局部可积性问题”的**方便假设**
	计算广义积分
		计算广义积分时，如果f(x)连续，那么可以用[[牛顿-莱布尼兹公式]]：
		$$\begin{align*}
若 f(x)在[a,+\infty)连续,F'(x)= f(x)\\
\int^{+\infty}_{a} f(x)dx = \lim_{ t \to \infty }  \int^t_{a} f(x)dx = \lim_{ t \to \infty } F(x)|^t_{a} = \lim_{ t \to \infty } F(t)- F(a) = \lim_{ x \to \infty } F(x)- F(a)\\
同时记作： F(x)|^{+\infty}_{a}
\end{align*}$$
		同理，$$\begin{align*}
\int^a_{-\infty} f(x)dx = F(x)|^a_{-\infty}
\end{align*}$$
		进一步：$$\begin{align*}
\int^{+\infty}_{-\infty} f(x)dx = \int^a_{-\infty} f(x)dx + \int^{+\infty}_{a} f(x)dx(这个a是一个常数，最后会被抵消掉，因此取值不限)\\
当且仅当右边两个广义积分收敛，左边的广义积分才收敛，且其值=右边两个广义积分的值的和\\
否则，则称 \int^{+\infty}_{-\infty} f(x)dx发散
\end{align*}$$但是，之前我们在[[无穷大量]]中提到，两个无穷大量之和不一定是无穷大量,但是有限个同号无穷大相加仍然保号且为无穷大
		那么这里为什么能够保证两个广义积分无穷大（即两个广义积分都发散）加起来仍然是无穷大？
		因为$$\lim_{ n \to \infty }n=+\infty,\lim_{ n \to \infty } (-n)=-\infty,\lim_{ n \to \infty } n+(-n)=0   $$
		==广义积分的极限是未混合的，没有写到同一个极限运算符里面==
		==但是在上面这个式子中，是写在同一个极限运算符里面的==
		因此，对于广义积分，如果其中有一个广义积分发散，那么整体的广义积分一定发散
		例题：$$\begin{align*}
\int^{+\infty}_{0} \frac{1}{1+x^2} dx= \lim_{ a \to +\infty } \int^a_{0} \frac{1}{1+x^2} dx = \lim_{ a \to +\infty } \arctan x|^a_{0}= \frac{\pi}{2} \\
这个式子∵ 上限>下限，被积函数\geq 0,且该极限存在且唯一，那么通过几何意义，我们可知：\\
这个式子表示y = \frac{1}{1+x^2} ,x=0,x=+\infty ,与x轴所围成的平面图形的面积
\end{align*}$$
		![[Pasted image 20250711131127.png]]
		进一步：$$\begin{align*}
在连续的基础上，因为连续一定存在极限（连续是函数极限的特殊情况）那么有：\\
\int^{+\infty}_{-\infty} f(x)dx = \int^a_{-\infty} f(x)dx + \int^{+\infty}_{a} f(x)dx = F(x)|^a_{-\infty} + F(x)|^{+\infty}_{a}\\
原式= F(a)-\lim_{ x \to -\infty } F(x) + \lim_{ x \to +\infty } F(x) - F(a) = \lim_{ x \to +\infty } F(x) -\lim_{ x \to -\infty } F(x) \\
\end{align*}$$
			也就是说==**有了连续，那么函数在连续区间内可以写成[[牛顿-莱布尼兹公式]]的形式，而且不用担心极限的存在性的问题**==
			例题：$$\begin{align*}
\int^{+\infty}_{-\infty} \frac{1}{1+x^2}dx = \lim_{ x \to +\infty }\arctan x|^{+\infty}_{0} -\lim_{ x \to -\infty } \arctan x|^0_{-\infty} = \pi \\
其几何意义就是：y = \frac{1}{1+x^2} 与x轴所围成的平面图形的面积
\end{align*}$$
			![[Pasted image 20250711132715.png]]
			例题：$$\begin{align*}
\int^{+\infty}_{-\infty} x^3 dx = \lim_{ x \to +\infty } \frac{1}{4}x^4|^{+\infty}_{0} - \lim_{ x \to +\infty } \frac{1}{4}x^4|^{0}_{-\infty}  \\
那么这里能不能根据奇函数的性质来计算呢？\\
答案是不行，因为奇偶性是在有限区间内才能使用的\\
对于这一道题，可以发现用牛顿莱布尼兹公式带入+\infty,然后得到+\infty\\
那么这个广义积分发散
\end{align*}$$
			但实际上对于偶函数而言，可以用：$$\begin{align*}
若 f(x)为偶函数，那么\int^{+\infty}_{-\infty} f(x)dx = 2\int^{+\infty}_{0} f(x)dx
\end{align*}$$
			那么背后是什么原因导致的呢？
				事实上，哪怕它们能相互抵消，广义积分也是发散的。
				因为+∞，-∞广义积分的敛散性取决于拆分后两个广义积分的存在性
				当且仅当两个都收敛，总体的广义积分才能收敛
				==**特别是两个广义积分的极限是分开的，不是像极限的四则运算那样合起来一起判断，所以导致一个广义积分发散，整体的都发散**==
		例题：$$\begin{align*}
\int^{+\infty}_{0} \frac{x}{(1+x)^3}dx = \lim_{ t \to +\infty } \int^t_{0}  \frac{x}{(1+x)^3} dx = \lim_{ t \to +\infty } \int^t_{0}  \frac{x}{x^3+3x^2+3x+1} dx \\
原式 = \lim_{ t \to +\infty } \int^t_{0}  \frac{1}{x^2 +3x+3+ \frac{1}{x}} dx但是这里 \frac{1}{x}在x=0时不存在了\\
∴我们要用：\int^{+\infty}_{0} \frac{x}{(1+x)^3}dx =\int^{+\infty}_{0} \frac{x+1-1}{(1+x)^3}d(1+x) = \left( - \frac{1}{1+x} + \frac{1}{2} * \frac{1}{(1+x)^2}  \right)|^{+\infty}_{0} = \frac{1}{2}
\end{align*}$$
