$$\begin{align*}
求曲线 \frac{1}{(x-1)^2},与x=1,x=2，x轴所围成的平面图形的面积\\
\end{align*}$$
![[Pasted image 20250711140410.png]]
这个怎么计算呢？首先这不能用正常的定积分来计算，因为当x->1时，函数是无界的
$$\begin{align*}
取 \epsilon >0,且 \epsilon <2-1 \implies \epsilon+1 <2,取 [1+\epsilon,2] \subset (1,2]\\
同时，f(x)在(1,2]连续，那么在[1+\epsilon,2]上也连续\\
那么，连续一定可积，\int^2_{1+\epsilon} \frac{1}{(x-1)^2} dx肯定存在\\
那么 \lim_{ \epsilon \to 0^+ }  \int^2_{1+\epsilon} \frac{1}{(x-1)^2} dx就能表示出在(1,2]上的平面图形所围成的面积\\
同时，\lim_{ \epsilon \to 0^+ }  \int^2_{1+\epsilon} \frac{1}{(x-1)^2} dx可记为： \int^2_{1} \frac{1}{(x-1)^2} dx
\end{align*}$$
定义（==**瑕点指的是x=什么**==）：
	$$\begin{align*}
设 f(x)在(a,b]上连续，且 \lim_{ x \to a^+ } f(x) = \infty(无论\pm ) \\
同时我们将 x=a称为瑕点，那么 \forall 0< \epsilon<b-a,有 [\epsilon+a,b] \subset (a,b] \\
从而，f(x)在[\epsilon+a,b]上连续（因为定积分要在闭区间上定义）\\
∴f(x)在[\epsilon+a,b]上可积,即存在 \int^b_{a+\epsilon} f(x)dx\\
那么对于 \lim_{ \epsilon \to 0^+ }  \int^b_{a+\epsilon} f(x) dx,称作f(x)在 [a,b]上的第二类广义积分\\
并记作：\int^b_{a} f(x)dx
\end{align*}$$
	那么，对于第二类广义积分的敛散性：$$\begin{align*}
\int^b_{a} f(x)dx  =\lim_{ \epsilon \to 0^+ }  \int^b_{a+\epsilon} f(x)dx = \begin{cases}
极限存在，A(常数) \\
极限不存在
\end{cases}\\
当\lim_{ \epsilon \to 0^+ }  \int^b_{a+\epsilon} f(x)dx不存在时，称第二类广义积分发散\\
当\lim_{ \epsilon \to 0^+ }  \int^b_{a+\epsilon} f(x)dx = A(常数)时，称第二类广义积分收敛
\end{align*}$$
	又∵在连续的时候，可用[[牛顿-莱布尼兹公式]]：$$\begin{align*}
\int^b_{a} f(x)dx,x=a为瑕点\\
原式 = \lim_{ \epsilon \to 0^+ }  F(x)|^{b}_{a+\epsilon} = \lim_{ \epsilon \to 0^+ }  (F(b)- F(a+\epsilon)) = F(b)-\lim_{ \epsilon \to 0^+ }   F(a+\epsilon) \\
进一步地，我们可以变量代换:令 a+\epsilon =x,那么原式 =F(b)- \lim_{ x \to a^+ }  F(x )\\
上式我们可以记为：F(x)|^b_{a} 其中x=a为瑕点
\end{align*}$$
	稍微总结一下：$$ \int^b_{a} f(x)dx  =\lim_{ \epsilon \to 0^+ }  \int^b_{a+\epsilon} f(x)dx = \lim_{ \epsilon \to 0^+ }  F(x)|^{b}_{a+\epsilon} = F(b)-\lim_{ \epsilon \to 0^+ }   F(a+\epsilon)$$
	如果令a+\epsilon =x,那么有(==**注意这里的差异是趋近的数值从0变到a**==)：$$ F(b)- \lim_{ x \to a^+ }  F(x ) $$
	同理，当x=b为瑕点时,有：$$\begin{align*}
\int^b_{a} f(x)dx  =\lim_{ \epsilon \to 0^- }  \int^{b+\epsilon }_{a} f(x)dx = \lim_{ \epsilon \to 0^- }  F(x)|^{b+ \epsilon}_{a} = [\lim_{ \epsilon \to 0^- }   F(b+\epsilon) ] - F(a)(这个形式不够直观，不采用)\\
\int^b_{a} f(x)dx  =\lim_{ \epsilon \to 0^+ }  \int^{b-\epsilon }_{a} f(x)dx = \lim_{ \epsilon \to 0^+ }  F(x)|^{b- \epsilon}_{a} = [\lim_{ \epsilon \to 0^+ }   F(b-\epsilon) ] - F(a)
\end{align*}$$
	令 b+\epsilon =x,得到：$$\begin{align*}
[\lim_{ x \to b^- } F(x)] - F(a) 
\end{align*}$$
	同理还可得：$$\begin{align*}
a<c<b,其中x=c为瑕点 \\
此时：\int^b_{a} f(x)dx = \int^c_{a}f(x)dx+ \int^b_{c} f(x)dx\\
当且仅当等式右边的两个广义积分收敛时，左边的总体的广义积分才收敛\\
\end{align*}$$
	例题：$$\begin{align*}
\int^1_{0} \frac{1}{\sqrt{ 1-x^2 }}dx \\
∵x=1为瑕点，∴= \lim_{ b \to 1^- } \arcsin x|^b_{0} \frac{\pi}{2} \\
这一题的几何意义在于：y=\frac{1}{\sqrt{ 1-x^2 }},与 x=1,x=0和x轴所围成的平面图形的面积
\end{align*}$$
	对于广义积分和正常积分而言，我们还有一个特殊的判断：$$\begin{align*}
若 f(x)在[a,b]上连续，那么\int^b_{a} f(x)dx为正常积分\\
如果我将这个正常积分看成瑕积分？会是什么表现形式呢？\\
假设将x=a看成瑕点，\int^b_{a} f(x)dx  =\lim_{ \epsilon \to 0^+ }  \int^b_{a+\epsilon} f(x)dx \\
令 a+ \epsilon =x,原式=F(b)- \lim_{ x \to a^+ }  F(x ) \\
∵由连续可知： \lim_{ x \to a^+ } F(x) = F(a),原式= F(b)- F(a)
\end{align*}$$
	广义积分不可以看成是正常积分，但是正常积分可以看成广义积分
