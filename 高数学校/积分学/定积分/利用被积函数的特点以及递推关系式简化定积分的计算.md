	奇偶函数特性
	$$\begin{align*}
若 f(x)在 [-a,a]上连续，那么 \int^a_{-a} f(x)dx = \begin{cases}
0 \\
2 \int^a_{0} f(x)dx
\end{cases}
\end{align*}$$
	证明[[定积分的性质]]中的区间可加性(因为这里使用了变量代换，那么就需要连续这个条件)：
	$$\begin{align*}
\int^a_{-a} f(x) dx = \int^a_{0} f(x)dx + \int^0_{-a} f(x)dx \\
如果 f(x)为奇函数，那么 f(x) = - f(-x)\\
那么 \because f(x)在[-a,a]连续，那么令 x= -t\\
\int^a_{0} f(x)dx + \int^0_{-a} f(x)dx  = \int^a_{0} f(x)dx - \int^0_{a} f(-t) dt\\
接下来就要用到一个重要的性质：函数的自变量用什么字母表示，不影响函数的本质\\
那么\int^a_{0} f(x)dx - \int^0_{a} f(-t) dt = \int^a_{0} f(x)dx + \int^a_{0} f(-x) dx = 0 \\
如果是偶函数，沿用上面的式子，就是 \int^a_{0} f(x)dx + \int^a_{0} f(-x) dx = 2 \int^a_{0} f(x)dx\\
∵ f(x) = f(-x)
\end{align*}$$
	推论[[定积分的性质]]中的线性运算法则：
		$$\begin{align*}
若 f(x)在[-a,a]上连续，那么 \int^a_{-a} f(x)dx = \int^a_{0} f(x)+ f(-x)dx \\
\end{align*}$$
		这个性质，我们可以用更简单的方法证明：$$\begin{align*}
f(x) = \frac{f(x)+ f(-x)}{2} + \frac{f(x)- f(-x)}{2} \\
那么 , 前面的那部分永远都是偶函数，后面的那个部分永远是奇函数\\
那么 \int^a_{-a} f(x) dx = \int^a_{-a} \frac{f(x)+ f(-x)}{2} + \frac{f(x)- f(-x)}{2} dx \\
原式=  \int^a_{-a} \frac{f(x)+ f(-x)}{2} dx + \int^a_{-a} \frac{f(x)- f(-x)}{2} dx = 2\int^a_{0} \frac{f(x)+ f(-x)}{2} dx\\
那么 得到：\int^a_{0} f(x)+ f(-x) dx
\end{align*}$$
		例题[[计算a平方减x平方的积分]]：$$\begin{align*}
\int^1_{-1} \frac{x^{101}}{x^{80} +x^{60} +1} + x^{91}\sqrt{ 1-x^2 }+ \sqrt{ 1-x^2}dx\\
 这一道题不能将其看作整体，要分开来，但是可以将前两部分看作整体\\
原式= \int^1_{-1} \frac{x^{101}}{x^{80} +x^{60} +1} + x^{91}\sqrt{ 1-x^2 } dx +\int^1_{-1}  \sqrt{ 1-x^2}dx = \int^1_{-1}  \sqrt{ 1-x^2}dx\\
因为前面那个部分是奇函数，后面那个部分是偶函数， 2\int^1_{0}  \sqrt{ 1-x^2}dx= 2 *  \frac{1^2}{4} \pi =  \frac{\pi}{2}
\end{align*}$$

\int^1_{-1} \frac{x^{10}}{1+ e^x} dx 这个时候四大法宝和奇偶函数性质都不好用\\
此时，可以试着用推论：∵f(x)连续\\

	周期函数的特性
		$$\begin{align*}
若 f(x)连续，且为周期函数，周期为T，a为常数,那么\\
\int^{a+T}_{a} f(x)dx = \int^T_{0} f(x)dx
\end{align*}$$
		证明（用到变量代换）：$$\begin{align*}
\int^{a+T}_{a} f(x)dx 令 x= a+t(t≠T),那么此时 \int^T_{0} f(a+t) dt \implies 变量代换变成 x ,\int^T_{0} f(a+x) dx\\
但是怎么化掉 a+x呢？那么这个方法就证明不了了，我们需要换一种方法
\end{align*}$$
		利用区间可加性和变量代换证明：$$\begin{align*}
\int^{a+T}_{a} f(x)dx = \int^0_{a} f(x)dx + \int^T_{0} f(x)dx + \int^{a+T}_{T} f(x)dx \\
此时令 x= T+t,让\int^{a+T}_{T} f(x)dx = \int^a_{0} f(T+t) dt = \int^a_{0} f(t) dt \\
根据：函数的自变量用什么字母表示，不影响函数的本质\\
得到:\int^a_{0} f(t) dt = \int^a_{0} f(x)dx ,\\
那么\int^0_{a} f(x)dx + \int^T_{0} f(x)dx + \int^{a+T}_{T} f(x)dx=\int^0_{a} f(x)dx + \int^T_{0} f(x)dx + \int^a_{0} f(x)dx  \\
实质上= \int^T_{0} f(x)dx ，从而\int^{a+T}_{a} f(x)dx= \int^T_{0} f(x)dx
\end{align*}$$

\end{align*}$$

	cosx和sinx的关系
		$$\begin{align*}
n ∈ N或 n=0时，那么 \int^{\frac{\pi}{2}}_{0} \cos^n xdx =\int^{\frac{\pi}{2}}_{0} \sin^n xdx 
\end{align*}$$
		证明：这个我们需要怎么变换呢？我们就需要用 pi/2 的性质来转化cos到sin$$\begin{align*}
 \int^{\frac{\pi}{2}}_{0} \sin^n xdx 令 x= \frac{\pi}{2} -t,那么 : \int^{0}_{\frac{\pi}{2}} -\sin^n(\frac{\pi}{2} -t) dt =\int^{0}_{\frac{\pi}{2}} -\cos^nt dt = \int^{\frac{\pi}{2}}_{0} \cos^n tdt\\
 根据函数的自变量用什么字母表示，不影响函数的本质，\int^{\frac{\pi}{2}}_{0} \cos^n tdt = \int^{\frac{\pi}{2}}_{0} \cos^n xdx
\end{align*}$$
		那么我们需要这个等式，是因为我们可以根据等式算出两者的值：$$\begin{align*}
I_{n} = \int^{\frac{\pi}{2}}_{0} \cos^n xdx =\int^{\frac{\pi}{2}}_{0} \sin^n xdx .其中n ∈ N或 n=0\\
这个式子原函数求不出来，但是我又要计算他的值，那么我们就需要建立递推关系式\\
而无论定积分还是不定积分，都需要用到分部积分法来建立地推关系式\\
I_{n} = \int^{\frac{\pi}{2}}_{0} \cos^n xdx = \int^{\frac{\pi}{2}}_{0} \cos^{n-1} xd (\sin x) = \cos^{n-1} x *\sin x|^{\frac{\pi}{2}}_{0} - \int^{\frac{\pi}{2}}_{0} \sin x d(\cos^{n-1}x) (n\geq 1)\\
此时计算\cos^{n-1} x *\sin x|^{\frac{\pi}{2}}_{0}会发现有 0^0=1和0^n(n\geq 2)情况，那么此时分类讨论：\\
n\geq 2时，I_{n} = - \int^{\frac{\pi}{2}}_{0} \sin x d(\cos^{n-1}x) = - \int^{\frac{\pi}{2}}_{0} \sin x * ((n-1)(\cos x)^{n-2}*-\sin x) dx \\
原式= \int^{\frac{\pi}{2}}_{0} \sin^2 x(n-1)(\cos x)^{n-2} dx =(n-1) \int^{\frac{\pi}{2}}_{0} (1- \cos^2 x)(\cos x)^{n-2} dx \\
I_{n}=(n-1) \int^{\frac{\pi}{2}}_{0} (\cos x)^{n-2} - \cos^n x dx <=> I_{n} = (n-1)I_{n-2}- (n-1)I_{n} \\
则得：I_{n} = \frac{n-1}{n} I_{n-2}, n\geq 2 \\
那么n=1时，∵1+(n-1) \int^{\frac{\pi}{2}}_{0} (1- \cos^2 x)(\cos x)^{n-2} dx =1
\end{align*}$$

		例题：$$\begin{align*}
\int^1_{-1} x^8 \sqrt{ 1-x^2 } dx = 2\int^1_{0} x^8 \sqrt{ 1-x^2 } dx \\
这一道题因为和\sin x和\cos x无关，也不是周期函数，所以还是只能用常规的变量代换解决\\
令x= \sin t,那么 先变上下限，2\int^{ \frac{\pi}{2} }_{0} \sin^8 t \cos^2 tdt (注意这里开偶次方根要注意绝对值)\\
这里是因为积分区间为\left[ 0, \frac{\pi}{2} \right]才直接去绝对值，如果积分区间中被积函数有<0的，就要用区间可加性\\
如果我们要用到上面的公式，我们就需要用线性运算法则化出来\\
2\int^{ \frac{\pi}{2} }_{0} \sin^8 t \cos^2 tdt = 2\int^{ \frac{\pi}{2} }_{0} \sin^8 t (1- \sin^2 t)dt = 2\int^{ \frac{\pi}{2} }_{0} \sin^8 t - \sin^{10} tdt = 2\left[ \int^{ \frac{\pi}{2} }_{0} \sin^8 t dt -\int^{ \frac{\pi}{2} }_{0} \sin^{10} t dt \right]\\
最后 = 2* \frac{7}{8} * \frac{5}{6} * \frac{3}{4} * \frac{1}{2} * \frac{\pi}{2} * \left(  \frac{1}{10}  \right)
\end{align*}$$
		例题[[计算三角函数周期的方式]]：$$\begin{align*}
\int^{2\pi}_{0} \sin^{2n} x dx, n∈N\\
我们尽量用已有的来求定积分，\sin^{2n} x \implies \left( \frac{1}{2} (1- \cos 2x) \right)^n ,T= \frac{2\pi}{2} \\
那么这个式子可以转化为：2\int^{\pi}_{0} \sin^{2n} x dx\left( 无论从\int^{\pi}_{-\pi} ,或者\int^{\pi}_{0} + \int^{2\pi}_{\pi}都可以得到 \right) \\
进一步地：2*2\int^{ \frac{\pi}{2} }_{0} \sin^{2n} x dx ,此时我们可以用到我们的公式了\\
2*2\int^{ \frac{\pi}{2} }_{0} \sin^{2n} x dx = 4I_{2n} \\
\end{align*}$$
		但是这里的式子太长了，因此我们可以写成更简单的形式，从而要引入[[双阶乘]]的符号：
		$$\begin{align*}
7!! = 7 * 5* 3* 1\\
那么:2*2\int^{ \frac{\pi}{2} }_{0} \sin^{2n} x dx = 4I_{2n} = 4 * \frac{2n-1}{2n}* \frac{2n-3}{2n-2} *\dots * \frac{1}{2 } * \frac{\pi}{2} = 4 * \frac{(2n-1)!!}{(2n)!!} *  \frac{\pi}{2} 
\end{align*}$$
	运用变量代换化简xf(sinx)
		$$\begin{align*}
f(x)在 [0,1]上连续，那么 \int^{\pi}_{0} xf(\sin x)dx = \frac{\pi}{2} \int^{\pi}_{0} f(\sin x )dx
\end{align*}$$
		这个公式的推导关键在于变量代换换谁的问题，怎么将x换走或者换成关于sin的函数变成 f(sint) dt
		$$\begin{align*}
令 x= \pi -t \implies dx = -dt \\
\int^{\pi}_{0} x f(\sin x)dx = - \int^0_{\pi} (\pi- t) f(\sin (\pi-t) )dt = \int^{\pi}_{0} (\pi-t) f(\sin t) dt \implies 现在就可以变x了 \\
即：原式 =\int^{\pi}_{0} (\pi-x) f(\sin x) dx= \int^{\pi}_{0} \pi f(\sin x)- xf(\sin x) dx = \int^{\pi}_{0} \pi f(\sin x) dx - \int^{\pi}_{0} xf(\sin x) dx\\
那么\int^{\pi}_{0} x f(\sin x)dx = \int^{\pi}_{0} \pi f(\sin x) dx - \int^{\pi}_{0} xf(\sin x) dx \\
则：\int^{\pi}_{0} x f(\sin x)dx = \frac{\pi}{2} \int^{\pi}_{0}  f(\sin x) dx 
\end{align*}$$

原式 = -\frac{\pi}{2}\left[ \int^{\pi}_{0} 1 d(\cos x) - \int^{\pi}_{0} \cos^2 xd(\cos x) \right]  =\frac{2\pi}{3}

		例题：$$\begin{align*}
\int^{\frac{\pi}{2}}_{0} \frac{f(\sin x)}{f(\sin x)+ f(\cos x)} dx,其中 f(x)连续 \\
对于这道题，我们可以想一想将 \cos x换成 \sin x ,这时候就需要用到变量代换的思想\\
就是将 x= \frac{\pi}{2} - t,那么 \int^{\frac{\pi}{2}}_{0} \frac{f(\sin x)}{f(\sin x)+ f(\cos x)} dx = -\int^{0}_{ \frac{\pi}{2} } \frac{f( \cos t )}{f(\cos t)+ f(\sin t)} dt  =  \int^{\frac{\pi}{2}}_{0} \frac{f(\cos t)}{f(\sin t)+ f(\cos t)} dt \\
我们令 \int^{\frac{\pi}{2}}_{0} \frac{f(\sin x)}{f(\sin x)+ f(\cos x)} dx =I,那么我们新推的： \int^{\frac{\pi}{2}}_{0} \frac{f(\cos t)}{f(\sin t)+ f(\cos t)} dt =I\\
那么 I+I = \int^{\frac{\pi}{2}}_{0} 1dx, 2I = \frac{\pi}{2} ,I= \frac{\pi}{4}
\end{align*}$$
