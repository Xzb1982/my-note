内容（==**这个公式的前提是被积函数只依赖于 t**==）
	在$$\begin{align*}
若 f(x)连续，那么 \left( \int^x_{a} f(t)dt \right)' = f(x)
\end{align*}$$
	的基础上，我们有一个定理：$$\begin{align*}
若 f(x)连续， u(x)可导，那么 \left( \int^{u(x)}_{a} f(t)dt \right)' = f(u(x))u'(x)
\end{align*}$$
	证明[[复合函数中外层函数与极限运算交换定理]]：
		$$\begin{align*}
将\int^{u(x)}_{a} f(t)dt ,看成是 \int^u_{a} f(t)dt 和 u= u(x)复合\\
那么  \left( \int^{u(x)}_{a} f(t)dt \right)' = \frac{d}{du} \left( \int^u_{a} f(t)dt \right) *  \frac{du}{dx} = f(u(x))* u'(x)
\end{align*}$$
	那么对于：$$\begin{align*}
\int^x_{a} f(t) dt 我们也可以看成是复合函数： \left( \int^x_{a} f(t)dt \right)' = f(x) * \frac{dx}{dx} = f(x)
\end{align*}$$
	如果上下限都是函数呢？
		推论：$$\begin{align*}
若 f(x)连续， u(x)、v(x)均可导，那么 \left( \int^{u(x)}_{v(x)} f(t)dt \right) = f(u(x))u'(x) - f(v(x))v'(x)\\
\end{align*}$$
		证明[[定积分的性质]]中的区间可加性：$$\begin{align*}
\left( \int^{u(x)}_{v(x)} f(t)dt \right)' = \left( \int^a_{v(x)} f(t)dt+ \int^{u(x)}_{a} f(t)dt \right) ' \\
原式 = \left( -\int^{v(x)}_{a} f(t)dt \right)' +f(u(x))u'(x) = f(u(x))u'(x) - f(v(x))v'(x)
\end{align*}$$
		同理，如果f(x)连续，v(x)可导，那么:$$\begin{align*}
\left( \int^a_{ v(x)} f(t)dt \right)' = f(a)* 0- f(v(x))v'(x) \\
或者直接交换上下限：\left( \int^a_{ v(x)} f(t)dt \right)' = (-\int^{v(x)}_{a} f(t)dt)' 最后得到的也是一样的结果
\end{align*}$$
		例题：$$\begin{align*}
求 \left( \int^{x^4}_{x^3} \cos t^2 dt \right)' \\
原式 = 4x^3 \cos x^8 - 3x^2 \cos x^6
\end{align*}$$
		例题：$$\begin{align*}
设 f(x)连续 \\
求 \left( \int^{x^2}_{0} (x^2 -t)f(t)dt \right)' \\
对于这一题，我们不能直接套公式，因为被积函数中既有x又有t，不符合这个公式的前提条件\\
那么我们要不拆分，要么当做双变量函数求偏导\\
\left( \int^{x^2}_{0} (x^2 -t)f(t)dt \right)' = \int^{x^2}_{0} x^2 f(t) dt - \int^{x^2}_{0} tf(t)dt \\
原式 = x^2\int^{x^2}_{0}  f(t) dt - \int^{x^2}_{0} tf(t)dt = 前面是乘法的求导法则\\
原式= 2x\int^{x^2}_{0}  f(t) dt + x^2\int^{x^2}_{0}  f(t) dt - 2x^3 f(x^2) = 2x\int^{x^2}_{0}  f(t) dt + 2x^3 f(x^2) - 2x^3 f(x^2)\\
最后=2x\int^{x^2}_{0}  f(t) dt
\end{align*}$$
