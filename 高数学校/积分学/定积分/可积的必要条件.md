==**若f(x)在 [a,b]内可积分，那么 f(x)在 [a,b]内一定有界**==
但是f(x)在 [a,b]内有界，不一定得出f(x)在 [a,b]内可积分
例子：狄利克雷函数：
	$$\begin{align*}
D(x) = \begin{cases}
1,x为有理数 \\
0,x为无理数
\end{cases}
\end{align*}$$
	|D(x)|≤1，有界，但是不可积
	$$\begin{align*}
\lim_{ \lambda \to 0 }  \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i}  \\
在 D(x)的任何一个区间上，既有有理数存在，也有无理数存在\\
那么 = \begin{cases}
\lim_{ \lambda \to 0 }  \sum^n_{i=1} 1* \Delta x_{i} = b-a ,任何一个小区间上都取有理数 \\
\lim_{ \lambda \to 0 }  \sum^n_{i=1} 0* \Delta x_{i}=0 ,任何一个小区间上都取无理数
\end{cases}\\
这意味着两个和式极限存在但不相等，那么D(x)在任何一个区间上都不可积
\end{align*}$$
	如果 x在 [0,1]上，那么一个是1，一个是0，那么找到两个极限存在但不相等，就不可积了
