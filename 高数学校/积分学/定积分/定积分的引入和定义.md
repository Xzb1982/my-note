	求曲线y=f(x)连续(>=0),当x=a,x=b(a<b),x轴所围成的曲边梯形面积
	我们以这个为例子，因为这个是典型情况，知道了这种曲边梯形的求法。我们就能知道其他曲边梯形的求法
		![[mmexport1745812208607.jpg]]
		就例如让边界a与x轴相交的曲边梯形的面积
	我们联系一下之前求斜率[[导数]]，我们的思想就是先求近似值，再无限逼近
	那么对于求曲边梯形的面积，我们需要怎么求近似呢？
		答案就是将这个曲边梯形切得很小很小，就是小梯形之间的间隔很小很小
		分割，近似，求和，取极限
		下面我们来分割
			在[a,b]内插入n-1个分点
			$$\begin{align*}
x_{0}<x_{1}<x_{2}<\dots <x_{i}<\dots<x_{n-1}< x_{n} \\
我们记 x_{0} = a, x_{n} = b \\
这样我们将[a,b]分成了n个小区间： [x_{0},x_{1}], [x_{1},x_{2}],\dots, [x_{i-1},x_{i}],\dots, [x_{n-1},x_{n}]\\
设 x_{i} - x_{i-1} = \Delta x_{i},i = 1,2,\dots,n\\
之后，我们用这些小区间，将曲边梯形分成n个小曲边梯形 \Delta s_{1}, \Delta s_{2}, \Delta s_{3} ,\dots ,\Delta s_{n}\\
对于 \Delta s_{i}的面积，我们仍用 \Delta s_{i}这一记号来表示（\Delta s_{i}既表示曲边梯形，也表示其面积）
\end{align*}$$
			当我们在[a,b]上插入若干个分点之后，因为f(x)是连续的，那么就可以化曲为直了
			此时，我们将其看成矩形
			那么底边就是 △x_i ,同时因为看成矩形，两对边平行，那么此时区间左右端点f(x)就是一样的值了
			现在我们就可以表示曲边梯形的面积了，但是为了下定义方便，我们在[x_i-1,x_i]任取一点
			$$\begin{align*}
\forall \xi_{i} ∈ [x_{i-1},x_{i}],取 f(\xi_{i})作为高\\
\end{align*}$$
			下面开始取近似并作和：
			$$\begin{align*}
\Delta s_{i} \approx \Delta x_{i} * f(\xi_{i}),i = 1,2,\dots , n \\
S = \Delta s_{1} + \Delta s_{2} +\Delta s_{3}+ \dots + \Delta s_{n} = \sum^n_{i=1} \Delta s_{n} \\
从而，对于 \Delta x_{i} * f(\xi_{i}),有 \sum^n_{i=1} f(\xi_{i}) \Delta s_{n}
\end{align*}$$
			写完表现形式之后，我们就需要求极限达到化曲为直，化梯形为矩形
			具体怎么做呢？就要用将区间宽度不断缩小，当每个曲边梯形都->0,变成线段时
			那么此时就可以完全看成矩形了
			$$\begin{align*}
设 \lambda = max{\Delta x_{i}, 1\leq i \leq n},这样设避免了啰嗦，又能表达意思\\
只要最大的 \to 0,那么那些区间也是越来越小\\
再推广到 \sum ,就是：\lim_{ \lambda \to 0 }  \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} = S
\end{align*}$$
			但现在有几个问题，①区间趋于无穷小，是不是意味着0* 0就没意义了？
				此时，虽然区间趋于无穷小，但是无限个无穷小量之和，不一定是无穷小
				[[无穷小量]]中提到==**有限个无穷小相加还是无穷小，但是无限个无穷小量相加就不一定了**==
			②为什么 n不能 -> ∞ ？因为如果n->∞，那么此时分点-> ∞，按照原来的max{\Delta x_{i}, 1\leq i \leq n}
			那么第一个区间长度固定，你在后面加点，λ不一定就是->0的，有可能还是第一个区间的长度
			既然不能保证λ -> 0自然就不能为整个曲边梯形的面积
			之后我们在着眼于这个表达式：
			$$\begin{align*}
\lim_{ \lambda \to 0 }  \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} = S
\end{align*}$$
			这个表达式的极限是存在且唯一的，因为面积是唯一的
	例子2
		质点做变速直线运动
			设质点做变速直线运动，在t时刻的瞬时速度v=v(t)连续
			求t在 u到b上的这段时间内的质点运动的路程S
			这一例题也能表示成上面的和式极限的形式
			核心在于==**在很短的时间内，速度几乎不变**==
			![[Pasted image 20250627232411.png]]
			在[u,b]内插入n-1个分点
			$$\begin{align*}
就是 t_{1},t_{2}, \dots , t_{i-1}, t_{i}, \dots t_{n-1} \\
那么在[u,x_{1}], [t_{1},t_{2}] ,[t_{2},t_{3}] \dots [t_{i-1}, t_{i}] \dots [t_{n-1},b]上面\\
只要 \to 一个很小的时间，就可以化变速为匀速\\
\end{align*}$$
			下面取近似求和：
			$$\begin{align*}
 \forall \xi_{i} ∈ [t_{i-1},t_{i}],取 v(\xi_{i})作为高 \\
 列式 ： v(\xi_{i} ) \Delta t_{i} \\
 求和： \sum^n_{i=1} v(\xi_{i} ) \Delta t_{i} \\
 取极限： \lim_{ \lambda \to 0 }  \sum^n_{i=1} v(\xi_{i} ) \Delta t_{i} = S
\end{align*}$$
			理解这个和式极限的唯一性
				一方面，插入分点的方式是多种多样的
				即使分点确定，每个小区间也有无数种取法
				在此时，得到的和是不一样的
				但是，无论这个和是否一样，区间、插入点的取法是否一样，当λ-> 0时，和式极限应当=同一个值
	从和式极限引入定积分的定义
		既然这个和式极限这么重要，那么我们应当怎么将这个式子抽象出来变成一个通用的解法呢？
		在定义的过程中，我们希望条件越弱越好，那么我们就需要在上面分割、近似、作和、取极限中删去一些，增补一些表述
		设 f(x)在 [a,b]上有定义，在[a,b]内插入n-1个分点
		$$\begin{align*}
x_{0}<x_{1}<x_{2}<\dots <x_{i}<\dots<x_{n-1}< x_{n} \\
我们记 x_{0} = a, x_{n} = b \\
这样我们将[a,b]分成了n个小区间： [x_{0},x_{1}], [x_{1},x_{2}],\dots, [x_{i-1},x_{i}],\dots, [x_{n-1},x_{n}]\\
设 x_{i} - x_{i-1} = \Delta x_{i},i = 1,2,\dots,n\\
\forall \xi_{i} ∈ [x_{i-1},x_{i}],取 f(\xi_{i})作为高\\
\end{align*}$$
		下面我们直接到这里：
		$$\begin{align*}
\Delta x_{i} * f(\xi_{i})这个就是积分元，代表很小区间上面的元素,i = 1,2,\dots , n \\
\sum^n_{i=1} f(\xi_{i}) \Delta s_{n}，这个称为积分和式 \\
设 \lambda = max{\Delta x_{i}, 1\leq i \leq n}\\
只要最大的 \to 0,那么那些区间也是越来越小\\
\lim_{ \lambda \to 0 }  \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} 如果这个极限存在且唯一 = I\\
那么记作： \lim_{ \lambda \to 0 }  \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i}  = I= \int^b_{a} f(x)dx,称为f(x)在[a,b]上的定积分
\end{align*}$$
		为什么这里不需要 f(x)在 [a,b]内连续且 x≥0
			因为我的目的在于==**让这个和式极限存在且唯一**==
			[[某点的极限存在定理]]、[[极限不存在的情况]]
			那么连续的话，极限一定存在；不连续，极限还是可以存在的 

定积分的定义
	设 f(x)在 [a,b]上有定义，若$$\begin{align*}
\lim_{ \lambda \to 0 }   \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} (存在且唯一) = I
则记作 \int^b_{a} f(x)dx ，称为 f(x)在 [a,b]上的定积分
\end{align*}$$
	否则，就是f(x)在[a,b]上不可积 
		不可积的情况包括：①在[a,b]内找到两个和式极限存在却不相等
		②找到一个在[a,b]内的和式极限不存在
	同时，[a,b]称为积分区间，a为积分上限，b为积分下限
	恒力问题
		假设有一个质点M，在一个恒力f作用下，沿着AB这条直线做功
		这个恒力的作用方向与AB线段方向一致,大小不变，且帮助质点从A移动到B点
		![[Pasted image 20250628154753.png]]
		此时W = |f| * |vec(AB)|
		现在改变一下情景，![[Pasted image 20250628154959.png]]
		此时恒力f不再是沿着AB的方向做功了，而是沿着与AB夹角为θ的方向做功（如图）
		恒力的大小在运动的过程中始终不变，M仍然是被恒力f的作用下做功
		$$\begin{align*}
W = |\vec{f}| * \cos \theta * |\vec{AB}| = \vec{f} · \vec{AB}
\end{align*}$$
		例子：
			设一个质点M，在变化的力F的作用下，沿着直线从A点移动到B点所做的功W
			变力F与运动方向AB平行且一致，|vecF| = f(x) ,f(x)为连续函数
			![[Pasted image 20250628160455.png]]
			那这里我们可以将变速化匀速，用分割、近似、作和、取极限来做
			在AB之间插入n-1个分点，那么在这些分点的区间上，可以看成是恒力在作用
			那么：$$\begin{align*}
\forall \xi_{i} ∈ [x_{i-1},x_{i}],取 f(\xi_{i})作为高 \\
 列式 ： f(\xi_{i} ) \Delta x_{i} \\
 求和： \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} \\
 取极限： \lim_{ \lambda \to 0 }  \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} = W
\end{align*}$$
