内容(前提条件只要求在积分区间连续)
	$$\begin{align*}
若 f(x)在[a,b]内连续，且 F(x)是 f(x)在[a,b]上的一个原函数\\
那么\int^b_{a} f(x)dx = F(b) - F(a) 
\end{align*}$$
	证明[[微积分基本定理]]：$$\begin{align*}
由 f(x)在 [a,b]上连续，知f(x)在[a,b]上一定有原函数\\
\because F(x)也是f(x)在 [a,b]上的原函数,那么 \int^x_{a} f(t)dt = F(x)+C,x∈[a,b],C为常数\\
当x=a时，0 = F(a)+C, -F(a) = C;当x= b时， \int^b_{a} f(t)dt = F(b)+C = F(b)- F(a) \\
那么 \int^b_{a} f(t)dt = F(b)- F(a) \\
最后我们可以将t改x，因为函数的自变量用什么字母表示，不影响函数的本质\\
那么\int^b_{a} f(x)dx = F(b)- F(a) \triangleq F(x) |^b_{a}
\end{align*}$$
	当 a>b即上限<下限时，结论仍成立：$$\begin{align*}
∵ f(x)在 [b,a]上连续，那么 \int^b_{a} f(x)dx =- \int^a_{b} f(x)dx = -(F(a)-F(b) ) = F(b)- F(a)
\end{align*}$$
	那么现在我们求定积分不仅有定义法，还有牛顿莱布尼兹公式
	例题$$\begin{align*}
求 \int^1_{0} x^{10} dx = \frac{1}{11} x^{11} |^1_{0} = \frac{1}{11}
\end{align*}$$

将数列极限转化为定积分的形式
	内容：$$\begin{align*}
若 f(x)在[0,1]上可积，即 \int^1_{0} f(x)dx存在\\
那么把[0,1]n等分，每个区间长度为 \frac{1}{n},\\
分成n个小区间为： \left[  0, \frac{1}{n}  \right], \left[ \frac{1}{n}, \frac{2}{n} \right] ,\dots , \left[ \frac{i-1}{n} , \frac{i}{n} \right], \dots, \left[ \frac{n-1}{n}, \frac{n}{n} \right] \\
此时\lambda = \frac{1}{n},当 \lambda \to 0时， n \to \infty \\
此时取 \xi_{i} = \frac{i}{n} ∈ \left[  \frac{i-1}{n} , \frac{i}{n}\right] ,i =1,2,\dots,n \\
那么根据定积分的定义可以得到\int^1_{0} f(x)dx = \lim_{ n \to \infty }\sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} = \lim_{ n \to \infty } \sum^n_{i=1} f\left( \frac{i}{n} \right) \frac{1}{n} \\
或者,\xi_{i} = \frac{i-1}{n}时，=\lim_{ n \to \infty } \sum^n_{i=1} f\left( \frac{i-1}{n} \right) \frac{1}{n}
\end{align*}$$
	那么根据这个，我们可以用定积分求数列的极限
		则：$$\begin{align*}
根据上式，反之，\lim_{ n \to \infty } a_{n} = \lim_{ n \to \infty }  \sum^n_{i=1} f\left( \frac{i}{n} \right) \frac{1}{n} \\
当 f(x)在[0,1]上可积时，有\lim_{ n \to \infty }  \sum^n_{i=1} f\left( \frac{i}{n} \right) \frac{1}{n} = \int^1_{0} f(x)dx\\
更一般地 , 当 f(x)在[a,b]上可积时,有 \lim_{ n \to \infty } a_{n} = \lim_{ n \to \infty }\sum^n_{i=1} f \left( a+ \frac{(b-a)i}{n} \right) \frac{b-a}{n} = \int^b_{a} f(x) dx
\end{align*}$$
		那么那个更一般的式子是怎么变的呢？==**(一定要将 i/n看作x，1/n看作dx，换成x时，上限=1，下限=0，而且先写出通项，再转化成一般形式)**==
		$$\begin{align*}
\lim_{ n \to \infty }\sum^n_{i=1} f \left( a+ \frac{(b-a)i}{n} \right) \frac{b-a}{n} \\
= (b-a) \lim_{ n \to \infty }\sum^n_{i=1} f \left( a+ (b-a)* \frac{i}{n} \right) \frac{1}{n} = (b-a) \int^1_{0} f(a+ (b-a)x)dx \\
令 t= (a+ (b-a)x),那么 (b-a) \int^b_{a} f(t) dx,∵ (b-a)dx = dt,∴ 得证\\
\lim_{ n \to \infty }\sum^n_{i=1} f \left( a+ \frac{(b-a)i}{n} \right) \frac{b-a}{n}= \int^b_{a} f(t) dt \\
换成x，就是\lim_{ n \to \infty }\sum^n_{i=1} f \left( a+ \frac{(b-a)i}{n} \right) \frac{b-a}{n}= \int^b_{a} f(x) dx
\end{align*}$$
		那么我们以后求数列极限，可以用这个公式转换成定积分来做，前提是一定要写成这种形式：$$ \lim_{ n \to \infty }\sum^n_{i=1} f \left( a+ \frac{(b-a)i}{n} \right) \frac{b-a}{n} $$
	在[[定积分的引入和定义]]中，我们提到过关于n -> ∞时，λ不一定-> 0
	因为分点的方式有很多种，我可以让第一个区间长度一直是最大
		==**在一般情况下，λ ->0 => n->∞==**
		**==在等分的前提下，λ ->0 <=> n->∞**==

\lim_{ n \to \infty } \frac{1}{n} \left(  \sqrt{ 1+ \frac{1}{n} } + \sqrt{ 1+ \frac{2}{n} } + \sqrt{ 1+ \frac{3}{n} } +\dots + \sqrt{ 1+ \frac{i}{n} }+\dots +\sqrt{ 1+ \frac{n}{n}  }  \right) \\
\lim_{ n \to \infty } \sum^n_{i=1} \sqrt{ 1+ \frac{i}{n} } * \frac{1}{n} ;将 \frac{i}{n} 看作 x ; \frac{1}{n}看作 dx,可得 :\int^1_{0} \sqrt{ 1+x } dx

	例题[[两个重要极限以及其他极限]]：$$\begin{align*}
\int^1_{0} a^x dx = 用牛顿莱布尼兹公式很快就能得出来\\
但是，如果用定积分的定义：就要用另一思路了\\
\int^1_{0} a^x dx = \lim_{ n \to \infty } \sum^n_{i=1} f\left( \frac{i}{n} \right) \frac{1}{n} = \lim_{ n \to \infty } \frac{1}{n} \sum^n_{i=1} a^{\frac{i}{n}}  \\
原式 = \lim_{ n \to \infty } \frac{1}{n} \left( a^{\frac{1}{n}} + a^{\frac{2}{n}}+\dots +a^{\frac{i}{n}} + a \right)  = 等比数列求和 \lim_{ n \to \infty } \frac{1}{n} \frac{a^{\frac{1}{n}}\left[ 1- \left( a^{\frac{1*n}{n}} \right) \right]}{1- a^{\frac{1}{n}}} \\
在这里，我们用第六个重要极限\lim_{ x \to 0} \frac{a^x-1 }{x}=\ln a \\
最后= \frac{a-1}{\ln a}
\end{align*}$$

定积分证明题的类型
	涉及到定积分的方程的根的存在性/方程中有定积分/条件里有定积分
	涉及到定积分的ξ的等式/方程中有关于ξ的定积分/条件里有关于ξ的等式的定积分
	涉及到定积分的各种不等式
		以上都需要用到定积分的性质 
	证明定积分的等式命题（需要用到变量代换）
	例题[[罗尔定理]]：$$\begin{align*}
f(x)在[0,1]内连续，在(0,1)内可导，且 3\int^1_{  \frac{2}{3}} f(x) dx = f(0)\\
证明至少存在一点\xi ,使得 f'(\xi) =0 \\
证明： 3\int^1_{  \frac{2}{3}} f(x) dx = f(0) \implies \int^1_{ \frac{2}{3} } f(x)dx = \frac{1}{3} f(0) \\
\implies \int^1_{ \frac{2}{3} } f(x)dx = \left( 1- \frac{2}{3} \right) f(0) \\
由积分中值定理可得： ∵\int^1_{ \frac{2}{3} } f(x)dx = \frac{1}{3} f(\xi) = \frac{1}{3} f(0) ,则得： f(\xi) = f(0)\\
∴ 由罗尔定理，∵f(x)在[0,\xi] \subseteq [0,1]内连续，在(0,\xi) \subseteq (0,1)内可导,有f(\xi) = f(0) \\
那么一定存在一点 c ∈ \left( 0, \xi \right) \subseteq (0,1),使得 f'(c) = 0
\end{align*}$$
	例题[[定积分的性质]]，中的严格不等式性质：$$\begin{align*}
证明： 1< \int^1_{0} \sqrt{ 1+x^4 } dx < \frac{6}{5} \\
 f(x)= \sqrt{ 1+x^4 } ∈C[0,1],且x∈ [0,1] , \sqrt{ 1+x^4 } \geq 1且严格单调递增， \sqrt{ 1+x^4 } \not≡ 1  \\
那么 1< \int^1_{0} \sqrt{ 1+x^4 } dx 成立 \\
若 \sqrt{ 1+x^4 } 、 1+x^4 ∈ C[0,1],\\
且 x∈ [0,1], 1+x^4\left( 因为1+x^4在[0,1]的积分是 \frac{6}{5} \right) > \sqrt{ 1+x^4 } \\
且 f(x) \not≡  g(x)\\
那么 \int^1_{0} \sqrt{ 1+x^4 } dx < \frac{6}{5} = \int^1_{0} 1+x^4 dx
\end{align*}$$
