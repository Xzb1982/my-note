定积分的性质非常重要，在二重积分、三重积分、第一类曲面积分、第二类曲面积分也是这些性质

定积分性质的先导规定
	规定①$$\begin{align*}
当 b<a 时，规定 \int^b_{a} f(x) dx = - \int^a_{b} f(x) dx
\end{align*}$$
	推导：$$\begin{align*}
还是在[b,a]区间内插入n-1个分点，其中a=x_{0} > x_{1}> x_{2}>\dots > x_{n} = b\\
x_{i} - x_{i-1} = \Delta x_{i}<0, 当 b<a 时， \int^b_{a} f(x) dx = \lim_{ \lambda \to 0 }   \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} = \lim_{ \lambda \to 0 }   \sum^n_{i=1} f(\xi_{i} ) -|\Delta x_{i}| \\= - \lim_{ \lambda \to 0 }   \sum^n_{i=1} f(\xi_{i} ) |\Delta x_{i}| = - \int^a_{b} f(x)dx
\end{align*}$$
	规定②$$\begin{align*}
\int^a_{a} f(x)dx = 0
\end{align*}$$
	推导：$$\begin{align*}
\lim_{ \lambda \to 0 }   \sum^n_{i=1} f(a ) (a-a) = 0
\end{align*}$$
	![[Pasted image 20250629141653.png]]
	规定③：$$\begin{align*}
\int^b_{a} 1dx = \int^b_{a}dx = b-a,在这里，无论b,a谁大谁小都成立\\
因为有规定①的存在，可以在前面加个负号
\end{align*}$$
	推导：注意，在使用定积分的几何意义时，要一个上限>下限，积分函数>=0,而且和式极限存在这样的极限才能表示面积，否则是一个抽象的积分$$\begin{align*}
当 b>a时，\int^b_{a} 1dx = b-a\\
当 b=a时，\int^b_{a} 1dx = 0\\
当 b<a时,\int^b_{a}1dx = \int^a_{b} - 1dx = -(a-b) = b-a 
\end{align*}$$

定积分的性质
	①[[定积分的线性运算法则]]：
		$$\begin{align*}
	若 \int^b_{a} f(x)dx 、 \int^b_{a} g(x)dx 均存在 \\
	\forall 常数 \alpha 、\beta,那么 \int^b_{a} \alpha f(x) + \beta g(x)dx 存在且= \alpha \int^b_{a} f(x)dx + \beta \int^b_{a} g(x)dx 
	\end{align*}$$
		这里我们不规定上下限a、b的大小，也不规定α和β的正负性
		证明：$$\begin{align*}
	① 证明当 b>a时，\int^b_{a} (\alpha f(\xi_{i}) +\beta g(\xi_{i}) ) \Delta x_{i} 这个极限存在且唯一\\
	\end{align*}$$
		要想证明这个，我们先要引入连加号∑的性质：
			$$\begin{align*}
	\sum^n_{i=1} (\alpha u_{i}+ \beta v_{i}) = \alpha \sum^n_{i=1} u_{i} + \beta \sum^n_{i=1} v_{i} \\
	证明：因为连加号，本质上就是很多个加法, 拆出来就可以了\\
	\sum^n_{i=1} (\alpha u_{i}+ \beta v_{i}) = \alpha (u_{1}+u_{2}+\dots+ u_{n} )+ \beta (v_{1}+v_{2}+\dots + v_{n}) = \alpha \sum^n_{i=1} u_{i} + \beta \sum^n_{i=1} v_{i}
	\end{align*}$$
		那么，我们就可以拆分它了：$$\begin{align*}
	当 b>a时，\int^b_{a} (\alpha f(\xi_{i}) +\beta g(\xi_{i}) ) \Delta x_{i} =\lim_{ \lambda \to 0 } \left[  \alpha  \sum^n_{i=1} u_{i} \Delta x_{i}+ \beta  \sum^n_{i=1} v_{i} \Delta x_{i}  \right]\\
	原式 =\lim_{ \lambda \to 0 } \left[ \alpha   \sum^n_{i=1} u_{i} \Delta x_{i}+ \beta \sum^n_{i=1} v_{i} \Delta x_{i} \right]\\
	= \lim_{ \lambda \to 0 }   \alpha   \sum^n_{i=1} u_{i} \Delta x_{i} +  \lim_{ \lambda \to 0 }\beta  \sum^n_{i=1} v_{i} \Delta x_{i} = \int^b_{a} \alpha f(x)dx + \int^b_{a} \beta g(x)dx 
	\end{align*}$$
		证明完之后，和[[不定积分的线性运算法则]]作对比，发现不定积分要求αβ不同时为0
		但是定积分是不做要求的，这是因为α，β同时为0时，0的积分还是0
		为什么不定积分这样要求呢？因为当它们同时为0时：
	$$\int 0 dx= 0？,左边是任意常数C，右边是0，显然不成立$$
	②区间可加性(==**被积函数要一样才可以加**==)
		$$\begin{align*}
设 a,b,c为不相等的三个常数，且 f(x)在[\text{"min{a,b,c}"},\text{"max{a,b,c}"}]上可积\\
那么无论 a,b,c三者有怎么样的大小关系，都有： \int^b_{a} f(x)dx = \int^c_{a} f(x)dx + \int^b_{c} f(x)dx
\end{align*}$$
		先证明：当 a<c<b 时的情况[[可积的充分条件]]：
			注意：==**定积分 =》和式极限存在，但是一个和式极限存在 =》这个和式极限代表的定积分存在，我们需要说明无论ξi怎么取、无论怎么分点、得到的和式极限都存在且唯一，那么我们才能说这个这个和式极限代表的定积分存在**==
		$$\begin{align*}
\int^b_{a} f(x)dx = \lim_{ \lambda \to 0}  \sum^n_{i=1} f(\xi_{i})\Delta x_{i} ,我们将C作为一个固定的分点\\
原式= \lim_{ \lambda \to 0}  \sum_{[a,c]} f(\xi_{i})\Delta x_{i} + \sum_{[c,b]} f(\xi_{i})\Delta x_{i} =\int^c_{a} f(x)dx + \int^b_{c} f(x)dx \\
（因为大连续，小连续推出大可积小可积）
\end{align*}$$
			那么，在c<b<a时，证明：
			$$\begin{align*}
\int^a_{c} f(x)dx = 这次我们不用定义去证明，用最新成果\\
由(1)得： \int^a_{c} f(x)dx = \int^a_{b} f(x)dx + \int^b_{c} f(x)dx \\
\because c<b<a,那么 \int^a_{c} f(x)dx - \int^a_{b} f(x)dx = \int^b_{c} f(x)dx\\
\therefore \int^a_{c} f(x)dx + \int^b_{a} f(x)dx = \int^b_{c} f(x)dx\\
那么得到： \int^b_{a} f(x)dx = \int^b_{c} f(x)dx - \int^a_{c} f(x)dx = \int^b_{c} f(x)dx + \int^c_{a} f(x)dx
\end{align*}$$
			也就是说：![[Pasted image 20250629164505.png]]
			区间可加性经常用于计算分段函数的积分
	③不等式性质
		$$\begin{align*}
若 x∈ [a,b] , f(x)\geq 0且 \int^b_{a} f(x)dx 存在\\
那么 \int^b_{a} f(x)dx \geq 0 
\end{align*}$$
		证明（用定义）[[函数极限性质(一)]]中的不等式性质：$$\begin{align*}
\lim_{ \lambda \to 0 } \sum^n_{i=1} f(\xi_{i}) \Delta x_{i} \\
\because \forall \xi_{i} ∈ [a,b], f(\xi_{i}) \geq 0,且 a<b,\Delta x_{i} \geq 0\\
\therefore \lim_{ \lambda \to 0 } \sum^n_{i=1} f(\xi_{i}) \Delta x_{i} \geq 0
\end{align*}$$
		这一个性质经常在不通过计算判断定积分大小的时候用到
		推论：
			$$\begin{align*}
x∈ [a,b] , f(x) \geq g(x),且 \int^b_{a} f(x)dx 和\int^b_{a} g(x)dx 存在\\
那么有  \int^b_{a} f(x)dx \geq  \int^b_{a} g(x)dx
\end{align*}$$
			证明：$$\begin{align*}
\forall x ∈ [a,b], f(x) - g(x) \geq 0\\
根据定积分的线性运算法则，\because \int^b_{a} f(x)dx 和\int^b_{a} g(x)dx 存在\\
\therefore \int^b_{a} f(x) - g(x) dx也存在 ,同时根据性质③ ,\int^b_{a} f(x) - g(x) dx\geq 0\\
那么 \int^b_{a} f(x)dx - \int^b_{a} g(x) dx \geq 0 \implies \int^b_{a} f(x)dx \geq \int^b_{a} g(x) dx 
\end{align*}$$
	注意：==**运用定积分的性质一定要满足①被积函数≥0②上限>下限③和式极限一定要存在**==
	而且前面说的条件都只要求f(x)在 [a,b]内可积
	==只有线性运算法则不需要[a,b]内可积==
	④严格不等式（==**注意这个性质要求被积函数在积分区间连续**==）
		$$\begin{align*}
若 f(x) ∈C[a,b],x∈ [a,b] , f(x) \geq 0， f(x) \not≡ 0(就是在[a,b]上，至少能找到一个f(x)≠0的点) \\
那么 \int^b_{a} f(x) dx> 0
\end{align*}$$
		证明[[函数极限性质(一)]]中的性质③：$$\begin{align*}
由 x∈[a,b],f(x) \geq 0， f(x) \not≡ 0 \\
那么∃ x_{0} ∈(a,b) , f(x_{0} ) >0 \\ 
由 f(x) ∈C[a,b]，即 \lim_{ x \to x_{0} }  f(x) = f(x_{0}) >0\\
取 0< \eta = \frac{f(x_{0})}{2} (这里其实除3都可以) < f(x_{0})\\
∃ \vartheta >0,当 x∈ (x_{0}- \vartheta, x_{0}+\vartheta)时，都有 f(x_{0}) > \frac{f(x_{0})}{2} \\
但是我要的是闭区间，那么我们就设 0<\vartheta_{0} < \vartheta\\
[x_{0}-\vartheta_{0},x_{0}+ \vartheta_{0}] \subset  (x_{0}- \vartheta, x_{0}+\vartheta) \\
那么 利用区间可加性 \int^b_{a} f(x) dx = \int^{x_{0}-\vartheta_{0}}_{a} f(x)dx + \int^{x_{0}+\vartheta_{0}}_{x_{0}-\vartheta_{0}} f(x)dx + \int^b_{x_{0}+\vartheta_{0}} f(x)dx \\
第一和第三项都 \geq 0,那么\int^{x_{0}-\vartheta_{0}}_{a} f(x)dx + \int^{x_{0}+\vartheta_{0}}_{x_{0}-\vartheta_{0}} f(x)dx + \int^b_{x_{0}+\vartheta_{0}} f(x)dx \geq \int^{x_{0}+\vartheta_{0}}_{x_{0}-\vartheta_{0}} f(x)dx \\
又\because ∃ \vartheta >0,当 x∈ (x_{0}- \vartheta, x_{0}+\vartheta)时，都有 f(x_{0}) > \frac{f(x_{0})}{2} \\
\int^{x_{0}+\vartheta_{0}}_{x_{0}-\vartheta_{0}} f(x)dx \geq \int^{x_{0}+\vartheta_{0}}_{x_{0}-\vartheta_{0}} \frac{f(x)}{2} dx  = \frac{f(x_{0})}{2} * 2 \vartheta_{0} >0
\end{align*}$$
		这个证明精彩在于：①用连续的性质，搞出了值；②用保号性，构造中间值，使其可以>0,而且还用这个思想，将开区间转换为闭区间；③利用区间可加性逐步比较
		![[Pasted image 20250629214813.png]]
		推论：
			$$\begin{align*}
若 f(x) 、g(x) ∈ C[a,b],且 x∈ [a,b], f(x) \geq g(x) 且 f(x) \not≡  g(x)(就是至少有一点≠ g(x))\\
那么 \int^b_{a} f(x) dx>  \int^b_{a} g(x)dx
\end{align*}$$
		性质④的作用在于可以让定积分做分母
	⑤绝对值函数
		$$\begin{align*}
若 f(x)在 [a,b]上可积，那么 |\int^b_{a} f(x)dx| \leq \int^b_{a} |f(x)|dx
\end{align*}$$
		证明方法1：用绝对值不等式[[收敛数列的性质(二)]]：$$\begin{align*}
		用和的绝对值 \leq 绝对值的和, |a+b| \leq |a|+|b|\\
|\lim_{ \lambda \to 0 } \sum^n_{i=1} f(\xi_{i}) \Delta x_{i} | \leq \lim_{ \lambda \to 0 } \sum^n_{i=1} |f(\xi_{i})| \Delta x_{i} (因为上限 > 下限，所以不需要对 \Delta x_{i}加||) \\
将和式极限看成是关于\lambda 的函数，那么由函数的不等式性质得到： |\int^b_{a} f(x)dx| \leq \int^b_{a} |f(x)|dx
\end{align*}$$
		这一性质主要适用于带有绝对值的积分
		证明方法2：$$\begin{align*}
a,c为常数,c\geq 0, -c\leq a \leq c \leftrightarrow |a| \leq c
\end{align*}$$
		用这个性质，证明（用性质③）：
		$$\begin{align*}
当 x∈ [a,b] , -|f(x)| \leq f(x) \leq |f(x)|\\
\int^b_{a} -|f(x)|dx \leq \int^b_{a} f(x)dx \leq \int^b_{a} |f(x)|dx\\
那么用我们引入的这个性质，可得: |\int^b_{a} f(x)dx| \leq \int^b_{a} |f(x)|dx
\end{align*}$$
	这些不等式性质让我们可以==**在等式/不等式两边没有积分号时，加上积分号**==
	⑥估值定理
		$$\begin{align*}
若 f(x)在 [a,b]内可积，m\leq f(x) \leq M, x∈[a,b] \\
其中 m,M为常数，那么 m(b-a)=\int^b_{a} mdx  \leq \int^b_{a} f(x)dx \leq \int^b_{a} Mdx = M(b-a)
\end{align*}$$
		这个性质一般用于估计定积分的范围
	⑦积分中值定理(==**只要求被积函数在[a,b]内连续**==)
		$$\begin{align*}
若 f(x)∈ C[a,b]，那么∃ 一点 \xi ∈[a,b],使得\int^b_{a}f(x)dx = f(\xi)(b-a)
\end{align*}$$
		证明：用分析法[[最大最小值定理]]
		$$\begin{align*}
要证原结论成立，只要证：∃  \xi ∈[a,b],使得 \frac{\int^b_{a} f(x)dx}{b-a} = f(\xi)成立\\
由 f(x)∈ C[a,b],则在 [a,b]上一定能取到最大值M和最小值m，值域[m,M]\\
那么只要证： m\leq \frac{\int^b_{a} f(x)dx}{b-a} \leq M成立即可\\
那么根据性质⑥，只要证：m(b-a)\leq \int^b_{a} f(x)dx \leq M(b-a)\\
\because  f(x)∈ C[a,b],而连续一定可积 \therefore m(b-a)\leq \int^b_{a} f(x)dx \leq M(b-a)显然成立
\end{align*}$$
		积分中值定理的几何意义
			$$\begin{align*}
\frac{\int^b_{a} f(x)dx}{b-a} = f(\xi),称为 f(x)在 [a,b]上的函数平均值
\end{align*}$$
			函数平均值，因为定积分是一个和式极限，是无穷多个数，但是为什么这仍然是个平均值呢？
			当 f(x) ≥ 0,a<b时：
			![[Pasted image 20250630142614.png]]
			我们画出$$ \int^b_{a} f(x)dx = f(\xi)(b-a) $$这个的图像，等号右边的图像我们可以看成矩形，等号左边是一个曲边梯形
			这个矩形刚好将曲边梯形变成一个规则的矩形，而且仍然保持曲边梯形面积相等
			当b< a时，我们交换积分上下限发现上式仍然成立：
			$$\begin{align*}
\int^b_{a} f(x)dx = - \int^a_{b} f(x) dx = -f(\xi)(a-b) (b\leq f(\xi) \leq a)\\
= f(\xi)(b-a) 
\end{align*}$$
			当 a=b时，该结论仍成立，只不过结果为0

总结：规定①要求上限>下限即可
规定②、③都没有特殊的前提条件
线性运算法则只要求定积分存在
区间可加性和估值定理只要求在区间内可积
不等式性质要求：被积函数≥0、上限>下限、对应和式极限存在且唯一
严格不等式和积分中值定理要求区间内连续
