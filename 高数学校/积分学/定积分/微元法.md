[[定积分的引入和定义]]中，我们提到了分割近似作和取极限，但是有没有更简洁的方法，将一个复杂的求曲边梯形的问题通过更简洁的方式转化为定积分呢？
内容
	$$\begin{align*}
求一个分布在[a,b]上的量Q的值，此时我们仍用Q来表示\\
且 当 Q=\sum^n_{i=1} \Delta Q_{i} (即总量=部分量之和，也就是分成n个小区间之后加和)\\
上面这个条件保证了能分割 \\
分割：在[a,b]内插入n-1个分点，a= x_{0} < x_{1}< x_{2}< \dots< x_{i} < \dots< x_{n-1}< x_{n} =b \\
相应地，分成n个小区间[ x_{i-1}, x_{i}], i=1,2,3 ,\dots,n \\
此时设 x_{i} - x_{i-1} = \Delta x_{i}, \lambda = MAX {\Delta x_{i} , 1\leq i\leq n}\\
那么此时分成了n个部分量，即： \Delta Q_{i}, i= 1,2,3,\dots , n(分割成的量的具体名字要看问题)\\
例如分成n个小曲边梯形，分成n段位移，分成n段位移所做的功\\
\end{align*}$$
	分割的这一步总结起来就是：==**插入n-1个分点，分成n个小区间，作差得到△，△取最大**==
	第二步：取近似：$$\begin{align*}
\forall \xi_{i} ∈ [x_{i-1}, x_{i}], \Delta Q_{i} \approx f(\xi_{i} ) \Delta x_{i}, i= 1,2\dots,n\\
对于上式，最关键的是f(\xi_{i})取什么样的函数值，在不同问题里面有不同的叫法\\
例如，f(\xi_{i})可以表示曲边梯形中分割后对应的矩形的高、第i个平均时间段的瞬时速度\\
又或者\xi_{i}也可以表示成在某区间上的力就是这个区间上的平均力
\end{align*}$$
	取近似进一步总结就是：取区间上任意一点，然后取对应函数值再乘以对应底边
	第三步：作和：$$\begin{align*}
∵ Q=\sum^n_{i=1} \Delta Q_{i} \\
∴ \approx \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i}
\end{align*}$$
	作和就是写∑负号
	第四步：取极限：$$\begin{align*}
\lim_{ \lambda \to 0 } \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} = Q
\end{align*}$$
	取极限就是写lim（形式上是这样的，实质上就是最大区间长度->0）
	那么根据[[定积分的引入和定义]]，我们可以得到：$$\begin{align*}
若 f(x)在[a,b]上可积，那么\lim_{ \lambda \to 0 } \sum^n_{i=1} f(\xi_{i} ) \Delta x_{i} = \int^b_{a} f(x)dx = Q
\end{align*}$$
	进一步的结论
		在这里面，==**取近似是最重要的，这里面反映着化曲为直的思想**==
		找到了Q所分布的区间，就说明找到了Q的积分区间
		那么被积表达式，就是取近似中的f(x)dx
		那么，我们可得到：$$\begin{align*}
求一个分布在[a,b]上的量Q的值，此时我们仍用Q来表示\\
且 当 Q=\sum^n_{i=1} \Delta Q_{i} (即总量=部分量之和，也就是分成n个小区间之后加和)\\
x_{i-1} = x,x_{i} = x+\Delta x= x+dx,取 x∈ [x, x+dx] \\
那么 \Delta Q \approx f(x)dx \implies Q = \int^b_{a} f(x)dx
\end{align*}$$
		但是这里面就有一个问题，就是取近似时，要有什么标准的问题
			例如A近似成：Delta Q = f(x)dx
			B近似成：Delta Q = 1/2 f(x)dx
			C近似成：Delta Q = g(x)dx
			而不同的近似结果，近似出来是不一样的，那么应该怎么改进呢？
			那么我们就要在这些近似中选出一个最佳近似
			那么怎么选择呢？在[[基本初等函数的微分公式]]中，我们提到：$$\begin{align*}
若 y = f(x)在 x处可微，即 \Delta y = A \Delta x+ o(\Delta x),其中 A是与\Delta x无关的常数\\
\Delta y = A \Delta x+ o(\Delta x) = f'(x)\Delta x+ o(\Delta x) = f'(x)d x+ o(\Delta x) = dy+o(\Delta x)\\
其中 \Delta x \to 0 ,那么这个近似 dy \approx \Delta y
\end{align*}$$
			现在我们先回顾一下微分 （==**微分强调的是对微小变化的分解**==）：$$\begin{align*}
若y= f(x)在x处可导,那么按导数定义\lim_{ \Delta x \to 0 } \frac{\Delta y}{\Delta x}= f'(x) \\
\implies \frac{\Delta y}{\Delta x}= f'(x)+ \alpha ,\lim_{ \Delta x \to 0 } \alpha=0 \\
\implies \Delta y= f'(x)\Delta x+ \alpha \Delta x\implies \lim_{ \Delta x \to 0 } \frac{\alpha \Delta x}{\Delta x}=0\implies  \alpha \Delta x=o(\Delta x)(\Delta x\to 0)\\
\implies \Delta y= f'(x)\Delta x+ o(\Delta x)(\Delta x \to 0)
\end{align*}$$
			微分的定义：$$\begin{align*}
设y= f(x),若\Delta y= f(x+\Delta x)-f(x)可以被表示成\Delta y= A\Delta x+o(\Delta x)(前提是\Delta x\to 0)\\
其中A与\Delta x无关，这里的A的可导条件弱化了，不再只是f'(x)所带有的可导的意义\\
那么我们可称y= f(x)在x处可微
\end{align*}$$
			可微的充要条件：$$f(x)在x处可微的充要条件是f(x)在x处可导且A=f'(x)$$
			即：函数在x0处可微 <=> 函数在x0处可导
			证明：$$\begin{align*}
当 f'(x)≠0 时，\lim_{ \Delta x \to 0 } \frac{\Delta y}{dy} = \lim_{ \Delta x \to 0 } \frac{f'(x)\Delta x+ o(\Delta x)}{f'(x)\Delta x} = \lim_{ \Delta x \to 0 } 1+ \frac{1}{f'(x)} *\frac{o(\Delta x)}{\Delta x} =1 \\
那么 \Delta y \sim dy(前提是 \Delta x\to 0),而实质上 dy = f'(x)\Delta x,\Delta y 的线性主部就是 dy
\end{align*}$$
			说了这么多，无非就是想说：Delta Q ≈ dQ就是最佳近似,同时： f(x)dx = dQ
			那么我们就可以得到微元法的定义
	[[微元法]]
		定义：$$\begin{align*}
求一个分布在[a,b]上的量Q的值，此时我们仍用Q来表示\\
且 当 Q=\sum^n_{i=1} \Delta Q_{i} 时(总量=部分量之和)\\
选取 [x,x+ dx] \subset [a,b] (dx>0)\\
\Delta Q \approx f(x)dx = dQ(一再强调 \Delta Q 和 dQ中间隔着一个o(\Delta x))\\
Q= \int^b_{a} f(x)dx
\end{align*}$$
		那么这种方法简称为微元法:简单来说就是找到图形的积分区间及其所代表的被积函数
		然后定积分计算即可
	

连续或单调，一定可积；可积一定有界
	那么定积分的计算就是：
	①四大法宝（线性运算法则、凑微分或变量代换、分部积分（通常和构造递推关系式结合在一起））
	②利用被积函数的特点（例如奇偶、周期，cos和sin的关系）来计算定积分
	③牛顿莱布尼兹公式
	那么定积分的性质：
		①交换上下限、一条线的定积分=0、1的定积分为积分区间作差
		②区间可加性（经常用于分段函数拆分区间）
		③不等式性质和严格不等式性质经常用于比大小或者和夹逼结合
		④绝对值函数通常和绝对值不等式结合一起考或者和③一起考,也可以和区间可加性一起考
		⑤估值定理和积分中值定理通常和定积分的几何意义结合
