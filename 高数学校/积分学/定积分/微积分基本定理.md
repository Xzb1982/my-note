又称变上限求导定理
内容
	$$\begin{align*}
若 f(x)在[a,b]内连续，那么 G(x) = \int^x_{a} f(t)dt在 [a,b]上可导\\
且 G'(x) = f(x)
\end{align*}$$
	我们猜测这个G'(x)肯定和f(x)有关，因为根据不定积分，积分和导数互为逆运算
	那么我们对定理的证明，要从导数定义下手[[导数的定义]]
		$$\begin{align*}
\forall x ∈ [a,b], \lim_{ \Delta x \to 0 } \frac{G(x+ \Delta x)- G(x)}{\Delta x} \\
∵ G(x) = \int^x_{a} f(t)dt, ∴ 原式= \lim_{ \Delta x \to 0 } \frac{ \int^{x+ \Delta x}_{a} f(t)dt  - \int^{x}_{a} f(t)dt }{\Delta x} \\
原式 = \lim_{ \Delta x \to 0 } \frac{ \int^{x+ \Delta x}_{a} f(t)dt + \int^{a}_{x} f(t)dt }{\Delta x} =\lim_{ \Delta x \to 0 } \frac{ \int^{x+ \Delta x}_{x} f(t)dt  }{\Delta x}
\end{align*}$$
		注意，当x -> a时，要从>0的方向 -> 0; x -> b时，要从<0的方向趋近
		同时[[定积分的性质]]中的积分中值定理，[[海涅定理]]：
		这里我们需要引用一下[[积分中值定理]]
		$$\begin{align*}
\because f(x) ∈ C[a,b]\\
\therefore 根据积分中值定理 ∃  \xi ∈[a,b],使得\int^{x+\Delta x}_{x}f(t)dt = f(\xi)\Delta x \\ 
\lim_{ \Delta x \to 0 } \frac{ \int^{x+ \Delta x}_{x} f(t)dt  }{\Delta x} = \lim_{ \Delta x \to 0 } \frac{f(\xi) \Delta x }{\Delta x} (\xi的取值介于 x与 x+ \Delta x之间) \\
因为\Delta x已经被约掉了，那么此时 \xi要趋于什么呢？\\
∵ \Delta x \to 0时， x+ \Delta x, x都 \to x .用夹逼定理 : \lim_{ \xi \to x } f(\xi) \\
又 \because \lim_{ \xi  \to x } f(\xi) = f(x) = G'(x) (实际上用到了海涅定理,即\forall \xi_{n} \to x,都有f(\xi_{n})\to f(x))  \\
而且这里的 \xi 受到 \Delta x 限制，给一个 \Delta x,通过积分中值定理确定一个 \xi \\
（因为f(\xi)在几何意义上代表的是平均高度,其背后所代表的\xi也会发生改变）
\end{align*}$$
		那么$$\begin{align*}
G'(x)(对x求导) = \left( \int^x_{a} f(t)dt \right)' = \frac{d}{dx} \left(  \int^x_{a} f(t)dt  \right) \\
注意，这里是关于x的变上限积分，t 只是告诉我们积分时沿着哪个方向取值 \\
或者这样理解：把它看成两个步骤的复合映射。\\
第一步：给定 x，确定区间 [a,x]。\\
第二步：对这段区间做积分运算，得到一个数。\\
\end{align*}$$
	定积分原函数存在定理
		因为在不定积分中，我们用原函数定义不定积分，因此不定积分本质上是求导的逆运算
		但是定积分不一样，定积分由和式极限定义，我们需要用一个定理来规定定积分积出来之后原函数存在的情况[[不定积分导论]]
		那么这个推论就出来了：$$\begin{align*}
若 f(x) ∈ C[a,b],那么 f(x) 在[a,b]上一定有原函数
\end{align*}$$
		证明：$$\begin{align*}
令 G(x) = \int^x_{a} f(t)dt,x∈ [a,b] \\
那么 G'(x) = f(x) 说明变上限函数就是f(x)的一个原函数
\end{align*}$$
		那么这就意味着如果被积函数在某区间上连续，那么我们就==**能够求出其原函数，而且其变上限函数就是这个被积函数的原函数之一**==
		但是，即便是有原函数，我们也不一定能求出它，因为原函数有可能是==**非初等函数**==
		非初等函数用不了不定积分的方法来求
下面我们需要一个更简单的方法求定积分
这个时候就需要[[牛顿-莱布尼兹公式]]
