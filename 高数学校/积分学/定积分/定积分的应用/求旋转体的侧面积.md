	先求小不规则立体的侧面积，再求大的
	那么如果我们将这个小不规则立体近似看成圆柱体，和看成圆台，所得出来的表达式可以是一样的，也可以是不一样的，==**关键在于dx还是ds（是直线还是弧线）**==
	下面我们边推导边理解：$$\begin{align*}
S_{x}分布在[a,b]上，取 [x,x+dx] \subset [a,b ] \\
\Delta S = (圆台的侧面积公式) \pi (|f(x)|+|f(x+dx)|) dx= \pi (|f(x)|+|f(x+dx)|) \Delta s\\
如果此时，用圆柱的侧面积公式：2\pi |f(x)|dx\\
\pi (|f(x)|+|f(x+dx)|) \Delta s = (因为连续,换掉f(x+dx)) \pi *2|f(x)|\Delta s\\
接下来换掉\Delta s变成ds,因为ds是\Delta s的最佳近似,那么 \pi *2|f(x)|\Delta s=\pi *2|f(x)|ds\\
根据弧长公式，在y=y, y=f(x)情况下,\pi *2|f(x)|ds=\pi *2|f(x)| \sqrt{ 1+(f'(x))^2 }dx= dS_{x} \\
S_{x} = \int^b_{a} 2\pi |f(x)|\sqrt{ 1+(f'(x))^2 }dx
\end{align*}$$
		*更进一步地，如果近似成圆柱的表达式和近似成圆台的表达式本质上只差一个高阶无穷小*
		*那么，两者本质上可以近似成一样的表达式*
		证明：$$\begin{align*}
\lim_{  \Delta x \to 0 } \frac{2\pi |f(x)| \sqrt{ 1+(f'(x))^2 }\Delta x - 2\pi |f(x)|\Delta x}{\Delta x}= 2\pi |f(x)| \sqrt{ 1+(f'(x))^2 } - 2\pi |f(x)|\\
原式 = \lim_{  \Delta x \to 0 } 2\pi|f(x)|(\sqrt{ 1+(f'(x))^2 } -1) \\
∵ \Delta x没了，一般来说， f(x) \not= 0, 而且一般来说f'(x)也 \not= 0
\end{align*}$$
			因为当f(x)=0时，为常值函数，是一个线段，没有侧面积
			当 f'(x)=0时，整个立体就是一个矩形
			再进一步推导，==**实际上dx和ds的差异本质上就是圆台与圆柱侧面积公式的差异**==
	值得一提的是，ds在这里才用到，[[求旋转体体积]]不用ds，因为ds是弧长，dx能更好地近似
		如果你用 ds（沿曲线方向的弧长）来当做“厚度”，切片面就从“垂直于 x轴”变成了“垂直于切线方向”——那既不是标准的圆盘法，也不是壳层法，几何意义完全不同。
		用 ds 厚的“切片”却会“斜着”塞，既有重叠也会有缝隙，无法准确贴合原体。
		**体积微元要沿着轴向切分**：==**当我们旋转曲线 y=f(x) 绕 x 轴时，每一个微元体都是“垂直于 x 轴”的薄壳或薄盘；它的厚度正是 dx，因为切片面要和轴垂直。**==
	例题：半径为R的球的球面面积
		![[Pasted image 20250709205212.png]]
		$$\begin{align*}
由 x^2 + y^2 = R^2 ,我们不需要将y解出来，直接借助方程来确定表达式：\\
两边同时对x求导：2x+ 2y*y' =0 \implies y' = - \frac{x}{y} \\
那么S_{x} = \int^b_{a} 2\pi |y|\sqrt{ 1+ \frac{x^2}{y^2} }dx = 4\pi R^2
\end{align*}$$
