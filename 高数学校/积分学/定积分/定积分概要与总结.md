计算曲边梯形的面积
	分割近似求和取极限
	定积分的概念：
		先分割$$\begin{align*}
若函数f(x)在 [a,b]上有界\\
那么我们将这个区间分成i个小区间，分法任意，\\
如： a= x_{0} < x_{1} < x_{2} \dots <x_{i}= b,[x_{0},x_{1}],[x_{1},x_{2}]\dots[x_{i-1},x_{i}]\\
并记： \Delta x_{m}= x_{m}- x_{m-1},其中 m=1,2,\dots,i\\
\end{align*}$$
		之后近似：$$\begin{align*}
在每个小区间 任取一点 \xi_{m},取法不限, 其中  x_{m-1}\leq \xi_{m}\leq x_{m}\\
f(\xi_{m})\Delta x_{m},其中m=1,2,\dots,i
\end{align*}$$
		然后求和：
		$$\begin{align*}
S_{n}= f(x_{1})\Delta x_{1}+ f(x_{2})\Delta x_{2}+ \dots + f(\xi_{m})\Delta x_{m}= \sum^i_{m=1} f(\xi_{m})\Delta x_{m}
\end{align*}$$
		最后从总体上取极限：
		$$\begin{align*}
		当差值\to 0时，即整个小矩形的面积被分得越来越小：\\
\lambda = max{ \Delta x_{1}, \Delta x_{2} \dots \Delta x_{m}},当 \lim_{ \lambda \to 0 } S_{n}= \lim_{ \lambda \to 0 } \sum^i_{m=1}f(\xi_{m})\Delta x_{m}= I
\end{align*}$$
		最后定义：
		$$\begin{align*}
\int^b_{a} f(x)dx= \lim_{ \lambda \to 0 } \sum^i_{m=1}f(\xi_{m})\Delta x_{m}= I
\end{align*}$$
定积分存在定理
	$$\begin{align*}
当 f(x)在 [a,b]上连续时，则称 f(x)在[a,b]上可积
\end{align*}$$
	原因：①在计算曲边梯形面积时，我们用到了取极限的思想，根据[[函数定义归纳]] 
	既然要能取极限，就要在这个点左右无穷小区间内连续

	$$\begin{align*}
函数 f (x)在区间[a,b]上有界，且只有有限个间断点\\
那么f(x)在 [a,b]上可积
\end{align*}$$
	f(x)及区间[a,b]确定后，定积分为确定的数

	定积分的性质：

变上限积分
	设函数 f (x) 在区间[a,b]上连续，并且设x为[a,b]上的一点
	$$\begin{align*}
\int^x_{a} f(x) dx ,此时上限x 在区间[a,b]上任意变动\\
则对于每一个取定的x值，定积分有一个对应值\\
所以f(x)在[a,b]上定义了一个函数
\end{align*}$$
	则记：$$\begin{align*}
\Phi (x)= \int^x_{a} f(t)dt 为积分上限函数或变上限积分
\end{align*}$$
	积分上限函数的性质
		①积分上限函数的导数：
			$$\begin{align*}
	f (x)在[a,b]上连续，若 \Phi(x) = \int^x_{a} f(t)dt在[a,b]上可导，那么其导数应为：\\
	\Phi'(x) = \frac{d}{dx}  \int^x_{a} f(t)dt = f(x) ,(a\leq x\leq b)
	\end{align*}$$
			因为积分和导数互为逆运算
			证明：
			$$\begin{align*}
	\forall x∈ (a,b),设增量h,h 使得 x+h ∈ [a,b],那么有：\\
	\frac{\Phi (x+h)- \Phi(x)}{h} = \frac{1}{h} \left(  \int^{x+h}_{a} f(t)dt - \int^x_{a} f(t)dt  \right)= \frac{1}{h} \int^{x+h}_{x} f(t)dt(这里用了区间可加性)\\
	在 [x,x+h]上, f(t)= f(x) +(f(t)- f(x)),那么  \frac{\int^{x+h}_{x} f(x)dt +  \int^{x+h}_{x} f(t)- f(x)dt}{h} \\
	原式= \frac{f(x)h +\int^{x+h}_{x} f(t)- f(x)dt}{h} \\x虽然是在[a,b]的变量，但是因为积分变量是t,因此，f(x)在这里可以看成是常数
	\end{align*}$$
			那么我们就将 f(x)求出来了:
			$$\begin{align*}
	 \frac{f(x)h +\int^{x+h}_{x} f(t)- f(x)dt}{h}= f(x)+ \frac{\int^{x+h}_{x} f(t)- f(x)dt}{h}\\
	 ∴ 用极限的思想： ∀ε>0，存在 N >0,当|t-x|< N \\
	 都有 |f(t)- f(x)| < ε
	\end{align*}$$

		②$$\begin{align*}
\left( \int^a_{x} f(t)dt \right)'= - f(x) \\

\end{align*}$$

\left( \int^{b(x)}_{a(x)} f(t)dt \right)' = f(b(x))* b'(x) - f(a(x))* a'(x)
\end{align*}$$

若 f(x)在 [a,b]上连续，\\

牛顿莱布尼兹定理
	如果F(x)是连续函数f(x)是[a,b]上的一个原函数
	$$\begin{align*}
\int^b_{a} f(x)dx = F(b)- F(a) 
\end{align*}$$
	证明：
		$$\begin{align*}
\because F(x)是f(x)在[a,b]上的一个原函数，根据原函数存在定理\\
\Phi (x) 也是 \int^x_{a} f(t)dt关于f(x)的一个原函数\\
又\because 原函数的唯一性 \therefore F(x)- \Phi(x)= C\\
又\because x∈ [a,b],那么 F(a)- \Phi(a)= C\\
又\because \Phi(a) = \int^a_{a} f(t)dt =0 ,\therefore F(a)=C\\
\because F(x)- \Phi(x)= C,那么 \Phi(x) = F(x)- F(a) => \int^x_{a} f(t)dt = F(x)- F(a)\\
那么当x=b时，\Phi(b) = \int^b_{a} f(x)dx = F(b)-F(a)
\end{align*}$$
		一个连续函数在[a,b]上的定积分=它的任意一个原函数在[a,b]上的增量
		定积分=增量

定积分的换元积分法和分部积分法
	$$\begin{align*}
假设 :\\
（1） f (x)在[a,b]上连续；\\
（2）函数 x= \phi(t) 在[a,b]上有连续导数； \\
（3） 当t 在区间[a,b]上变化时，x = \phi(t)的 值在[a,b]上变化，且 \phi(\alpha) = a、 \phi(\beta)= b\\
那么有： \int^b_a f(\phi(x))dx =   \int^\beta _\alpha f(\phi(x))\phi'(x) dx
\end{align*}$$
	也就是变成d phi(t)，同时积分上下限也发生变化（因为积分变量发生了变换，变成了dPhi（x））

广义积分
	$$\begin{align*}
设y=f(x)在[a,+ \infty)上连续，取 b>a \\
若\lim_{ b \to +\infty } \int^b_{a} f(x)dx 存在称此极限为f(x)在[a,+\infty)上的反常积分\\
记作： \int^{+\infty}_{a} f(x)dx
\end{align*}$$
	广义积分性质
		极限存在的情况$$\begin{align*}
\int^{+\infty}_{a} f(x)dx = \lim_{ b \to +\infty } \int^b_{a} f(x)dx\\
如果该极限存在，则称该反常积分收敛\\
当该极限不存在时，则称该反常积分发散
\end{align*}$$
		类似地，推导负无穷的反常积分：
		$$\begin{align*}
设y=f(x)在(-\infty,b]上连续，取 a<b \\
若\lim_{ a \to -\infty } \int^b_{a} f(x)dx 存在称此极限为f(x)在(-\infty,b]上的反常积分\\
记作： \int^{b}_{-\infty} f(x)dx\\
\int^{b}_{-\infty} f(x)dx = \lim_{ a \to -\infty } \int^{b}_{a} f(x)dx\\
如果该极限存在，则称该反常积分收敛\\
当该极限不存在时，则称该反常积分发散
\end{align*}$$
		因此，我们将反常积分问题变成了求极限问题
		例题1$$\begin{align*}
\int^{+\infty}_{0} e^{-t} dt = \lim_{ b \to +\infty } \int^b_{0} e^{-t} dt =  -\lim_{ b \to +\infty } \int^b_{0} e^{-t} d(-t)\\
原式 = - \lim_{ b \to +\infty } (e^{-b}-1) =1 
\end{align*}$$
		例题2
		$$\begin{align*}
\int^{0}_{-\infty} \sin x dx = \lim_{ a \to -\infty } \int^0_{a} \sin x dx =- \lim_{ a \to -\infty }  (1- \cos a)极限不存在
\end{align*}$$
		反常积分的和==**（在R上和在(-∞，+∞)上是等价的）**==
			$$\begin{align*}
设y= f(x)在 R上连续，若反常积分\int^0_{-\infty} f(x)dx和 \int^{+\infty}_{0} f(x)dx都收敛\\
那么上述两反常积分之和为函数f(x)在R上的反常积分\\
记作： \int^{+\infty}_{-\infty} f(x)dx = \int^0_{-\infty} f(x)dx+ \int^{+\infty}_{0} f(x)dx = \lim_{ a \to -\infty } \int^0_{a} f(x)dx + \lim_{ b \to +\infty }   \int^b_{0} f(x)dx\\
若该极限存在，则称反常积分收敛；否则称反常积分发散
\end{align*}$$
		例题：
		$$\begin{align*}
\int^{+\infty}_{-\infty} \frac{1}{1+x^2} dx = \lim_{ b \to -\infty } \int^0_{b} \frac{1}{1+x^2} dx+ \lim_{ a \to +\infty } \int^a_{0} \frac{1}{1+x^2}dx\\
\lim_{ b \to -\infty } (\arctan 0- \arctan b)+ \lim_{ a \to +\infty } (\arctan a- \arctan 0 ) = \frac{\pi}{2}+ \frac{\pi}{2} =\pi
\end{align*}$$
	无界函数的广义积分
		$$\begin{align*}
设 f(x)在(a,b]内连续，而在a的右邻域内无界。\\
取 \epsilon>0,若极限 \lim_{ \epsilon \to 0^+ } \int^b_{a+ \epsilon}f(x)dx存在\\
则称此极限为f(x)在(a,b]上的反常积分,记作:\\
\int^b_{a} f(x)dx = \lim_{ \epsilon \to 0^+ } \int^b_{a+ \epsilon} f(x)dx\\
当该极限存在时，称反常积分收敛；当极限不存在时，称反常积分发散
\end{align*}$$
	对于这个定理的解释
		这里虽然无界，但是我们考察的是(a+epsilon,b],而不是 包含a的区域
		那么又因为==**连续的定义就是由求极限得出来的**==[[函数的连续与间断]] ，那么连续一定有极限
		那么在远离a的区域，有一个a+epsilon让f(x)在(a,b]内连续，不然就与条件矛盾
		![[Pasted image 20250616194930.png]]
	那么，同理：
		$$\begin{align*}
设 f(x)在[a,b)内连续，而在b的左邻域内无界。\\
取 \epsilon>0,若极限 \lim_{ \epsilon \to 0^+ } \int^{b- \epsilon}_{a}f(x)dx存在\\
则称此极限为f(x)在[a,b)上的反常积分,记作:\\
\int^b_{a} f(x)dx = \lim_{ \epsilon \to 0^+ } \int^{b- \epsilon}_{a} f(x)dx\\
当该极限存在时，称反常积分收敛；当极限不存在时，称反常积分发散
\end{align*}$$
	综上：
		$$\begin{align*}
f(x)在[a,b]上除点C(a<c<b )外连续，而在点C的邻域内无界\\
在该情况下，\int^b_{c} f(x)dx 和 \int^c_{a} f(x)dx都收敛，那么定义：\\
\int^b_{a} f(x)dx =  \int^c_{a} f(x)dx + \int^b_{c} f(x)dx =  \lim_{ \epsilon \to 0^+ } \int^{c- \epsilon}_{a} f(x)dx+ \lim_{ \epsilon' \to 0^+ } \int^{b}_{c+ \epsilon'} f(x)dx\\
若该极限存在，则称该积分收敛；否则称该积分发散\\
同时定义:C为瑕点，以上积分称为瑕积分
\end{align*}$$
	例题1：
	$$\begin{align*}
\int^a_{0} \frac{1}{\sqrt{ a^2-x^2 }} dx (a>0)\\
设 x= a\sin t , dx = a\cos t dt, x^2 = a^2 \sin^2 t\\
\frac{1}{a} \int^{\frac{\pi}{2}}_{0} a \sec t \cos t dt = \frac{\pi}{2}
\end{align*}$$
	例题2：
	$$\begin{align*}
\int^3_{0} \frac{1}{(2-x)^2} dx ,在这里 2-x≠0，即x≠2,2是瑕点\\
\int^3_{0} \frac{1}{(2-x)^2} dx =\lim_{ \epsilon \to 0^+ }  \int^{2-\epsilon }_{0} \frac{1}{(2-x)^2} dx+\lim_{ \epsilon \to 0^+ }  \int^3_{2+ \epsilon} \frac{1}{(2-x)^2} dx =\lim_{ \epsilon \to 0^+ }  \frac{1}{\epsilon} - \frac{1}{2} +\frac{1}{\epsilon} -1 发散
\end{align*}$$
