$$\begin{align*}
	若 \int^b_{a} f(x)dx 、 \int^b_{a} g(x)dx 均存在 \\
	\forall 常数 \alpha 、\beta,那么 \int^b_{a} \alpha f(x) + \beta g(x)dx 存在且= \alpha \int^b_{a} f(x)dx + \beta \int^b_{a} g(x)dx 
	\end{align*}$$
		这里我们不规定上下限a、b的大小，也不规定α和β的正负性
		证明：$$\begin{align*}
	① 证明当 b>a时，\int^b_{a} (\alpha f(\xi_{i}) +\beta g(\xi_{i}) ) \Delta x_{i} 这个极限存在且唯一\\
	\end{align*}$$
		要想证明这个，我们先要引入连加号∑的性质：
			$$\begin{align*}
	\sum^n_{i=1} (\alpha u_{i}+ \beta v_{i}) = \alpha \sum^n_{i=1} u_{i} + \beta \sum^n_{i=1} v_{i} \\
	证明：因为连加号，本质上就是很多个加法, 拆出来就可以了\\
	\sum^n_{i=1} (\alpha u_{i}+ \beta v_{i}) = \alpha (u_{1}+u_{2}+\dots+ u_{n} )+ \beta (v_{1}+v_{2}+\dots + v_{n}) = \alpha \sum^n_{i=1} u_{i} + \beta \sum^n_{i=1} v_{i}
	\end{align*}$$
		那么，我们就可以拆分它了：$$\begin{align*}
	当 b>a时，\int^b_{a} (\alpha f(\xi_{i}) +\beta g(\xi_{i}) ) \Delta x_{i} =\lim_{ \lambda \to 0 } \left[  \alpha  \sum^n_{i=1} u_{i} \Delta x_{i}+ \beta  \sum^n_{i=1} v_{i} \Delta x_{i}  \right]\\
	原式 =\lim_{ \lambda \to 0 } \left[ \alpha   \sum^n_{i=1} u_{i} \Delta x_{i}+ \beta \sum^n_{i=1} v_{i} \Delta x_{i} \right]\\
	= \lim_{ \lambda \to 0 }   \alpha   \sum^n_{i=1} u_{i} \Delta x_{i} +  \lim_{ \lambda \to 0 }\beta  \sum^n_{i=1} v_{i} \Delta x_{i} = \int^b_{a} \alpha f(x)dx + \int^b_{a} \beta g(x)dx 
	\end{align*}$$
		证明完之后，和[[不定积分的线性运算法则]]作对比，发现不定积分要求αβ不同时为0
		但是定积分是不做要求的，这是因为α，β同时为0时，0的积分还是0
