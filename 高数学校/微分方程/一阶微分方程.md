形如$$ F(x,y,y')=0或 F\left( x,y, \frac{dy}{dx} \right)=0 $$
称为一阶微分方程

变量已分离的微分方程
	形如$$ g(y)dy = f(x)dx $$
	意思就是纯粹的x表达式在一边，纯粹的y表达式在另一边
	那么求解的时候，直接求不定积分（求原函数）就可以了
	$$ \int g(y)dy = \int f(x)dx +C$$
	+C的原因是因为求不定积分时，原函数都要+C，所以干脆提前加上，求出原函数之后可以理解为都归总到C里面了
	证明：$$\begin{align*}
不妨设 y= y(x),由 g(y)dy = f(x)dx \\
那么 g(y(x))d(y(x)) = f(x)dx ,∵ y是关于x的函数\\
∴ g(y(x)) * y'(x)dx = f(x)dx \implies g(y(x)) * y'(x)= f(x)\\
两边都是关于x的函数且表达式相同，那么它们的原函数难道不同吗？\\
因此：\int g(y(x)) * y'(x)dx = \int f(x) dx +C \implies \int g(y(x)) * d(y(x)) = \int f(x) dx +C\\\end{align*}$$
	这一步的意思就是原来我们还不能从g(y)dy = f(x)dx直接变成 $$\int g(y(x)) * d(y(x)) = \int f(x) dx +C$$
	但现在，∵g(y(x)) * y'(x)= f(x),那么我们就可以放心加∫了：$$\begin{align*}
\int g(y) dy = \int f(x) dx +C
\end{align*}$$
	由上面方程确定y= y(x)是方程的解，且含有一个独立的任意常数，那么该解就是方程的通解
	例：$$\begin{align*}
y^2 dy = x^3 dx \implies \frac{1}{3} y^3 = \frac{1}{4}x^4 +C\\
y^3 = \frac{3}{4} x^4 +C \implies y = \sqrt[3]{ \frac{3}{4} x^4 +3C },可将3C = C_{1}
\end{align*}$$
	对于化出来之后怎么解，可以有多种方式，例如y好解，就把y解出来
	x好解，就把x解出来；都不好解就不解，当成隐函数

变量可分离的微分方程
	第一种形式：形如$$ \frac{dy}{dx} = f(y)g(x) $$的方程，称为变量可分离的微分方程
		但是在求解普通的方程时，除以含有未知数的表达式，容易失去解，因为分母=0无意义，那么，就需要讨论=0和≠0
		==乘以未知数的表达式，容易增解==
			因为*若该表达式在某些点上为 0，乘法就把原方程的信息“抹掉”了，导致原本不满足原方程的解也成了新方程的解。*
		那么对于微分方程来说，在乘除因变量的表达式时，一定要讨论=0或≠0
		但是对于自变量而言就不用，因为乘除关于自变量的表达式，就相当于将定义域缩小了，不影响求函数表达式
			意思就是：$$\begin{align*}
	除以 g(x) 的含义只是“要求我们不在 g(x)=0 的点上去用这个公式”\\
	——也就是说，你必须在每一个 g(x)\neq0 的区间上写出通解。但这并不会“丢掉”某个特定的 y(x) 解\\
	只不过你的显式解在那些使 g(x)=0 的点处失去意义
	\end{align*}$$
			不过x=0上有解的话，你还是要写出来
			如果==**微分方程中看不出那个因变量那个自变量，就都要讨论**==
		回到上式
			$$\begin{align*}
	当 f(y) \not= 0时, \frac{1}{f(y)}dy = g(x)dx\\
	\int \frac{1}{f(y)}dy = \int g(x)dx +C \\
	但是这个解只是 f(y) \not= 0时的通解\\
	当 f(y) = 0时，如果有解，那么y = k_{i},k为常数,i=1,2,3\dots ,m\\
	如果没解，那除因变量表达式就不需要讨论了\\
	可以检验一下， \frac{d(k_{i})}{dx} = f(k_{i})g(x),常数对x求导是0，等式右边的f(y)也是0\\
	那么y = k_{i},k为常数,i=1,2,3\dots ,m；就是这个微分方程的解
	\end{align*}$$
			那么，如果≠0的解和=0的解都求出来了，那么两者合起来就是原方程的通解
	第二种形式
		形如：$$\begin{align*}
M_{1}(x)N_{1}(y)dx + M_{2}(x)N_{2}(y)dy = 0的方程，也是变量可分离的微分方程
\end{align*}$$
		对于这种微分方程，我们一下子还看不出来谁是因变量，谁是自变量，那么在乘除的时候就要讨论=0和≠0了
		因此该方程可以转化为：$$当N_{1}(y)和M_{2}(x)≠0时，\begin{align*}
\frac{N_{2}(y)}{N_{1}(y)}dy = - \frac{M_{1}(x)}{M_{2}(x)}dx \\
那么 \int \frac{N_{2}(y)}{N_{1}(y)}dy = \int - \frac{M_{1}(x)}{M_{2}(x)}dx +C\\
当 N_{1}(y) = 0时的常数解y= k_{i},i=1,2,3\dots ,m \\
当 M_{2}(x) = 0时的常数解x = p_{j}, j=1,2,3\dots,n ；其中k_{i},p_{j}都为常数\\
经检验，都是原方程的解的话,三类合起来就是原方程的通解
\end{align*}$$
	例题：$$\begin{align*}
ydy = x\sqrt{ 1+y^2 }dx\\
y^2 \geq 0,那么 y^2+1 \geq 1, \int \frac{y}{\sqrt{ 1+y^2}}dy = \int xdx +C\\
(1+y^2)^{ \frac{1}{2} }= \frac{1}{2} x^2 +C
\end{align*}$$
	例题：$$\begin{align*}
求方程 e^x \frac{dy}{dx} = \sqrt{ y }(1+y)的通解\\
当y \not= 0或-1时，
\int \frac{1}{\sqrt{ y }(1+y)} dy = \int e^{-x} dx+C \\
2 \int \frac{1}{1+ (\sqrt{ y })^2}d(\sqrt{ y }) = - e^{-x} +C\\
2\arctan \sqrt{ y } = - e^{-x}+C \\
y=-1时，\sqrt{ y }无意义，故舍去;y=0是原方程的解,因为0的导数是0，等号右边也算出为0\\
那么y=0确实是原方程的解\\
那么原方程的通解，就是2\arctan \sqrt{ y } = - e^{-x}+C和y=0
\end{align*}$$
	继续上一题，如果是求特解呢？$$\begin{align*}
y|_{x=x_{0}} =0 的特解\\
2\arctan \sqrt{ y } = - e^{-x}+C回代，C=1\\
那么特解就是：2\arctan \sqrt{ y } = - e^{-x}+1,或 y=0
\end{align*}$$

一阶齐次微分方程
	定义：$$\begin{align*}
形如 \frac{dy}{dx} = f\left(  \frac{y}{x} \right)的方程称为一阶齐次微分方程\\
\end{align*}$$
	齐次就齐次在 y/x,幂的次数是一样的，当然不是说幂的次数一样就可以
	还要满足比的形式，不附带+C，只剩下y/x或 y^2/x^2 
	解法
		这个方程根据前面的方法，例如恒等变形，用变量分离是变不了的
		那么我们就需要用变量代换试一试了
		那么怎么变量代换呢？因为y/x是一个抽象的整体，除此之外也没有哪个更合适去做变量代换了：$$\begin{align*}
由条件知：y=y(x),令 \frac{y}{x} = u,得知u是关于x的函数 \\
y= ux \implies 对x求导, \frac{dy}{dx} = u+ \frac{du}{dx} * x\\
由∵ \frac{dy}{dx} = f(u) = u+ \frac{du}{dx} * x \implies f(u)-u = \frac{du}{dx} *x \\
该方程就通过变量代换变成了变量可分离的微分方程\\
\end{align*}$$
		接下来解变量可分离式的微分方程：$$\begin{align*}
当 f(u)-u \not= 0时, \frac{1}{f(u)-u}du = \frac{1}{x} dx \\
之后，我们两边求不定积分，不过+ \ln C,以便于我们之后取指数 \\
这里+ \ln C还有另外一个目的,+\ln C相当于将任意常数缩小为>0的常数\\
那么，我们去掉对数相关的绝对值时，全部统一>0而不加绝对值\\
那么，当去掉对数时，所有部分都恢复到原来的范围,从而方便我们运算\\
那么可以得到： \ln x = \ln C + \int \frac{1}{f(u)-u} du 
\end{align*}$$
		之后去对数：$$\begin{align*}
x = e^{\ln C} + e^{\int \frac{1}{f(u)-u} du } \\
那么我们就得到了： x = Ce^{\int \frac{1}{f(u)-u} du } \\
上面我们知道，当 f(u)-u =0时，求出常数解 u = k_{i},经检验，是原方程的解\\
回代 u = \frac{y}{x},得到： \frac{y}{x} = k_{i}
\end{align*}$$
		总结一下：$$\begin{align*}
\frac{dy}{dx} = f\left(  \frac{y}{x}  \right)的通解\\
令 \frac{y}{x} = u,那么 x= C e^{\int \frac{1}{f(u)-u} du } ,之后回代 \frac{y}{x} = u\\
那么当 f(u) - u=0时，求出常数解 u= k_{i},即 \frac{y}{x} = k_{i}\\
合起来就是原方程的通解
\end{align*}$$
		注意f(u) = dy/dx
	例题：$$\begin{align*}
解方程： \begin{cases}
x \frac{dy}{dx} = y+ x\tan \frac{y}{x} \\
y|_{x=1} = \frac{\pi}{2}
\end{cases}\\
令 \frac{y}{x} = u,得到： \frac{dy}{dx} = u+ \tan u\implies u+ \frac{du}{dx}x =  \implies \frac{1}{x} dx = \cot u du \\
\ln x  = \ln C + \ln \sin u \implies x = C\sin u,即 x = C \sin \frac{y}{x} \\
那么根据初始条件, 1 = C \sin \frac{\pi}{2} \implies C=1 \\
∵ f(u)-u = \tan u,那么 \tan u = 0,u = k\pi ,k∈ Z\\
即 \frac{y}{x} = k\pi，k∈Z是原方程的解\\
那么将特解的C=1回代得到： x = \sin \frac{y}{x} \\
将 y|_{x=1} = \frac{\pi}{2}回代到 u= k\pi, \frac{\pi}{2} = k\pi显然不成立（因为k∈Z（整数））
\end{align*}$$

一阶线性微分方程
	形如$$\begin{align*}
\frac{dy}{dx} + P(x)y = Q(x)(其中 P(x)和Q(x)都是关于x的函数)\\
在这里自变量是x，因变量是y
\end{align*}$$
	称为一阶线性微分方程
	如果自变量是y，因变量是x呢？$$\begin{align*}
\frac{dx}{dy} + P(y)x = Q(y)
\end{align*}$$
	一阶线性齐次微分方程
		如果在上面的 Q(x),Q(y)恒等于0
		那么称为一阶线性齐次微分方程
		即：$$\begin{align*}
以x为自变量，y为因变量为例，当 Q(x) ≡0时，此时方程为 \frac{dy}{dx} + P(x)y = 0\\
该方程称为一阶线性齐次微分方程
\end{align*}$$
		那么：$$\begin{align*}
当 Q(x) \not≡ 0时，称方程 \frac{dy}{dx} + P(x)y = Q(x)为一阶线性非齐次微分方程
\end{align*}$$
		为什么是线性的
			因为该微分方程是关于y和y的一阶导(y')的线性组合
		我们先求简单的，也就是一阶线性齐次微分方程(属于变量可分离型)
			$$\begin{align*}
\frac{dy}{dx} + P(x)y = 0 \implies dy + P(x)ydx = 0 \implies dy = -P(x)ydx\\
当y ≠0时， \frac{1}{y} dy = -P(x)dx \implies \int \frac{1}{y} dy = -\int P(x) dx + \ln C 
\end{align*}$$
			+ln C的目的是==**为了让等式两边求不定积分时，统一不加绝对值**==
			$$\begin{align*}
\ln y = - \int P(x)dx + \ln C \implies y = Ce^{- \int P(x)dx} \\
当y =0时，经检验，y=0是原方程的解 \\
\end{align*}$$
		那么对于非齐次的呢？怎么恒等变换好像都不起作用
		因此我们需要一种新的方法，常数变易法来解决：
		常数变易法，就是如果Q(x)不为0，那么得出来的常数C的解，会不会是一个函数？
		那么设：$$\begin{align*}
y = u(x)e^{- \int P(x)dx}为原方程的解，要用类似于待定系数法的方式，通过设解，利用等式两边相等求出u(x)\\
是不是解，要带入到原来的方程去检验\\
那么 \frac{dy}{dx} = u'(x)e^{- \int P(x)dx} + u(x)e^{- \int P(x)dx} * -P(x) \\
u'(x)e^{- \int P(x)dx} + u(x)e^{- \int P(x)dx} * -P(x) + P(x)u(x)e^{- \int P(x)dx}= Q(x)(注意这里代入的是非齐次的方程)\\
u'(x)e^{- \int P(x)dx} = Q(x) , u'(x) = Q(x)e^{ \int P(x)dx} \\
那么 对 u'(x)这个等式求不定积分，可以得到：u(x) = \int Q(x)e^{ \int P(x)dx} dx + C\\
那么 y = e^{- \int P(x)dx} (\int Q(x)e^{ \int P(x)dx} dx + C)
\end{align*}$$
		虽然确实是求出u(x)了，但是我最后的目的是要求原方程的通解
		但是，因为得出来的确实只有一个独立常数，而且又是一阶微分方程。那么这个$$ y = e^{- \int P(x)dx} (\int Q(x)e^{ \int P(x)dx} dx + C) $$
		就是原方程的通解
		怎么记忆呢？
			首先齐次方程的通解，然后常数C变成u(x)之后把原来的e^{-∫ P(x)dx}挤出去
			然后开始求u(x)：对 u(x)e^{- ∫P(x)dx} 写成 u'(x)e^{- ∫P(x)dx} = Q(x),然后变成 u'(x)= Q(x)e^{ ∫P(x)dx} ,最后求不定积分得到u(x)
		那么如果自变量是y，因变量是x呢？
			$$ y = e^{-\int P(y)dy}\left(  \int Q(y)e^{\int P(y)dy}dy+C  \right) $$
	例题：$$\begin{align*}
\frac{dy}{dx} - y\cot x = 2x\sin x \\
初始条件： y|_{x= \frac{\pi}{2}} =0 \\
\frac{dy}{dx} + P(x)y = Q(x), P(x) = -\cot x; Q(x) = 2x\sin x\\
y= e^{\int \cot xdx }\left[ \int 2x\sin xe^{-\int \cot xdx} dx +C \right]\\
e^{\int \cot xdx}= C\sin x; e^{- \int \cot xdx} = \frac{1}{\sin x}* C \\
那么 y= C\sin x\left[  \int \frac{2x}{C} dx+C  \right]= (x^2 +C)\sin x\\
代入初始条件： x= \frac{\pi}{2}时 y=0; 0= \frac{(\pi)^2}{4}+C\\
C = - \frac{\pi^2}{4},回代：\left( x^2  - \frac{\pi^2}{4} \right)\sin x
\end{align*}$$
	例题：$$\begin{align*}
\frac{dy}{dx} = \frac{y}{x+y^3} \implies 化成一阶线性微分方程,但是因为 \frac{1}{x+y^3}不是只关于x的函数\\
那么此时，我们需要转换一下角度，将 y看成自变量 \\
用  y = e^{-\int P(y)dy}\left(  \int Q(y)e^{\int P(y)dy}dy+C  \right) , \frac{dx}{dy} +P(y)x = Q(y)\\
若 y \not= 0,那么\frac{dx}{dy} = \frac{x+y^3}{y} \implies \frac{dx}{dy} - \frac{x}{y} = y^2 , P(y)x = - \frac{x}{y},Q(y) = y^2 \\
=e^{\int \frac{1}{y}dy}\left( \int y^2 * e^{- \int \frac{1}{y}dy}dy +C\right) =e^{\ln y +\ln C}\left( \int y^2 * e^{- \ln y- \ln C}dy +C\right)  = Cy \left(  \int y^2 * \frac{1}{Cy}dy +C\right) \\
原式 = Cy\left(  \frac{1}{C} * \left( \frac{1}{2}y^2 +C_{1} \right) +C  \right) = y\left( \frac{1}{2}y^2 +C \right),而且当y=0时，对x求导确实是0\\
而且 y=0时，\frac{y}{x+y^3} = 0，那么y=0也是原方程的通解
\end{align*}$$
伯努利方程
	定义：$$形如 \frac{dy}{dx} + P(x)y = Q(x)y^{a} (a是 \not= 0,1的常数)，称为伯努利方程$$
	当a=0或1时，为一阶线性微分方程
	同时这里的a是常数，也就是负数也可以，小数也可以，无理数也可以，只要不为0或1
	那么怎么求解呢？这里用不了恒等变形，只能用变量代换
	$$\begin{align*}
当 y^a ≠0 ，即y ≠0,即当a>0,且 y ≠0时\\
两边同时乘 \frac{1}{y^a},那么 \frac{1}{y^a} \frac{dy}{dx} + P(x) y^{1-a} = Q(x) \\
∵ \frac{1}{y^a} \frac{dy}{dx} = \frac{y^{-a}dy}{dx} \implies \frac{d \frac{1}{1-a} y^{1-a}}{dx} = \frac{1}{1-a} * \frac{dy^{1-a}}{dx} (a ≠1) \\
那么\frac{1}{1-a} * \frac{dy^{1-a}}{dx} + P(x) y^{1-a} = Q(x) \\
最后,两边同乘(1-a): \frac{dy^{1-a}}{dx} + (1-a)P(x) y^{1-a} = (1-a)Q(x) \\
令 y^{1-a} = z,dz/
\end{align*}$$
