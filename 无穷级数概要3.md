交错级数
	$$\begin{align*}
形如: u_{1}- u_{2}+ u_{3}- u_{4}+\dots + (-1)^{n-1}u_{n} +\dots= \sum^\infty_{n=1} (-1)^{n-1}u_{n}\\
或 -u_{1}+u_{2}-u_{3}+\dots+ (-1)^n u_{n}+\dots = \sum^\infty_{n=1} (-1)^n u_{n}\\
其中u_{n}>0,n=1,2,3\dots
\end{align*}$$
	莱布尼兹定理
		$$\begin{align*}
若交错级数 \sum^\infty_{n=1} (-1)^{n-1}u_{n}满足:\\
①u_{n} \geq u_{n+1}(n=1,2,\dots); ② \lim_{ n \to \infty } u_{n}=0\\
此时交错级数收敛，其和为s \leq u_{1}
\end{align*}$$
		对于①，我们可以用：(注意，这里的方法全部都排除了(-1)这一项)$$\begin{align*}
①作差 : u_{n+1}- u_{n} \leq 0是否成立\\
②作商： \frac{u_{n+1}}{u_{n}} \leq 1 是否成立\\
③导数法： 找一个函数 f(x),使得 f(n)=u_{n}\\
当 x充分大时，f'(x) \leq 0是否成立
\end{align*}$$
		例子
		$$\begin{align*}
\sum^\infty_{n=1} \frac{(-1)^{n-1}}{n}的敛散性\\
\frac{1}{n+1}< \frac{1}{n},条件①满足\\
\lim_{ n \to \infty } \frac{1}{n} =0，根据莱布尼兹判别，此时交错级数收敛
\end{align*}$$
		那么我们可以发现，==**调和级数发散，但是交错调和级数收敛**==
		例子：
		$$\begin{align*}
\sum^\infty_{n=1} \frac{(-1)^n(n+2)}{(n+1)\sqrt{ n }}的敛散性\\
\lim_{ n \to \infty } \frac{n+2}{(n+1)\sqrt{ n }} = \lim_{ n \to \infty } \frac{1}{\sqrt{ n }}+ \frac{1}{(n+1)\sqrt{ n }} = 0\\
\frac{u_{n+1}}{u_{n}}= \frac{n+3}{(n+2)\sqrt{ n+1 }}*  \frac{(n+1)\sqrt{ n }}{(n+2)}= \frac{(n+3)\sqrt{ n }}{(n+2)(n+1)^{ \frac{1}{2}}} 或者变成：\\
\sqrt{ \frac{n}{n+1} }* \frac{(n+3)(n+1)}{(n+2)^2}< 1* \frac{n^2+4n+3}{n^2+4n+4}(<1的数字)<1\\
那么根据莱布尼兹判别，该级数收敛
\end{align*}$$
	绝对收敛和条件收敛
		对于任意项级数：
		$$\begin{align*}
\sum^\infty_{n=1} u_{n} = u_{1}+ u_{2}+\dots+ u_{n} + \dots\\
一般项取绝对值 \sum^\infty_{n=1} |u_{n}| = |u_{1}|+ |u_{2}|+\dots+ |u_{n}| + \dots\\
如果  \sum^\infty_{n=1} |u_{n}| 收敛，则称\sum^\infty_{n=1} u_{n}原级数绝对收敛\\
如果 \sum^\infty_{n=1} |u_{n}| 发散，但是 \sum^\infty_{n=1} u_{n}原级数收敛，则称原级数条件收敛
\end{align*}$$
		绝对收敛定理[[无穷级数概要1]]
			$$\begin{align*}
 \sum^\infty_{n=1} |u_{n}| 收敛，\sum^\infty_{n=1} u_{n}原级数绝对收敛\\
 同时原级数也收敛
\end{align*}$$
			证明：需要用到柯西收敛准则（注：我的证明有缺陷）
				柯西收敛准则
				$$\begin{align*}
\forall 数列 a_{n},其级数\sum^\infty_{n=1} a_{n} 收敛 \leftrightarrow 这个级数的任一一个部分和数列 S_{n}= \sum^\infty_{n=1}a_{n}满足柯西收敛准则\\
即： \forall \xi>0, ∃N_{0}>0,使得\forall n,m >N_{0} ,都有 |S_{n}- S_{m}|= | \sum^\infty_{k=m+1} a_{k}|< \xi\\
用俗话说就是数列的每一项的距离都缩小
\end{align*}$$
				从而我们可以得到：==**一个收敛的数列一定是柯西数列，==**
				**==而一个柯西数列一定收敛**==
				即： ==**是柯西数列 <-> 数列收敛**==

\because \sum^\infty_{n=1} |u_{n}| 收敛,那么根据柯西数列与数列收敛的充要条件\\

其中：\forall \xi>0, ∃N_{0}>0,使得\forall n,m >N_{0},m<n ,都有 |A_{n}- A_{m}|=  \sum^\infty_{k=m+1} |u_{k}|< \xi\\

		达朗贝尔判别在绝对条件收敛下的运用
			[[无穷级数概要2]]，我们提到：
			$$\begin{align*}
若正项级数 \sum^\infty_{n=1} u_{n}= u_{1}+ u_{2}+ \dots+ u_{n}+\dots (u_{n}>0,n=1,2,\dots)\\
的后一项和前一项的比值 \lim_{ n \to \infty }  \frac{u_{n+1}}{u_{n}}= \rho\\
当 \rho >1或\infty 时，级数发散\\
当 \rho <1时，级数收敛\\
当 \rho =1时，无法用该法判断
\end{align*}$$
			那么因为任意项级数需要我们判断|u_n|，而加了||不就是正项级数吗？
			因此：$$\begin{align*}
如果任意项级数 \sum^\infty_{n=1} u_{n}的 \lim_{ n \to \infty } | \frac{u_{n+1}}{u_{n}} |= \rho时\\
那么 当 \rho >1或\infty 时，级数发散且 \lim_{ n \to \infty } u_{n}= \infty\\
当 \rho <1时，级数绝对收敛\\
当 \rho =1时，无法用该法判断
\end{align*}$$
			例题：[[无穷级数概要2]]
			$$\begin{align*}
判断\sum^{\infty}_{n=1} \frac{\sin n\alpha}{n^2}的敛散性\\
这是一个任意项级数，可以用达朗贝尔判别的扩展，也可以用绝对收敛的条件\\
∵ | \frac{\sin n\alpha}{n^2}| >0,那么\sum^{\infty}_{n=1} |\frac{\sin n\alpha}{n^2}|是正项级数\\
用大收敛，小收敛，判断出： | \frac{\sin n\alpha}{n^2}| \leq \frac{1}{n^2}\\
那么p级数当p>1时收敛，那么根据大收敛，小收敛，\sum^{\infty}_{n=1} |\frac{\sin n\alpha}{n^2}|收敛\\
那么原级数绝对收敛，则\sum^{\infty}_{n=1} \frac{\sin n\alpha}{n^2}收敛
\end{align*}$$
			例题：
			$$\begin{align*}
判断\sum^\infty_{n=1} \frac{(-1)^n}{\sqrt{ n }}的敛散性\\
这实际上是一个交错级数,可以用莱布尼兹定理\\
①u_{n} \geq u_{n+1}(n=1,2,\dots); ② \lim_{ n \to \infty } u_{n}=0\\
看一下满不满足这样的条件: \frac{1}{\sqrt{ n }} \geq \frac{1}{\sqrt{ n+1 }}对,同时 \lim_{ n \to \infty } \frac{1}{\sqrt{ n }} =0成立\\
满足莱布尼兹定理，\sum^\infty_{n=1} \frac{(-1)^n}{\sqrt{ n }}收敛
\end{align*}$$
			进一步问下去，这个级数是条件收敛的还是绝对收敛的:
			$$\begin{align*}
既然这个原级数收敛，那么 \sum^\infty_{n=1} |\frac{(-1)^n}{\sqrt{ n }}|= \sum^\infty_{n=1} \frac{1}{\sqrt{ n }}\\
这是一个p级数，p\leq 1发散，那么\sum^\infty_{n=1} |\frac{(-1)^n}{\sqrt{ n }}|发散\\
∴\sum^\infty_{n=1} \frac{(-1)^n}{\sqrt{ n }}条件收敛
\end{align*}$$
			例题：
			$$\begin{align*}
判定 \sum^\infty_{n=1} \frac{x^n}{n}的敛散性\\
首先判断这是什么级数：任意项级数，因为x可>0,也可<0\\
\sum^\infty_{n=1} |\frac{x^n}{n} | 用一下达朗贝尔判别的拓展 ,\lim_{ n \to \infty } |\frac{x^{n+1}}{n+1} * \frac{n}{x^n}|= \lim_{ n \to \infty }  |x|* \left( 1- \frac{1}{n} \right)= |x|\\
当 |x| >1 时，级数发散\\
当 |x| <1时，级数绝对收敛\\
当 |x| =1时，无法用该法判断\\
单独验证|x|=1，x=1时， 原级数为调和级数，发散;x=-1时，为交错调和级数，原级数收敛，绝对值级数发散\\
∴当 |x| >1或x=1 时，级数发散\\
当 |x| <1时，级数绝对收敛\\
当x=-1时，级数条件收敛
\end{align*}$$
			例题：
			$$\begin{align*}
判断\sum^\infty_{n=1} \frac{(-1)^n*n}{3^{n-1}}的敛散性\\
\sum^\infty_{n=1} \frac{(-1)^n*n}{3^{n-1}} = \sum^\infty_{n=1} (-1)^n * \left(  \frac{1}{3}  \right)^{n-1} *n\\
首先先判断是什么级数：交错级数，①u_{n} \geq u_{n+1}(n=1,2,\dots); ② \lim_{ n \to \infty } u_{n}=0\\
看一下满不满足这样的条件:\left(  \frac{1}{3} \right)^{n-1} *n\geq \left( \frac{1}{3} \right)^n * (n+1),在n> \frac{1}{2}满足;但是\lim_{ n \to \infty } \left(  \frac{1}{3} \right)^{n-1} *n = \infty\\
∴用比值审敛： \lim_{ n \to \infty }  \frac{\left( \frac{1}{3} \right)^n * (n+1)}{\left( \frac{1}{3} \right)^{n-1} * n} = \frac{1}{3} * \left( 1+ \frac{1}{n} \right) = \frac{1}{3} <1\\
那么这个级数绝对收敛
\end{align*}$$
