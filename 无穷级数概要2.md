正项级数
	如果常数项无穷级数$$\begin{align*}
\sum^\infty_{i=1} u_{i}= u_{1}+ u_{2}+ u_{3}+\dots +u_{n}+\dots \\
如果 u_{n}\geq 0(n=1,2,3\dots),则称为正项级数
\end{align*}$$
	正项级数收敛的充要条件
		其任意一个部分和数列有界
		必要性：
			因为正项级数收敛时，其任何一个子数列都一定收敛于某个数L
			那么既然收敛于某个数L，那么任意一个子数列（包括其本身）一定有界
		充分性：
			[[单调有界定理]]
			定理的条件可减弱为，数列an在n>N0上单调有界时，则数列an收敛
			因为从已知条件可知：该级数的任意一个部分和数列有界，
				那么我只需要证明这个级数单调就可以了
				$$\begin{align*}
\because u_{n} \geq 0,\forall n都有 S_{n+1}= \sum^{n+1} _{k=1} u_{k}= \sum^n_{k=1}u_{k}+ u_{k+1} \geq \sum^n_{k=1} u_{k}= S_{n}\\
∵ \forall n,都有 S_{n+1}\geq S_{n},那么该数列单调递增
\end{align*}$$
			那么根据单调有界定理，该数列的任意一个部分和数列有极限，那么原级数收敛
	那么，我们现在可以通过级数与级数之间的关系，来得到收敛的结论，就像夹逼定理一样
		差值审敛法：
		$$\begin{align*}
如果正项级数 u_{n}和 v_{n},满足：\\
u_{n} \leq v_{n}(n=1,2,3\dots),实际上就是作差\\
如果 \sum^\infty_{n=1} v_{n} 收敛，那么 \sum^\infty_{n=1} u_{n}一定收敛（单调有界定理）\\
如果 \sum^\infty _{n=1} u_{n}发散，那么 \sum^\infty_{n=1} v_{n}一定发散（逆否命题保真性）
\end{align*}$$
		总结就是==**用一般项来比较，一般项大的，对应的级数收敛，那么小的对应的级数也收敛；小发散，大发散**==
	p级数的敛散性
		$$\begin{align*}
形如：\sum^\infty_{n=1} \frac{1}{n^p},p>0的，我们称为p级数
\end{align*}$$
		现在试讨论其敛散性
		$$\begin{align*}
我们要用比较审敛法来讨论\\
首先我们有调和级数,根据比较审敛法小发散，大发散\\
那么 因为p=1时，该级数为调和级数，发散\\
p>1时， \frac{1}{n^p} < \frac{1}{n} ;当 0<p<1时, \frac{1}{n^p} > \frac{1}{n} \\
那么,当 0<p<1时,\because \frac{1}{n^p} > \frac{1}{n}\\
\therefore 当0<p<1时, \sum^\infty_{n=1} \frac{1}{n^p}发散\\
\end{align*}$$
		那么怎么判断p>1的情况呢？我们可以用积分判别法或柯西凝聚判别
		$$\begin{align*}
\sum^\infty_{n=1} \frac{1}{n^p},p>1\\
现在取 f(x)= \frac{1}{x^p},p>1\\
\int^{+\infty}_{1} \frac{1}{x^p} dx= \int^{+\infty}_{1} x^{-p}dx= \frac{x^{1-p}}{1-p} 从1到+\infty的定积分,\\
但是这个+\infty很难表示，因此我们用个T表示\\
\lim_{ T \to +\infty } \frac{T^{1-p}}{1-p}(x=t时的值) - \frac{1}{1-p},因为p>1,那么T^{1-p}反而是\to 无穷小\\
那么得到值就是 - \frac{1}{1-p}= \frac{1}{p-1}
\end{align*}$$
		那么，既然定积分收敛，根据[[海涅定理]]，
		对于数列而言，对应的函数收敛，其数列也一定收敛
		那么对应的级数：
		$$\begin{align*}
\sum^\infty_{n=1} \frac{1}{n^p},p>1收敛
\end{align*}$$
		==**综上p级数 只有p>1时收敛 , 0<p≤1时发散**==
		拓展结论：
			$$\begin{align*}
\sum^\infty_{n=1} \frac{1}{an},其中 a≠0，根据数乘不变性 \sum^\infty_{n=1} \frac{1}{an}发散
\end{align*}$$
			$$\begin{align*}
\sum^\infty_{n=1} \frac{1}{an+b}，a≠0，b为任意常数，仍然发散，因为在一个级数的前面加上或去掉有限项，级数的敛散性不变
\end{align*}$$
	现在判断：$$\begin{align*}
\sum^\infty_{n=1} \frac{1}{\sqrt{ n(n+1) }}的敛散性\\
\frac{1}{n+1} =\frac{1}{\sqrt{ (n+1)^2 }}  \leq \frac{1}{\sqrt{ n(n+1) }} \\
那么 \sum^\infty_{n=1} \frac{1}{n+1}发散，小发散，大发散\\
那么\sum^{\infty}_{n=1}\frac{1}{\sqrt{ n(n+1) }}发散
\end{align*}$$
	比值审敛法[[无穷小量阶的比较]]
		$$\begin{align*}
	\sum^\infty_{n=1} u_{n}和 \sum^\infty_{n=1}v_{n}都是正项级数\\
	若\lim_{ n \to \infty } \frac{u_{n}}{v_{n}} =0 ，且v_{n}收敛，那么根据无穷小量阶的比较或者“大收敛 \to 小收敛”,u_{n}一定收敛\\
	若 \lim_{ n \to \infty } \frac{u_{n}}{v_{n}}=+ \infty,且v_{n}发散，那么根据小发散 \to 大发散，v_{n}比u_{n}更小，那么u_{n}一定发散\\
	若\lim_{ n \to \infty } \frac{u_{n}}{v_{n}}= l, 0< l< +\infty（常数）,那么\sum^\infty_{n=1} u_{n}和 \sum^\infty_{n=1}v_{n}有相同的敛散性（因为是同阶无穷小，谁也打不败谁）
	\end{align*}$$

		$$\begin{align*}
	判断 \sum^\infty_{n=1} \sin \frac{1}{n} 的敛散性\\
	用比值审敛法, \lim_{ n \to \infty } \frac{\sin \frac{1}{n}}{ \frac{1}{n} } = 1也就是有相同的敛散性\\
	那么 \sum^\infty_{n=1} \frac{1}{n}为调和级数，发散，那么 \sum^\infty_{n=1} \sin \frac{1}{n}也发散
	\end{align*}$$
	达朗贝尔判别法
		$$\begin{align*}
若正项级数 \sum^\infty_{n=1} u_{n}= u_{1}+ u_{2}+ \dots+ u_{n}+\dots (u_{n}>0,n=1,2,\dots)\\
的后一项和前一项的比值 \lim_{ n \to \infty }  \frac{u_{n+1}}{u_{n}}= \rho\\
当 \rho >1或\infty 时，级数发散\\
当 \rho <1时，级数收敛\\
当 \rho =1时，无法用该法判断
\end{align*}$$
		证明：（比值审敛法）
		ρ <1的情况,该级数收敛的证明$$\begin{align*}
 \lim_{ n \to \infty }  \frac{u_{n+1}}{u_{n}}= \rho \\
 设 \rho < k< 1,当n\geq N(一个足够大的常数)时，有 \frac{u_{n+1}}{u_{n}} <k\\
 u_{n+1} < ku_{n} \implies u_{n+2}< ku_{n+1} < k^2u_{n} \implies \dots u_{n+k} <k^qu_{n} \\
 而 \sum^\infty_{n=1} u_{n} \leq u_{n} \sum^\infty_{q=1} k^q,\because k^q为等比级数\\
 根据大收敛\to 小收敛，那么 \sum^\infty_{n=1} u_{n} 收敛
\end{align*}$$
		同理 ρ>1时：
		$$\begin{align*}
\lim_{ n \to \infty }  \frac{u_{n+1}}{u_{n}}= \rho \\
 设 \rho >k> 1,当n\geq M(一个足够大的常数)时，有 \frac{u_{n+1}}{u_{n}} >k\\
 u_{n+1} > ku_{n} \implies u_{n+2}> ku_{n+1} > k^2u_{n} \implies \dots u_{n+k}> k^qu_{n} \\
 而 k>1,\lim_{ q \to \infty } k^q = \infty \\
 根据小发散\to 大发散，那么 \sum^\infty_{n=1} u_{n} 发散
\end{align*}$$
		例题1：$$\begin{align*}
判定级数 \sum^\infty_{n=1} \frac{x^n}{n}(x>0)的敛散性\\
\lim_{ n \to \infty } \frac{\frac{x^{n+1}}{n+1} }{\frac{x^n}{n}} = \lim_{ n \to \infty } x* \frac{n}{n+1} = \lim_{ n \to \infty } x* \left( 1 - \frac{1}{n+1}  \right)= \lim_{ n \to \infty }  x- \frac{x}{n+1}= x- x\lim_{ n \to \infty } \frac{1}{n+1}=x= \rho\\
0<x<1时，级数收敛; x\geq 1时，该级数发散
\end{align*}$$
例题2：
$$\begin{align*}
判定级数 \sum^\infty_{n=1} \frac{1}{(2n-1)2n}的收敛性\\
\lim_{ n \to \infty } \frac{\frac{1}{(2n-1)2n}}{\frac{1}{2n}} = \lim_{ n \to \infty } \frac{1}{2n-1}= 0判断不了\\
\frac{1}{(2n-1)2n} < \frac{1}{n}也得不出什么，需要找到更好的, \frac{1}{n^2}就不错
\end{align*}$$
任何正项级数不是收敛就是发散
只有当你“遇上对的”参照级数——满足上述不等号或极限条件，才能通过比较审敛法下结论；否则一律“无结论”。
无结论是没有办法判定是收敛还是发散的


小结：
	==**对于正项级数，我们可以用差值或者比值审敛法==**
	**==根据大收敛，小收敛；小发散，大发散的级数之间的相互关系进行判别==**
	**==p级数p>1收敛，0<p<=1发散(这与等比级数相反,|q|<1收敛，|q|>=1发散)==**
	**==达朗贝尔判别是根据自身的通项后一项/通项的比值来判别的==**
		**==rho <1收敛，rho >1发散，rho=1无法判别==**
	**==只有当你“遇上对的”参照级数——满足上述不等号或极限条件，才能通过比较审敛法下结论；否则一律“无结论”。==**
	**==且任何正项级数不是收敛就是发散**==
