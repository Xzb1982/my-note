开店的三表
	资产负债表Balance Sheet
		反映某个时点上公司的财务状况
		三要素：资产负债所有者权益
		会计恒等式：资产Asset = 负债Liability+所有者权益Equity
		所有者权益
			你出的钱
		负债
			你借的钱
		那么资产就是你的资金变成资产（例如原料、雇工、固定资产）
		记账
			![[Pasted image 20251115164414.png]]
			资产和负债，我们根据变现时间来分成流动资产和非流动资产
			流动资产就是1年以内就能变现的资产
				货币资金
					公司的现金、银行存款
				有价证券
					买的一些短期的证券（债券、股票）
			非流动资产一年以上才能变现的资产
			负债就是按照偿还时间来划分流动和非流动
			应收账款（Accounts Receivable）A/R
				提供服务/商品后应收取的款项
				应收账款不一定能收回来，这时候就需要坏账计提Allowance for Doubtful Debt
					通常按账龄比例计提
					![[Pasted image 20251115174346.png]]
					时间越后，计提比例越大
				应收账款通常体现的是==**公司对下游渠道的议价能力**==
			预付账款（Prepaid Expenses）
				买商品/服务时，提前支付的订金
				通常体现的是==**公司对上游渠道的议价能力**==
			存货Inventory
				公司购买的原材料、半成品、成品
					原材料:茶叶、水果、牛奶、果酱等
					半成品:茶萃、切好的水果、煮好的珍珠、糖水等
					成品:做好的奶茶
				存货减值准备
					产品发生损毁或过期导致价格下跌
					例:翠花在搬水果的时候，由于不小心，摔坏了2个榴莲，价值300元，奶茶店需要将存货价值下调300元
				生存周期：从生产到变现的过程
				减值
					一般来说很多东西都会发生减值损失
					除了一些可以价值储藏的东西，例如黄金
				盘点存货的时间
					发生卖出时立即做账（永续盘存）
					一段时间做一次账（定期盘存）
				存货卖出计量
					个别计价法（Specific Identification Method）[[存货]]
					先进先出法（First in First out Method，FIFO）
					后进先出法（Last in First out Method，LIFO）
					加权平均法（分为月末一次加权平均和移动加权平均，Weighted Average Cost Method）
					就例如食堂打菜，拿一份鸡排，那么一碟鸡排售出，可以用个别法
					走到前面，是一大锅番茄炒蛋，现在食堂阿姨端出又一锅番茄炒蛋倒里面。那么食堂阿姨给你盛的是原先的，就是先进先出
					如果食堂阿姨给你盛的是新的，就是后进先出
					加权平均就是食堂阿姨把新的旧的一起搅匀，然后盛给你
				![[Pasted image 20251119105526.png]]
					在这个案例里面，我们假设西瓜的价格持续上涨
					按照这个情境，如果你分得清楚每个西瓜，那么使用的时候， 可以用个别法来计算
						那么营业成本就是用掉的，期末存货就是没用掉的
					FIFO就是先买进的，就在==资产负债表上“用掉”，不在意是不是特定的==
					LIFO就正好相反
						这里就可以造成利润的区别，因为不同的存货计量方法算出的利润是不一样的。因为==算出来的成本不一样==
					Weighted Ave
						将每个存货的价值合在一起然后均摊
			固定资产PPE（制造业中比较重要的会计科目）
				折旧Depreciation(基于==“本来就要发生的损失”==)
					直线法Straight line
						![[Pasted image 20251126002405.png]]
					加速折旧法 Double Declining Balance
						![[Pasted image 20251126002540.png]]
						注意，是：“原值-累计折旧”，而且比率的分子是固定为2的
						而且加速折旧到了最后一年要结合残值来考虑：即最后一年的折旧不能超过原值-残值
							![[Pasted image 20251126003223.png]]
					产量法Units of production
						![[Pasted image 20251126003414.png]]
					这里也会导致利润的差异，因为折旧==导致的资产被去掉的部分的量不一样==
				减值（Impairment）（基于无法预见的损失）
					资产带来的经济利益==明显低于预期==
					于是要将固定资产账面价值下调
			无形资产（Intangible Asset）
				公司拥有或者控制的没有实物形态的==可辨认==非货币性资产，包括==专利权、版权、商标权、著作权==等
				例如消费电子公司、医疗器械公司、软件公司、文化传媒公司
				摊销（Amortization）（本质上算是无形资产的“折旧”）
					无形资产也是有时限的，例如专利保护期
					计算方法和折旧相同
				商誉（特殊的无形资产）
					收购产生的
					例如A公司要收购B公司，B公司市价为300万，A给出的收购价一般是≥300万的，特别是业绩比较好的公司
					但是做账的时候就出现了：实际上花的>收购需要的价格，也就是出现了收购溢价（商誉），也就是我认为==**未来你能把溢价赚回来**==
					但是商誉要考虑减值的问题，==你真的能管理好这个被收购的公司吗==
			金融资产
				公司==**不准备短期卖出（即长期持有）**==的一些金融资产
			[[会计计量属性]]
				历史成本Historical Cost
					购买资产时支付的金额
					例如固定资产，买入时，直接用买进来的价格来记账
					后面折旧的时候，第一次折旧就用历史成本（也就是买入值）减去折旧
					==固定资产一般用历史成本计量==
				公允价值Fair Value（可以视作市价）
					买卖双方在公平交易的条件下所确定的价格
					也可以理解为：==在当前市场、在正常公平交易条件下，理性的买方和卖方大概率愿意成交的价格。==
					例如股票，股票波动带来的资产负债表的损益计入公允价值变动损益
				摊余成本Amortized Cost
					和发债券相关
				现行成本Current Cost(又称重置成本)
					例如我今天把固定资产重新买一遍要多少
				可变现净值Realizable Value
					也就是==将我现在有的资产现在拿去市场卖掉值多少==
				现值Present Value
					未来现金流贴现的价值
			负债
				有息负债Debt：短期借款、长期借款、应付债券
				无息负债
				流动负债
					一年内或者一个经营周期内需要偿还的债务
					短期借款
						为了维持正常经营向银行或其他金融机构借入的一年内还的借款
					应付账款Accounts Payable
						拿了货没给钱
						这是用于衡量公司==对上游供应商的议价能力==的
						应付账款的越多，信誉越好，但是风险也比较大
						应付账款越少，可能就不知名，但是风险小
					预收账款Unearned Revenue
						先拿钱后给货
						可以衡量==公司对下游客户的议价能力==
						也可以间接衡量量产能力和订单之间的差距
					应计负债类Accured Expenses
						==应付职工薪酬、应交税费、应计利息==
						==已发生但尚未支付==的负债
						![[Pasted image 20251126141340.png]]
				[[非流动负债]]
					长期借款
					应付债券
			权益类
				股本/注册资本(Capital)
					指的是==股东在公司中所占的权益==
					例如：小明(25)、小红(15)、翠花(10)合伙投入的50万启动资金，按照1元1股计量，总计50万股，小明占50%，小红占30%，翠花占20%
				那么当生意进行得如火如荼时，就不止1元1股了
				资本公积(Additional-paid-in Capital)
					==超出股价1元==的部分
						假设经过一年的经营，小明奶茶店已经蒸蒸日上，小黑也想插一腿，入股30万元并获得9.1%的股份
						新股本:50+5=55万（这是商定出来的，30万中5万作为新股本，25万作为资本公积）
						资本公积:25万
						新持股比例:小明(45.45%)、小红(27.27%)、翠花(18.18%)、小黑(9.1%)（==**原股东的股份被稀释**==）
				一般来说这两个科目是同时出现的
				留存收益(Retained Earning)
					公司未分配的利润
					用于扩大再生产、维修设备等
					例如净利润-分红得到未分配利润，这些未分配利润可以拿去修设备
			资产负债表透露了什么？
				资产端：钱被拿去做什么了？
				负债端：企业的钱从哪来
				这里首先要介绍一下复式记账
					其实就是一笔交易，影响两个科目
					例如发行10万股票，那么股本增加，货币资金也增加
					又如买价值5万的设备，资产端固定资产增加，银行存款/库存现金减少
				股东是公司里最后一个拿去自己利益的人
					这体现了剩余索取权的概念
					例如奶茶店先给银行支付利息、给供应商支付货款、给员工发完工资、给税务局交税，剩下的才属于股东
					也就是==**先还债、交税，再分红**==
					而经营公司最终的目的是帮股东赚钱
					资产负债表能帮助股东==了解公司的财务状况==
		利润表Income Statement
			用于反映某个时间段公司的==财务表现==（动态的）
			两要素
				收入Income = 主营收入+其他业务收入+利得（正差价，又称为营业外收入）
					例如奶茶店既卖奶茶，又卖包装盒，同时又把之前的一个机器以高于买入价卖掉了
				费用Expense = 主营成本+其他业务成本+损失（负差价，又称为营业外支出）
					损失就例如那些罚款
				收入-费用=净利润Net Income
				![[Pasted image 20251128133346.png]]
				营业收入/主营与其他业务收入Revenue
					R = P* Q，要么提价要么卖更多，例如==降价提升总量，或者卖奶茶同时卖面包==
				营业成本/主营与其他业务成本Cost of Goods Sold,COGS
					包括料工费：人工、原料、制造费用（水电费、折旧）、运费等
					毛利润Gross Profit（行业对比的重要指标）：主营+其他业务收入-（主营+其他业务成本）
						毛利率 = 毛利润/营业收入
						例如奢侈品、医美/医疗行业（接近90%）
				营业税金及附加Business Tax and Surcharges
					==不包含所得税的税费==
					营业税、消费税、城市维护建设税、资源税、教育费附加等
				费用/期间费用Expenses
					销售费用Selling Expenses
						在销售过程中发生的费用
						例如销售人员的工资、销售所产生的差旅费、广告费
					管理费用Administrative Expenses
						公司行政部门==为组织和管理生产经营活动==而发生的各种费用
						管理人员的工资和福利、差旅费
					研发费用(Research and Development Expense)
						研究与开发过程中发生的各项的费用
						例:小明尝试创新口味，例如将榴莲、臭豆腐、酸笋进行了混合榨汁，共计支出180元
					财务费用(Financing Expense)
						筹集生产经营所需资金（向银行借款）等而发生的费用
						例:小明从银行借款5万元，利率5%，支付的2500元利息
				公允价值变动收益(Gains on the Changes in the Fair Value)
					金融资产的价格变动
					例如小明炒股(短线)赚了100元（==但是还没卖掉==）
				投资收益(Investment Income)
					对外投资取得的利润、分红、利息减去投资损失后的净收益
					例:小明买入中国平安股票一年获得了300元的分红或者==卖出了股票==
				资产减值损失(Loss of Assets Impairment)
					存货、固定资产、无形资产等发生的减值
					翠花摔坏榴莲及碎冰机，导致了700元损失
				信用减值损失(Loss of Credit Impairment)
					==**应收账款**==收不回导致的坏账损失
					原先小明的朋友小白一次性买了10杯奶茶，由于没带钱，小白打了张150元的欠条，约定10日后结清，但是过了2个月都没有来结账，小明只好把这150元全部计提为坏账，同时拉黑了小白
				==营业利润Operating Profit==
					==**毛利-税金及附加-期间费用**==
				营业外收入(Other Income)
					主营、其他业务之外的收入
					包括:固定资产处置利得(Gain)、政府补贴、出售无形资产收益、债务重组利得等
						例如京东方将电视价格打下来，政府补贴占了很大的功劳
				营业外支出(Other Expense)
					主营业务之外的支出
					包括:固定资产处置损失、政府罚款、出售无形资产损失、债务重组损失等
					例:翠花因为随手在奶茶店门口丢垃圾，被城管罚款100元
				==利润总额Earnings Before Tax（EBT）==
					==营业利润-营业外支出+营业外收入==
					==**EBIT = 利润总额+财务费用**==，这被称为息税前利润（即利息和税前利润）
					EBITDA = 利润总额 + 财务费用 + 折旧（Depreciation） + 摊销（无形资产的折旧就叫摊销Amortization）
				净利润Net Income(NI)

		做账的权责发生制
			相对的是收付实现制：收到/支出钱就记一次账
			但是权责发生制则不同，要事情发生了才记账
				例如确认收入但钱还没收到：应收账款
				收到钱但是不一定能确认收入：预收账款
				发生费用不一定付了钱：==应计费用（应付职工薪酬等应付未付款项）==
				付了钱但还没产生费用：预付账款
			例如小明要进茶叶，在淘宝上进货
				买了茶叶之后，钱先到支付宝，支付宝通知茶叶店发货，确认收货了才转给茶叶店
				那么在茶叶店肯发货的那一时刻起，茶叶店可以确认收入
					发货之后，==货物的所有权发生了转移==

		三张表的关系
			资产负债表是静态的，一年度的资产负债表，由期初的资产负债表和期末的资产负债表组成
			利润表是动态的，反映的是一年之中的盈利状况
			利润表的收入-费用 = 毛利 =》毛利-期间费用-税金及附加 = 营业利润
			营业利润 - 营业外支出+营业外收入 = 利润总额
			利润总额 - 所得税 = 净利润
			将净利润 - 分红 = 留在企业的未分配利润（留存收益） = 盈余公积+未分配利润
			而这里和==**期末的资产负债表是联系起来的**==
			现金流量表中，==当期现金流 = 期末现金 - 期初现金==
			==期末与期初现金都在资产负债表体现==
			总结起来就是：Beginning + Addition - Substraction  = Ending
