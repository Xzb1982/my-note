货币是什么？
	**货币＝在某一制度/清算网络内，被广泛接受、以共同记账单位计价、并能“最终清偿”债务的可转让名义权利。**  
	（关键词：记账单位＋广泛可接受＋最终清偿力）
	更详细地：一种由国家—银行—支付网络共同背书的“可转让清偿权”，它以统一的记账单位把经济体里的价格与债务连接起来，并在清算时终结债务关系。

货币的核心属性
	- **记账单位**：所有价格、债务都用它来标价。
	- **交换媒介/流动性**：日常交易里被多数人愿意接收。
	- **最终清偿力**：到这一步，债权债务关系“画句号”，不需要再找第三方兑付。
	例如支票、存款也算货币

货币的四大职能
	价值尺度
	流通手段
	价值储藏
	支付手段

货币材料要支撑货币的三/四大功能（交换媒介、计价单位、价值储藏、延期支付），就必须把几类**交易成本**压低：匹配成本、验证成本、找零成本、搬运成本、价格波动风险、数量错配风险。

价值稳定性（压低价格波动风险）
	- **为何必要**：若购买力剧烈波动，没人愿意持有它作“价值储藏”，用它做“计价单位”也会频繁改价，合同无法写清。
	- **缺失后果**：只剩“即时结算的以物易物式媒介”，信用合同崩坏（延期支付溢价暴涨）。

普遍接受性
	- **为何必要**：货币的交换媒介地位是一个**网络效应的均衡**：人人相信他人会收，才愿意收。法律强制力、赎回承诺、历史惯性都会把体系推向“接受均衡”。
	- **缺失后果**：即便内在材质再好，也停留在“小圈子代券”，无法跨市场通用。

可分割性（降低找零/粒度成本）
	- **为何必要**：不同交易额需要不同面额；可分割才能精准定价、顺畅找零。
	- **缺失后果**：只能处理“大额整块交易”（如牛、布匹），日常小额交换被迫赊欠或改用别的媒介。

易于辨认和携带（压低验证与搬运成本）
	- **为何必要**：交易现场必须**快核验、难伪造、好携带**。越容易真伪判断、越轻便，越能提高流通速度。
	- **缺失后果**：验证耗时、争议频发，或运输/保管成本过高（巨石钱、笨重金属在远距交易吃亏）。

供给富于弹性（缓解数量错配风险）
	- **为何必要**：经济体对交易媒介的需求会随季节、金融环境、产出增长波动。**适度的供给弹性**让货币数量能配合名义交易需求，避免因为“钱太少/太多”把价格水平或产出推离稳态。
	- **缺失后果**：供给刚性→易通缩、挤兑与利率飙升；过度弹性→通胀与信用失序。

- 价值稳定性 → 价格波动风险 ↓ → 价值储藏 / 计价单位 / 延期支付
- 普遍接受性 → 匹配/协调成本 ↓ → 交换媒介（网络外部性成均衡）
- 可分割性 → 找零成本 ↓ → 交换媒介 / 计价单位（价格精度）
- 易辨认携带 → 验证与搬运成本 ↓ → 交换媒介（高周转、低纠纷）
- 供给弹性 → 数量错配成本 ↓ → 宏观稳定（反脉冲/顺增长），反向支撑①

货币和通货
	通货是处于流通中的现实货币
	货币>通货

货币与财富
	财富由各种资产构成，包括实物资产和金融资产
	财富>货币

货币与收入
	收入是某一时间单位的货币额，是货币的流量概念
	价值稳定性 → 价格波动风险 ↓ → 价值储藏 / 计价单位 / 延期支付

货币支付体系的演变
	实物货币
		价值稳定、普遍接受、难于分割但不便携带与辨认、供给缺乏弹性
	金属货币
		包括称量货币和铸币
		称量货币不便携带、铸币

货币的第二重定义
	基于想象的价值
		一方面发行者权威，另一方面被统治者相信其价值
	从而这是社会信用体系中，最有效的媒介
