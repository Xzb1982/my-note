经营比率
	衡量公司的经营效率
	周转率
		周转天数 =  365/周转率
		存货
			存货周转率（账上的货销售出去的效率）：
			$$\begin{align*}
	存货周转率 = \frac{营业成本}{平均存货}，其中平均存货 = \frac{期初存货+期末存货}{2}
	\end{align*}$$
			因为营业成本是income statement的科目，反映的是整个时间段
			但是存货是balance sheet的科目，反映的是时间点
			一个产品的成本，==由料工费等汇总到一个产品，即一个存货==
			存货被卖出时至少要高于营业成本，==从而被卖出时，产品所有权转移确认为收入==
			存货周转天数：
			$$\begin{align*}
	存货周转天数 = \frac{365}{存货周转率}
	\end{align*}$$
			一般来说存货周转天数越小越好，存货周转率越大越好
		应收账款
			应收账款周转率：
			$$\begin{align*}
应收账款周转率 = \frac{业务收入}{平均应收账款}
\end{align*}$$
			你得到的收入要么收到钱，要么还被人欠着（应收账款）
			如果应收账款周转率>1，==例如10:2，那么10份收入才产生2份应收款，说明回款多==
			应收账款周转天数：$$ 应收账款周转天数 = \frac{365}{应收账款周转率} $$
			一般来说应收账周转天数越小越好，应收账周转率越大越好
		应付账款
			应付账款周转率：$$\begin{align*}
应付账款周转率 = \frac{购买量}{平均应付账款}\\
其中购买量 = 期初存货+本期增加存货 - 本期减少存货 = 期末存货
\end{align*}$$
			应付账款周转天数：$$应付账款周转天数 = \frac{365}{应付账款周转率}   $$
		经营周期Opreating Cycle
			卖货时间与收钱时间：因为卖货要么收到现钱要么先被别人欠
			所以一个产品的卖货时间+收钱时间 = 一个经营周期
			卖货时间：==**从原料被卖出到生产再到销售**==
			收钱时间：客户付款到销售端收款
			还钱时间：销售端还款到供应端收款
			净经营周期 = ==卖+收-还钱时间==，因为钱越多越好，收款越多越好，但是还钱是需要付钱的，那么减去付钱的部分才是净经营现金流
			每个行业的净经营周期是不一样的，例如快消品行业的净经营很短，但是一些污水处理的可能净经营很长
		总资产周转率
			衡量每单位资产的创造收入能力
			这是==**净资产收益率ROE的组成部分**==
			$$  总资产周转率 = \frac{业务收入}{平均总资产}$$

流动性比率（衡量短期偿债能力）
	流动比率
		$$流动比率 = \frac{流动资产}{流动负债}$$
		这个值应当尽量>1，这意味着==流动资产比流动负债要大==，至少可以资产抵掉债务
	速动比率
		$$速动比率 = \frac{货币资金+交易性金融资产+应收账款}{流动负债}$$
		一般来说现金越多越好，但是==如果太多现金不用，那么有可能说明这个公司达到了发展的瓶颈期==
		就例如海天味业和宁德时代，海天味业不会很大地扩张产能，市场饱和度高；宁德时代就需要大量的钱来扩产能，需要大量现金
	现金比率
		$$现金比率 = \frac{货币资金+交易性金融资产}{流动负债}$$
	这三个比率>1是好事，代表有现金，能抗风险

偿债比率（衡量长期偿债能力）
	负债率
		有息负债/总负债
		==**一般关注负债比率都是关注有息负债比率**==
		负债权益比率：$$负债权益比率 = \frac{总有息负债}{总所有者权益}$$
		负债资产比率：
		$$负债资产比率 = \frac{总有息负债}{总资产}$$
		权益乘数比率/杠杆倍数（ROE的又一个组成部分）：
		$$权益乘数比率 = \frac{平均总资产}{平均总权益}$$
		日常里面，买房和买车会用到杠杆，例如1000万的房子，先付300万首付，银行贷700万附带利息[[权益投资]]
			在这里的权益就是你出的钱300万，1000/300 = 杠杆乘数
		保证倍数Coverage Ratio
			利息保障倍数比率：
			$$利息保障倍数比率 = \frac{EBIT(利润总额+财务费用)}{利息}$$
			EBIT是息税前利润
			==有些公司的财务费用是负值，没有利息支出但有利息收入==
			==越小越好，负值为无压力==

盈利比率
	第一条线：收入到利润
		毛利率：$$\frac{毛利(营业收入-营业成本)}{业务收入}$$
		可用于对比行业间的毛利率高低，例如养猪的牧原股份、正邦科技、天邦
		猪价在非洲猪瘟时，需求高，猪价贵，毛利率上升；牧原股份是自繁自养
		正邦科技、天邦是自养猪+对外买猪仔
		营业利润率：$$营业利润率 = \frac{营业利润}{业务收入}$$
		净利润率：$$净利润率 = \frac{净利润}{业务收入}$$
	第二条线：资本到利润
		ROA总资产收益率：$$ROA = \frac{净利润}{平均资产}$$
		==资产里面有多少转化成净利润，而且因为资产是资产负债表中的科目，因此求平均转换成时间段数据==
		但是ROA有个问题，因为资产 = 负债+所有者权益；==**财务费用反映要还债的部分、净利润反映要分红的部分**==；但是作为投资者更需要净利润比上权益，也就是==**我投入的钱有多少变成了我自己可以分红的那部分利润**==
		ROE Return On Equity净资产收益率：$$ROE = \frac{净利润}{平均权益}$$
		这意味着，==股东投入的钱有多少转化成了净利润，也就是股东分红上限==
		ROE越多越好，一般==经营周期（卖货时间+收钱时间）==越短，ROE能更大
		如果将==时间拉长，如果ROE能保持在20%，那么这个公司确实不错==
		例如茅台、格力电器、双汇、海天、恒瑞医药、海康威视
			ROE杜邦三分法（ROE的三驾马车）：
			$$\begin{align*}
ROE = 净利润率 \times 资产周转率 \times 权益乘数\\
= \frac{净利润}{业务收入} \times \frac{业务收入}{平均总资产} \times \frac{平均总资产}{平均总权益}
\end{align*}$$
			①高利润率：这意味着==**高附加值，要差异化、去同质化**==
				装修、创新（原料）
			②资产周转率（翻桌率）：==销售速度快，控制成本==，==**以更低的价格把产品快速卖出去**==
				例如双汇（渠道优势）
			③杠杆（借钱）：借得越多赚得越多
				银行赚钱途径：息差
				==地产、保险、券商也需要杠杆来赚钱==
				例如地产：买地，盖房，抵押给银行然后进一步融资
				其他行业杠杆越低越好
