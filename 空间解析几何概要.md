多元函数的无条件极值
	仿照一元函数，凡能使一阶偏导数同时为零的点，均称为函数的驻点.
	即，极值点一定是驻点，但驻点不一定是极值点
	那么怎么求出极值点呢？
		设函数f在点(x0,y0)的某领域内连续，而且有一阶和二阶连续偏导数
		先令对x的一阶导，对y的一阶导都等于0，求出对应x,y的值
		设对x的二阶导fxx=A，混合偏导fxy = B，对y的二阶导fyy=C
		那么在 AC- B^2 >0 时，极值存在，A<0,有极大值；A>0有极小值
		AC-B^2 <0时，极值不存在
		AC-B^2 = 0,需要另做讨论
		例题：
		$$\begin{align*}
求 F(x,y) = x^3-y^3 + 3x^2 + 3y^2-9x的极值\\
\frac{\partial F}{\partial x} = 3x^2+6x-9 = 0 \implies x= -3或1\\
\frac{\partial F}{\partial y} = -3y^2 +6y =0 \implies y=0或2\\
(-3,0),(-3,2),(1,0),(1,2) ;f_{x x}= 6x+6=A; f_{xy} = 0=B; f_{yy} = -6y+6=C\\
经检验 (-3,2),AC>0,A<0,x=-3为极大值,(1,0),AC>0,A>0,x=1为极小值\\
之后代入求y
\end{align*}$$

二重积分
	和定积分类似，只不过变成了两个变量
	因此，定积分和二重积分我们可以共同地考虑
		本质：都是和形式极限，可以通过分割、求和、取极限来求出
		二重积分的概念
			$$\begin{align*}
设 f(x,y) 是有界闭区域D上的有界函数，\\
将闭区域D任意分成n个小闭区域（有界）\Delta \sigma_{1},\Delta \sigma_{2},\dots,\Delta \sigma_{n}\\
其中\Delta \sigma_{i} 表示第i个小闭区域，也表示它的面积\\
在每个\Delta \sigma_{i}任取一点( \xi_{i},\eta_{i} ),作乘积 f( \xi_{i},\eta_{i} )\Delta \sigma_{i}( i=1,2,3\dots n)\\
并做和： \sum^n_{i=1} f( \xi_{i},\eta_{i} )\Delta \sigma_{i}
\end{align*}$$
			做出和形式之后，就开始取极限
			于是我们定义：
			$$\begin{align*}
如果当各小闭区域的直径中的最大值\lambda 趋近于零时\\
这和式的极限存在，则称此极限为函数f(x,y)在有界闭区域D上的二重积分，记为：\\
\iint_{D} f(x,y)d\sigma = \lim_{ \lambda \to 0 } \sum^n_{i=1} f( \xi_{i},\eta_{i} )\Delta \sigma_{i}
\end{align*}$$

			二重积分的存在性定理
				当f(x,y)在有界闭区域D上连续时，二重积分必存在
				在连续的情况下，二重积分仅与==**被积函数和积分区域**==有关
			二重积分的几何意义
				当f(x,y)>0时，∬是柱体的体积
				当f(x,y)<0时，∬是柱体体积的负值
