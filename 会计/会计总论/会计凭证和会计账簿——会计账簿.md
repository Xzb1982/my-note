这里直接跳到会计账簿的分类
	按用途
		序时账簿(按照时间顺序来登)
			例如库存现金日记账，银行存款日记账
		分类账
			例如总分类账和明细分类账
		备查账簿
			对某些在分类账或序时账未能记载或登记不全的经济业务进行补充登记的账簿
			例如`租入固定资产登记簿，代替商品物资登记簿`
	按账页格式
		==三栏式账簿（日记账，总账）==
		![[Pasted image 20250420153843.png]]
		==多栏式账簿（收入、成本、费用明细账）==
		是对三栏式账簿的细化，将借（成本或费用类）贷（收入类）某一方放大
		![[Pasted image 20250420153856.png]]
		数量金额式账簿（原材料，库存商品明细账）
		将三栏式细化成数量、单价、金额
		![[Pasted image 20250420154409.png]]
	按外形特征
		订本式
			例如总分类账和明细分类账
		活页式
			明细分类账
		卡片式
			对固定资产核算用卡片式
	按详细程度
		总分类账
		明细分类账

会计账簿的登记
	![[Pasted image 20250420155435.png]]
	注意，明细账有总页次和分户页次，例如说应收账款是在总页次第五页，那么假设应收账款有2页，那么分户页次就有两页
	会计账簿的核对
		账证核对，账账核对，账实核对，账表核对
		![[Pasted image 20250420155942.png]]
		账可以和凭证和报表核对，也可以和仓库或者其他地方的实有数进行核对
		而账簿与账簿之间也可以进行核对
			![[Pasted image 20250420160204.png]]
			账证核对就是和原始凭证，记账凭证核对
			账账核对是重点
				账簿的内部就是总账、日记账和明细账
				==**①总账的核对就是核对余额，根据资产=负债+所有者权益   [[试算平衡]]中的余额试算平衡法**==
				还有本期总账发生额，通过发生额的试算平衡法来处理
				②总账和明细账要核对，就是==**根据总账各账户的期末余额=其对应的明细账的期末余额之和**==
				![[Pasted image 20250420161308.png]]
				③总账和日记账的核对具体表现为：
				==**库存现金总账的期末余额=库存现金日记账的期末余额==**
				**==银行存款总账的期末余额=银行存款日记账的期末余额**==
				④会计机构的财产物资明细账和财产物资保管和使用部门的有关明细账核对
				==**例如会计部门有明细账，物资保管部门有台账，那么两者可以相互核对**==
			账实核对
				`现金日记账的账面余额和现金实际库存数`
				`银行存款日记账的账面余额和银行对账单（银行会定期给企业发对账单）`
				`各种财物的明细账账面余额与财务实存数`
				`各种应收应付明细账账面余额与有关债务债权单位或者个人进行核对`

会计账簿的结账
	结账的主要工作
		①结清各种损益类账户，结转并计算本期利润
		![[Pasted image 20250420170400.png]]
		结转需要编写分录，但是不需要附原始凭证（因为结转不是新的经济业务发生）
		②结出各资产、负债和所有者权益账户的本期发生额合计和期末余额
		例如资产账户的结转不需要写分录，只需要在账户内进行加减运算结余即可
		![[Pasted image 20250420170806.png]]
	结账的要点
		![[Pasted image 20250420170920.png]]
		其实就是先画一道红线提醒不要再往下写，紧接着在第一道下面写本月合计，再划一道结束本月合计
		对于12月份，我们这样处理：
		![[Pasted image 20250420171252.png]]
		==**封年的时候要画双红线==**
		**==但如果只是1-10月，那么只划单红线即可**==
	关于结转下年
		年度终了，要把各账户的余额结转到下一会计年度;并在摘要栏注明“结转下年”
		在下一会计年度新建有关会计账簿的第一行余额栏内填“上年结转”字样 ，并写上年结转的余额，并在摘要栏注明”上年结转“
