外汇期货
	交易双方以约定的币种、金额以及汇率，在未来某一约定时间进行交割的标准化合约
	例如3000元/吨，10t，大豆
		这个3000元/t可以转换成汇率，10t转换成数量，大豆可以理解成外汇
		现在1$换成7￥，数量是10000，大豆是美元
	外汇期货交易不仅给避险者提供了有效的套保方式
		例如我作为一个进口商，未来要支付一笔3000万$的货款，担心美元升值
		如果怕涨，就买入$期货
	外汇期货的卖出套保
		例如出口商，未来要收到3000万$的货款，担心汇率下跌，于是做空 $期货
		适用于外汇期货做空套保的情形
			1.持有外汇资产者，==**担心未来货币贬值**==
			2.出口商和从事国际业务的银行预计未来某一时间将会得到一笔外汇，为了==**避免外汇汇率下跌**==造成损失
		案例
			某美国投资者发现中国的r>美元r，于是决定购买630万人民币以获取高息，计划投资3个月，但又担心投资期间美元兑人民币升值。为规避人民币贬值的风险，该投资者利用芝加哥商业交易所外汇期货市场进行==**卖出套期保值**==，每手人民币期货合约为10万美元。
			操作如下表：

|     | 即期($/￥)采用直接标价          | 期货(￥/$)采用间接标价           |
| --- | ---------------------- | ----------------------- |
| 开仓  | 6.2974(630万￥=100.04万$) | 做空0.1493(10手=100万$)     |
| 平仓  | 6.2985(630万￥=100.02万$) | 做多0.1471                |
| 盈亏  | -0.02万$                | +0.22万$（0.0022* 100万 $） |
			这里的630￥除上汇率得到结果；左边列需要先算成美元再说
	外汇期货的买入套保
		适合做买入套保的情形
			(一)外汇短期负债者==**担心未来货币升值**==
			(二)国际贸易中的进口商==**担心付汇时外汇汇率上升造成损失**==
		案例
			6.1，美国进口商预计3个月后，需要支付进口货款2.5亿日元
			当前的即期利率为$/JPY = 119.65
			为避免3个月后日元升值，因此在CME里面做多20手JPY期货合约，每手JPY期货合约代表1250万JPY
			操作如下：

|     | 即期($/JPY)采用直接标价                           | 期货(JPY/$)采用间接标价                              |
| --- | ----------------------------------------- | -------------------------------------------- |
| 开仓  | 119.65                                    | 做多0.007957(相当于USD/JPY=125.68)                |
| 平仓  | 117.25                                    | 做空0.008126(相当于USD/JPY=123.06)                |
| 盈亏  | -42768.67（(1/119.65-1/117.25)* 250000000） | +42250((0.007957-0.008126) * 20 *  12500000) |
	外汇期货交叉套保（还没考过相关计算）
		对于一些币种之间没有相应的期货合约的，就需要通过一个中间货币作为桥梁，进行套保
		交叉套保把握的关键
			1.==**只有相关程度高的品种，**==才是为手中持有的现汇进行保值的适当工具（例如至少要同涨同跌）
			2.正确==**调整期货合约的数量**==，使其与被保值对象相匹配
	外汇期货套利交易
		外汇期货套利交易是指交易者根据不同市场、期限或品种间的理论性价差的==**异常波动**==，==**同时买进和卖出两种相关的外汇期货合约**==，以期价差朝有利方向变化后将手中合约同时对冲平仓而获利的交易行为。
	外汇期现交易
		通过==**做空被高估的外汇期货合约/现货**==，同时==**做多被低估的外汇期货合约/现货**==的方式来达到获利的目的。
		在实际情况中，因为交易成本和冲击成本因素的存在，会影响套利策略的实施
			例如大单买入/大单卖出造成的价格冲击
		案例
			在3月1日，交易者发现美元兑人民币的现货价格为1$=6.1130￥
			而6月份的 $/￥的期货价格为6.1190；期现价差为60个点。
			同时，交易者==认为==6月份的外汇期货理论价格为6.1160，无套利区间大概为6.1150-6.1170，交易者判断当前的外汇期货价格>无套利区间，==**期现价差将会缩小。**==
			因此交易者卖出10手6月份的美元/人民币期货，并买入相应金额的现货。4月1日，现货和期货价格分别变为6.1140和6.1180，价差缩小到40个点。交易者同时将现货和期货平仓，从而完成外汇期现套利交易。
			操作如下表所示：

|     | 现货            | 期货6月    |
| --- | ------------- | ------- |
| 3.1 | 多$/￥ = 6.1130 | 空6.1190 |
| 4.1 | 空6.1140       | 多6.1180 |
| 盈亏  | +0.0010       | +0.0010 |
			最后：0.002* 10* 100000
	外汇期货的跨期套利
		是指交易者同时买入/卖出相同品种不同交割月份的外汇期货合约，以期合约间价差朝有利方向发展后平仓获利

	外汇期货交割
		在外汇期货合约中，既有实物交割，也有现金交割
		CME的大部分合约都采用实物交割方式
		CME中的==**巴西货币雷亚尔期货合约**==和纽交所和伦敦国际金融期货交易所的==**欧元期货合约**==采取了现金交割制度。
			为啥外汇也分现金和实物交割？
			就例如欧元期货合约，实物交割是直接给€，现金交割是欧元汇兑美元
		目前国内没有外汇期货，==**外汇交易很大程度上由银行间外汇市场来支撑**==
	外汇保证金交易
		外汇现货交易的类型
			实盘交易
				持有货币/货币等价物（如现金和旅游支票）的双方进行交换
				例如==**持有￥去银行换取外币**==
			外汇保证金交易（是一种场外交易）
				由期货保证金交易衍生而来
				指的是交易者在专门从事外汇保证金交易的公司（例如银行、外汇交易商/经纪商），来签订委托买卖外汇的合同，并且缴付按合同金额一定比率的保证金（一般≥10%）
				外汇保证金交易和外汇期货交易的区别：
				

|        | 外汇现货保证金交易           | 外汇期货交易    |
| ------ | ------------------- | --------- |
| 交易场所   | 场外交易；做市商制度          | 场内交易；结算系统 |
| 合约细节   | 标准化合约但是交易商允许交易非整数手数 | 标准化合约     |
| 交易方式   | 保证金交易               | 保证金交易     |
| 到期日    | 无到期日                | 有多个到期日    |
| 结算时间   | 每日无负债               | 每日无负债     |
| 合约结束方式 | 现金结算                | 对冲了结      |
| 交易币种   | 任何币种                | 少数主要币种    |
| 交易时间   | 工作日24hrs            | 交易所营业时间   |
	外汇套汇交易
		利用在不同外汇市场/外汇交易商之间直接的价格差异，==**在低价的市场做多，在高价的市场做空**==；利用低买高卖赚取差额利润
		案例
			如果一个外汇交易商对欧元/美元的卖价报价为1.3200，而另一个外汇交易商的买价报价为1.3201，那么交易者就可以从第一个交易商中买入欧元，然后在第二个交易商处卖出欧元，赚取1个点位的无风险利润。
		外汇套汇需要具备的条件
			1.市场报价存在汇率差异，无论直接套汇还是间接套汇；==**市场存在报价的差异是套汇的前提**==
			2.要求套汇交易者有一定的技术和经验
		外汇套利交易和套汇交易的区别
			

| 套利交易                                                 | 套汇交易                                               |
| ---------------------------------------------------- | -------------------------------------------------- |
| 在两个不同的合约之间进行套利                                       | 在相同品种的外汇中/3个不同品种但存在三角关系的外汇中进行的交易                   |
| 在价差不正常时开仓，等价差回归正常时平仓                                 | 买卖同时发生，关注的是价格，然后低买高卖                               |
| 是交易者根据价差未来变化的判断，进而根据判断来交易；只有当未来价差走势与判断一致时，套利交易才能获得盈利 | 只要两个交易商之间给出的价格有套汇的机会，就可以通过套汇来盈利；==**因此市场风险相对较小**== |
| 机会比套汇交易来得多，因为价差是时时刻刻存在的                              | 需要了解交易商之间的报价差异，否则机会稍纵即逝                            |
