	期现套利
假设我做一个套利交易，如表所示：

|     | 合约1      | 合约2      |
| --- | -------- | -------- |
| 1月  | 做多 3000  | 做空3200   |
| 4月  | 卖出平仓3300 | 买入平仓3450 |
| 盈亏  | +300     | -250     |
最后得到50的净收益
也就是说我需要保证最后赚的那个合约能够>亏钱的那个合约
同时，我们可以知道，跨期套利需要保证同市场同品种合约、跨市场套利需要保证同期同品种合约、跨合约套利需要同期同市场
	例如上海一级铜期货合约，买5月铜期货，做空9月的，因为5月和9月同涨同跌（趋势一样），那么一定会一个盈利一个亏损
	又例如做多9月上海一级铜，做空9题伦敦一级铜
	再例如做多4月沪铜合约，做空4月沪铝合约
套利的作用
	一、套利行为有助于期货价格与现货价格、不同期货合约价格之间的==**合理价差关系的形成**==，==**套利交易的获利来自对不合理价差的发现和利用**==
	二、套利行为有助于市场流动性的提高，套利行为的存在增大了期货市场的交易量
		毕竟套利交易涉及到两份合约的买卖，那么必然提高交易量
	
案例，假设套利交易沪铜期货合约，设持仓费150，那么合理价差是150

| 9月     | 12月    |
| ------ | ------ |
| 做多3000 | 做空3200 |
因为3200-3000=200，价差不合理，于是有套利机会，做空3200
其实在这里渗透了关于价差的知识

价差
	在期货价差套利中，交易者不关注某一个期货合约的价格向哪个方向变动，==**而是关注相关期货合约之间的价差是否在合理的区间范围内**==。
	![[Pasted image 20260417151922.png]]
	根据第一幅图，我们可以看到价差趋于一个合理区间，那么我可以做多上面1的合约，做空2的合约，因为是从合理=》不合理
	第二幅图，是从不合理=》合理，在不合理的时候买入下面的，做空上面的
	对于合约的涨跌我们不关注
	价差的计算
		建仓时的价差：==**用价格高的期货合约-价格低的**==
		计算平仓价差：按照建仓的顺序，用==**建仓时价格较大的合约-建仓时价格较小的**==
			也就是和建仓时相减的顺序保持一致
	假设：

|     | 6月   | 9月   |
| --- | ---- | ---- |
| 开仓  | 2000 | 2100 |
| 平仓  | 2300 | 2290 |
	价差和基差的相减顺序不一样，==**价差需要用价格高的-价格低的**==
	==**开仓必须大-小，平仓必须按开仓的来，不管价格**==
	也就是2100-2000;以及2290-2300
	案例
		某套利者买入5月份铝期货合约的同时卖出6月份的铝期货合约，价格分别为21730元/吨和21830元/吨，6月份价格高于5月份价格，因此价差为6月份价格减去5月份价格，即100元/吨。
		上述套利者建仓之后，5月份铝期货价格上涨至22010元/吨，6月份涨幅相对较小，为21870元/吨，如果套利者按照此价格同时将两个合约对冲了结该套利交易，则在平仓时的价差仍应该用6月份的价格减去5月份的价格，即为-140元/吨。

|     | 5月      | 6月      | 价差   |
| --- | ------- | ------- | ---- |
|     | 做多21730 | 做空21830 | 100  |
|     | 22010   | 21870   | -140 |
		==**此时价差变动为从100-(-140)=-240**==
	价差的变化
		==**如果当前/平仓时的价差>建仓时价差，那么价差扩大==**
		**==相反，如果当前/平仓时价差<建仓时的价差，那么价差缩小**==
		因为价差在建仓时总是用价格高的-价格低的，那么价差≥0恒成立
	价差变动与套利交易盈亏的计算

	买入套利
		预期==**2个/＞2个的合约价差将扩大**==，那么套利者将==**做多其中价格较高**==的合约；同时==**做空其中价格较低**==的合约
		![[Pasted image 20260417163401.png]]

		![[Pasted image 20260417163928.png]]
		还有这里，做多合约1（价格高），做空合约2
		这里始终遵循买入套利的性质：==**做多价格高的合约，做空价格低的合约**==
		而且==**价差扩大，才能盈利>亏损的部分**==
			注意，==**盈利和亏损和相不相交无关，和所处的正反向市场有关**==
		案例：
		1月4日，某套利者以370元/克卖出4月份黄金期货，同时以381元/克买入9月份黄金。假设经过一段时间之后，2月4日，4月份价格变为375元/克，同时9月份价格变为392元/克，该套利者同时将两合约对冲平仓，套利结果可用两种方法来分析。

			总结：价差扩大+买入套利=盈利
