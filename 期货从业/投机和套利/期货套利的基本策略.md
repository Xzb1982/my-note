跨期套利
	[[期货套利交易]]中，我们提到了买入和卖出套利
	那么在分类中，套利分为价差套利和期限套利；价差套利分成跨市场、跨期、跨品种套利；
	==**跨期套利按照交易策略分成牛市套利、熊市套利、蝶式套利**==
		实际上这也是==**按照套利者对近月远月合约买卖方向的不同来分的**==
		==**这和买入卖出套利的分类维度是不一样的**==
	牛市套利
		假设我做多近月合约；做空远月合约
		如果两份合约价格都上涨，那么近月合约赚，远月合约就亏
		如果都下跌，那么近月合约亏，远月合约赚
		既然走势是这样，那么要盈利就需要：
			==**两合约上涨，近月合约上涨幅度>远月合约上涨幅度**==
			==**两合约下跌，近月合约下跌幅度<远月合约下跌幅度**==
		案例
			假设10月26日，次年5月份棉花合约价格为17075元/吨，次年9月份合约价格为17725元/吨，两者价差为650元/吨。交易者预计棉花价格将上涨，5月与9月的期货合约的价差将有可能缩小。于是买入50手5月份棉花期货合约的同时卖出50手9月份棉花期货合约。12月26日，5月和9月的棉花期货价格分别上涨为17555元/吨和18060元/吨，两者的价差缩小为505元/吨。交易者同时将两种期货合约平仓，完成套利交易。
	|       | 5月     | 9月     | 价差   |
	| ----- | ------ | ------ | ---- |
	| 10.26 | 多17075 | 空17725 | +650 |
	| 12.26 | 17555  | 18060  | +505 |
	| 盈亏    | +480   | -335   |      |
			角度1：仍然分析价差，价差缩小145，卖出套利能盈利145
			角度2：近月合约涨480，远月合约涨335，近月合约涨幅>远月合约涨幅
		案例2：
			若两个月后后棉花价格并没有出现交易者预期的上涨行情，而是出现一定程度的下跌，假设12月26日，5月和9月的棉花期货价格不涨反跌，价格分别下跌至16985元/吨和17480元/吨，两者的价差缩小为495元/吨。交易者同时将两种期货合约平仓，从而完成套利交易。
	|       | 5月     | 9月     | 价差   |
	| ----- | ------ | ------ | ---- |
	| 10.26 | 多17075 | 空17725 | +650 |
	| 12.26 | 16985  | 17480  | +495 |
	| 盈亏    | -90    | +245   |      |
			角度一：价差缩小155，卖出套利=盈利
			角度二：近月下跌幅度<远月下跌幅度，盈利
		变式
			现在我们将原来的数字换一换：
	|       | 5月     | 9月     | 价差   |
	| ----- | ------ | ------ | ---- |
	| 10.26 | 多17725 | 空17075 | +650 |
	| 12.26 | 18060  | 17555  | +505 |
	| 盈亏    | +335   | -480   |      |
			价差缩小，买入套利，一定亏损
	综上
		==**在正向市场，也就是近月合约价<远月合约价时，牛市套利=卖出套利==**
		**==在反向市场上，近月合约价>远月合约价，牛市套利=买入套利**==
	熊市套利
		做空近月合约，做多远月合约
		如果市场两个合约都涨，那么近月份合约就亏钱，远月份合约就赚钱；
		如果两个合约都跌，那么近月合约赚钱，远月合约亏钱
		==**两个合约都涨，赚钱的前提是近月合约涨幅<远月合约涨幅**==
		==**两个合约都跌，赚钱的前提是近月跌幅>远月跌幅**==
		案例
			假设交易者在7月8日看到，11月份上海期货交易所天然橡胶期货合约价格为12955元/吨，次年1月份合约价格为13420元/吨，前者比后者低465元/吨。交易者预计天然橡胶价格将下降，11月与次年1月的期货合约的价差将有可能扩大。于是卖出60手(1手为5吨)11月份天然橡胶期货合约的同时买入60手次年1月份合约。到了9月8日，11月和次年1月的天然橡胶期货价格分别下降为12215元/吨和12775元/吨，两者的价差为560元/吨，价差扩大。交易者同时将两种期货合约平仓，完成套利交易。
	|     | 11月    | 次年1月   | 价差   |
	| --- | ------ | ------ | ---- |
	| 7.8 | 空12955 | 多13420 | +465 |
	| 9.8 | 12215  | 12775  | +560 |
	| 盈亏  | +740   | -645   |      |
			盈利95，卖近买远，近月跌幅>远月跌幅，正向市场
			95* 60* 5=28500
			因为买入套利，卖出套利不用考虑正反向市场，所以更简单
		案例2
			若两个月后天然橡胶期货价格并没有像交易者预计的那样下跌，而是出现了上涨行情，交易者在7月8日看到，假设9月8日，11月和次年1月的天然橡胶期货价格不降反涨，价格分别上涨至13075元/吨和13625元/吨，两者的价差为550元/吨，价差扩大。交易者同时将两种期货合约平仓，从而完成套利交易。
	|     | 11月    | 次年1月   | 价差   |
	| --- | ------ | ------ | ---- |
	| 7.8 | 空12955 | 多13420 | +465 |
	| 9.8 | 13075  | 13625  | +550 |
	| 盈亏  | -120   | +205   |      |
			价差扩大，盈利85
			近月涨幅<远月涨幅，都是正向市场
		结论：
			正向市场，熊市套利卖近买远（因为近月合约价<远月合约价），熊市套利=买入套利
			反向市场，熊市套利卖近买远，需要价差缩小才能盈利，熊市套利=卖出套利
		口诀
			==**小牛正盈：小指的是价差缩小，牛是牛市套利，正是正向市场，那么可以盈利==**
			**==然后用负号的法则来判断，“小牛正”变一个变量都是亏；变两个就负负得正**==
			辅助口诀：买近卖远是牛套，卖近买远是熊套，正投学牛反学熊（正向市场投资就用牛市套利，反向市场就要熊市套利）
				因为正向市场投机，近月价格<远月价格，买近卖远，就是牛市套利
				反向市场投机，近月价格>远月价格，预测会回归正向，就要卖近买远，就是熊市套利
	蝶式套利
		案例
			2月1日，3月份、5月份、7月份的大豆期货合约价格分别为5650元/吨、5730元/吨和5775元/吨，某交易者认为3月份和5月份之间的价差过大，而5月份和7月份之间的价差过小，预计3月份和5月份的价差会缩小而5月份与7月份的价差会扩大，于是该交易者以该价格同时买入150手(1手为10吨)3月份合约、卖出350手5月份合约、买入200手7月份大豆期货合约。到了2月18日，三个合约的价格均出现不同幅度的下跌，3月份、5月份和7月份的合约价格分别跌至5450元/吨、5510元/吨和5570元/吨，于是该交易者同时将三个合约平仓。
	|      | 3月    | 5月    | 7月    |
	| ---- | ----- | ----- | ----- |
	| 2.1  | 多5650 | 空5730 | 多5775 |
	| 2.18 | 5450  | 5510  | 5570  |
	| 盈亏   | -200  | +220  | -205  |
			-200* 150* 10+ 220* 350* 10-205* 200* 10=60000
			==**注意这里不能用价差了，因为合约手数不一样**==
			![[Pasted image 20260419193137.png]]
			蝶式套利要遵循的规则：
				==**①居中的仓位方向和两旁的仓位方向必然相反，两旁的仓位方向必然相等==**
				**==②居中的建仓手数=两旁的建仓手数相加**==
				那么蝶式套利一定是牛市+熊市/熊市+牛市
			==**蝶式套利理论上的收益和风险都比正常的套利要小**==

跨品种套利
	跨品种套利的逻辑是
		一般来说，商品的价格总是围绕==**其内在价值上下波动**==，而==**不同的商品因其内在的某种关联**==，如==**需求替代品、需求互补品、生产替代品或生产互补品**==等，使得他们的价格存在着某种==**稳定且合理的比值关系。**==
			[[需求]]里面讲到，互补品是==**两种商品AB，A价格上升，买B商品的人的数量下降，例如轮胎是汽车的替代品**==
			替代品是==**两种商品AB，A价格上升，买B商品的人的数量上升，例如雪梨是苹果的替代品**==
		案例
			6月1日，次年3月份上海期货交易所铜期货合约价格为69260元/吨，而次年3月该交易所铝期货合约价格为21170元/吨，前者比后者高48090元/吨。某套利者根据两种商品合约间的价差分析，认为价差小于合理的水平，如果市场机制运行正常，这两者之间的价差将会恢复正常。于是决定买入30手(1手为5吨)次年3月份铜期货合约的同时卖出30手次年3月份铝期货合约，以期在未来某个有利时机同时平仓获取利润。6月28日，上述铜期货合约和铝期货合约分别降至69030元/吨和20890元/吨，该套利者同时将两种期货合约对冲平仓。

| 6.1  | 多69260 | 空21170 | +48090 |
| 6.28 | 69030  | 20890  | +48140 |
| 盈亏   | -230   | +280   |        |
			价差扩大50，买入合约
			盈利：30* 5* 50=7500
	跨品种套利的第二种逻辑
		原材料与成品之间的套利是指==**利用原材料商品和它的制成品之间的价格关系进行套利**==。
		比较典型的是大豆与其两种制成品--==**豆油和豆粕之间的套利**==。
		大豆提油套利

			==**卖出大豆期货合约，买进豆油和豆粕期货合约**==，同时缩减生产，减少豆粕和豆油的供给量

期现套利
	定义
		通过==**利用期货市场和现货市场的不合理价差**==进行反向交易而获利。
		理论上，期货价格和现货价格之间的价差==**主要反映持仓费的大小**==。
		但现实中，==**期货价格与现货价格的价差并不绝对等同于持仓费**==，==**有时高于或低于持仓费**==。当价差与持仓费出现较大偏差时，就会产生期现套利机会。
	情形
		价差>>持仓费
			买入现货，做空期货合约
			==**因为预测这个价差肯定会降低，因为要满足持仓费+现货价=期货价**==
		价差<<持仓费
			做空现货，做多期货合约
			因为==**预测价差上升**==
	案例
		6月30日，9月份郑州商品交易所白糖期货价格为5800元/吨，郑州现货市场白糖价格为5700元/吨，期货价格比现货价格高100元/吨。套利者根据分析，估算出持仓费约为每吨30元，认为存在期现套利机会。按照5700元/吨的价格买入白糖现货，同时在期货市场以5800元/吨的价格卖出白糖期货合约。如果套利者一直持有到期并进行交割，赚取的价差为100元/吨(即5800-5700=100)，扣除持有白糖所花费的持仓费30元/吨之后，套利者盈利70元/吨。
		

|      | 现货    | 期货    | 价差           |
| ---- | ----- | ----- | ------------ |
| 6.30 | 多5700 | 空5800 | +100（持仓费是30） |
		价差>持仓费，期货价太大，现货价太低
		到时候实物交割，成本5730，同时期货5800平仓，赚70
	变式
		又例如期货价格是5730，现货价5700，持仓成本100
		那么原来的成本是5700+100=5800；
		那么我不如现在把现货卖掉，然后做多期货，于是就按照5730来交割
		于是比原先5800-5730=70，节省70

26.【判断是非题】牛市套利对于可储存的并且作物年度相同的商品较为有效。(
==**A.对**==
B.错
为什么强调“可储存”？因为可储存商品的近月和远月价格之间，有一条比较稳定的联系线，那就是 持有成本，也就是仓储、利息、保险之类的成本。==**对于谷物、大豆这类能存放的商品，不同月份合约的价差，往往就围绕这类持有成本变化，所以近月和远月的联动更强，跨期价差更有规律。**==反过来，不可储存商品例如活牛、生猪，不同月份之间的价格联系就弱很多，牛市套利的基础也就差。

为什么还要强调“同一作物年度”？因为一旦跨作物年度，近月和远月就不只是“时间不同”，而==**是可能对应 旧作 和 新作 两套不同的基本面。**==旧作的库存已经大致明确，新作还要受播种、天气、单产、收成预期影响，所以两者可能像“同名但不同阶段”的两种货。CME 的材料就明确说过，旧作和新作虽然是同一种商品，但在基本面上可能表现得很不一样；Iowa State 也提醒，**同一作物年度内部** 的跨期风险通常较小，而把旧作滚到新作时要特别谨慎。
