衍生品市场的功能
	价格发现
		为什么能起到价格发现的作用：
		1.期货交易参与者众多，可以==代表供求双方的力量==以帮助公平价格的形成
		2.期货交易中的交易人士==大多熟悉行情，有丰富知识和广泛的信息渠道以及一套科学的分析和预测方法==
		3.期货交易的==透明度高，竞争公开公平==，有助于形成公正的价格
		特点：
		①预期性：
			形成的期货价格反映了市场上大多数人的预期，进而能反映供求变动趋势
		②连续性：
			期货价格能连续不断地反映供求关系以及变化趋势
			因为期货价格是不断变动的
		③公开性：
			期货价格是在交易所内公开竞争达成的，根据信息披露制度，所有在期货交易所达成的交易以及价格必须向会员报告并公开
		④权威性：
			根据上述性质，既然供求力量相当，而且不断根据形势变动，而且是公开公平的
			那么期货的市场价格就能作为一种权威价格，成为现货交易的重要参考依据，指导对现货进行定价
	规避风险
		通过套期保值实现规避风险
		套期保值

|     | 现货      | 期货      |
| --- | ------- | ------- |
| 开   | 3000元/吨 | 3200元/吨 |
| 平   | 3200元/吨 | 3410元/吨 |
		假设我们是豆油的生产商，三个月以后需要打豆，我担心三个月以后大豆价格上涨
		我们的计划是：提前买大豆然后放三个月拿来用，但是这会产生仓储成本
		假设现在的大豆期货为3200元每吨，三个月以后大豆价格真的涨到了3200元每吨
		假设大豆现货和期货价格都受到同样的供求因素影响，那么大豆期货在三个月之后涨到了3410元每吨
		于是在三个月之后，我用3200元每吨买入大豆；3410卖出大豆期货
		+3410-3300-(3200-3000)=10,用盈利弥补亏损的过程就是套期保值
		假设情况反过来

|     | 现货      | 期货      |
| --- | ------- | ------- |
| 开   | 3000元/吨 | 3200元/吨 |
| 平   | 2820元/吨 | 3010元/吨 |
		+3200-3010-(3000-2820)=10
		这里我们用现货的成本节省＝》弥补了期货的亏损
		==**套期保值的目的是为了规避风险，不是为了投机**==
		套期保值的实现条件
			==**套期保值不是消灭风险，而是将其转移**==。期货投机者是期货市场的风险承担者
			这意味着，==**投机者在获取投机利润的同时也在承担相应的价格波动风险**==
	资产配置（次要功能）
		原因
			借助衍生品为其他资产做风险对冲（通过套期保值的方式）
			期货和衍生品风险大，但是能回去高额收益（投机赚钱）
		原理
			1.除了套期保值之外，商品期货可以在一定程度上抵消通胀的影响
				特别是贵金属期货，能以比投资现货更低的成本来为投资者实现资产保值
			2.将衍生品纳入投资组合能实现更好的“风险－收益”组合

衍生品市场的作用（常考）
	宏观经济
		提供分散、转移价格风险的工具以稳定国民经济
		为制定宏观经济政策提供参考
		促进本国经济国际化
		有助于市场经济体制的建立和完善
	微观经济（对于企业）
		==**1.锁定生产成本，实现预期利润（套期保值）==**
		**==2.利用期货价格信号，组织安排现货生产==**
		**==3.期货以及衍生品市场拓展现货销售和采购渠道**==

你可以这样理解：

- 用期货价格判断未来价格趋势
- 用套期保值锁定采购价或销售价
- 提前安排生产、库存和销售计划
