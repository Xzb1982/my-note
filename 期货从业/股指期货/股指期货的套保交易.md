最优套保比率和β系数
	用股指期货进行的套保大多数是交叉套保
		例如基金公司现在持有的30只股票，但是市场上是没有这对应30只股票的指数的
		但是沪深300，上证50这些指数里面的股票是相对一段时期内不变的
		因此导致==**基金公司持有的股票≠股指里面筛选出来的股票**==
		但是，==**既然能做套保，那么必然要这30只股票和股指期货要同涨同跌；即需要有关联性**==
		同时也意味着需要确定一个最优套保比率
	确定最优套保比率
		个股的β系数（代表系统性风险）
			自变量是指数的收益
			因变量是我做套保的股票组合的收益
			![[Pasted image 20260430090528.png]]
			收益点如上图。
			经过一元线性回归之后确定一条拟合直线
			那么这条线的斜率tanα=β系数值
			就例如指数的收益为1%，持有的股票组合是收益为1.5%
				指数的收益为2%，持有的股票组合是收益为3%
				指数的收益为3%，持有的股票组合是收益为4.5%
				那么假设我有1000万元，那么我需要1500万元价值的合约才能套保
			β>1，股票比市场整体波动性高，承担的系统性风险>平均市场风险
			β<1,股票比市场整体波动性低，承担的系统性风险<平均市场风险
			组合的β系数的计算
				例如我的组合有A B C三只股票
				持有价值分别是30万，40万，30万
				β系数分别是1.5,1.2,0.9 
				股票组合的β系数 = (30/100)* 1.5+ (40/100)* 1.2+ (30/100)* 0.9
			最优套保比率的算法
				==**买卖期货合约数量 = β * 现货总价值/(期货指数点* 每点乘数)**==
					例如我30t大豆，10t/手的合约需要多少手才能满足套保需求？
						==30/10即可得到手数，这个手数就是套保所需手数==
					对于股指期货也是一样
						假设手里持有100000000的股票
						那么100000000/3000* 300
						这个是合约价值，也就是100000000元/ 3000点/手* 300元/点 
						之后再乘上β系数，因为乘上β系数才能相关
					公式：$$买卖期货合约数量 = \beta \times \frac{现货总价值}{期货指数点* 每点乘数}$$
					==**β系数越大，买卖所需合约的数量越大**==
			案例（卖出套期保值）
				国内某证券投资基金在某年9月2日时，其收益率已达到26%，鉴于后市不太明朗，下跌的可能性很大，为了保持这一业绩到12月，决定利用沪深300指数期货实行保值。假定其股票组合的现值为2.24亿元，并且其股票组合与沪深300指数的系数为0.9。假定9月2日时的现货指数为3400点，而12月到期的期货合约为3650点。该基金首先要计算==**卖出**==多少期货合约才能使2.24亿元的股票组合得到有效保护。
				(224000000/ 3650* 300 )* 0.9 ≈ 184.11 => 184
				12月2日，现货指数跌到2200点，而期货指数跌到2290点(现货指数跌1200点，跌幅为35.29%，期货指数跌1360点，跌幅为37.26%)，这时该基金买进184张期货合约进行平仓，
				==**1360* 300 * 184/100000000=0.75072亿元**==
				市值缩水部分不用管
			案例（买入套期保值）
				某机构在4月15日得到承诺，6月10日会有300万元资金到账。该机构看中A、B、C三只股票，现在价格分别为20元、25元、50元，如果现在就有资金，==**每个股票投入100万元就可以分别买进5万股、4万股和2万股**==。（这说明持有比例每个都是1/3）
				由于现在处于行情看涨期，他们担心资金到账时，股价已上涨，就买不到这么多股票了。于是，采取买进股指期货合约的方法锁定成本。
				假定相应的6月到期的期指为2500点，每点乘数为100元。三只股票的系数分别为1.5、1.3和0.8，则首先得计算应该买进多少期指合约。
				1/3* 1.5+1/3* 1.3+1/3* 0.8 = 1.2
				（3000000/2500* 100)* 1.2=15
				6月10日，该机构如期收到300万元，这时现指与期指均已涨了10%，即期指已涨至2750点，而三只股票分别上涨至23元(上涨15%)、28.25元(上涨13%)、54元(上涨8%)。如果仍旧分别买进5万股、4万股和2万股，则共需资金:23x5+28.25x4+54x2=336(万元)，显然，资金缺口为36万元。由于他们在指数期货上做了多头保值，==**6月10日将期指合约卖出平仓，共计可得:15x(2750-2500)x100=37.5(万元)**==，弥补资金缺口后尚有余裕。

2.【单项选择题】当现货总价值和期货合约的价值已定下来后，所需买卖的期货合约数随着β系数的增大而(
==**A.变大**==
B.不变
C.变小
D.以上都不对
因为 β系数 = $$\beta = \frac{股票组合对市场指数的敏感程度​}{市场组合自身波动}$$
而且==**买卖期货合约数量 = β * 现货总价值/(期货指数点* 每点乘数)**==
现货组合是一块需要遮雨的面积，$\beta$ 决定这块面积实际有多大；
一张期货合约是一把伞，期货指数点乘以每点乘数决定这把伞能遮多少面积。
==**要遮住全部风险，就用“总风险面积 ÷ 一把伞的面积”。**==
