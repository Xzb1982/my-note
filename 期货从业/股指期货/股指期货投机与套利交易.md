股指期货投机
	看涨做多，看跌做空的交易行为
	股指期货的期现套利理论依据
		股指期货理论价格===**标的股指价格、收益率和无风险利率共同决定**==
		在正常情况下，期货指数和现货指数维持一定的动态关系
		但在现实中，股指和期货只收都在不断变化，期货和现货价格关系经常偏离，当偏离c超过一定的范围，就产生了套利的机会
	股指期货持有成本的构成
		已知期货=现货+持有成本
		资金占用成本
			==**按照市场资金利率来度量**==
			就像[[国债期货及其应用]]中一样，为了买债券放弃市场利息收入
		持有期内可能得到的股票分红红利
			持有资产的收入，在这里算是==**负值成本**==
			这算是收益，但是在成本口径来讲，要变成负值
		净持有成本=资金占用成本-持有期内可能得到的股票分红红利
		案例==**（必须掌握）**==
			买卖双方签订一份3个月后交割一篮子股票组合的远期合约，该股票组合与沪深300指数构成完全对应，现在市场价值为99万元，即对应于沪深300指数3300点(沪深期货合约的乘数为300元)。
			假定市场年利率为6%，且预计一个月后可收到6600元现金红利，该远期合约的合理价格计算过程是:
			资金占用99万元，相应的利息为==**990000x6%* 3÷12=14850(元)**==，
			一个月后收到红利6600元，==**再计剩余两个月的利息为6600x6%* 2/12=66(元)**==，红利和利息共计为6666元;
			净持有成本=14850-6666=8184(元);
			该远期合约的合理价格应为990000+8184=998184(元)。
			实际计算
				先算占用成本：990000* 6% * 3/12
				之后算红利生息：6600 * 6%* 2/12
				再算红利+利息:6666
				最后占用成本-利息-红利
				==**如果题目让你算点位,就将钱/300这个合约乘数**==
				点位算成钱就是钱* 300
			如果将上述金额用指数点表示，则为:
				99万元相当于3300指数点;
				3个月的利息为3300x6%x3÷12=49.5(个指数点);
				红利6600元相当于22个指数点，再计剩余两个月的利息为22x6%x2÷12=0.22(个指数点)，利息与红利共计为22.22个指数点;
				净持有成本为49.5-22.22=27.28(个指数点);
				该远期合约的合理价格应为3300+27.28=3327.28 (个指数点)。
			==**当实际价格>或<这个理论价格时，可以进行期现套利**==
			直接给股息率时，怎么计算股指期货的理论价格
				设r=5%，d=1.5%，6月30日为6月期货合约的交割日。4月1日、5月1日、6月1日及6月30日的现货指数分别为1400点、1420点、1465点及1440点，这几日的期货理论价格
				r代表市场利率，是资金占用成本的代表；d是股息率
				那么r-d就是净成本
				4-6一共三个月，期货理论价格 = 期货现货价格+持有成本
				那么 4.1的股指期货理论价格：1400+ 1400* (5%-1.5%)* 3/12
				5.1的股指期货理论价格：1420+ 1420* （5%-1.5%）* 2/12
				6.1：1465+ 1465* （5%-1.5%）* 1/12
				6月30日，期货到期，期、现价格相等，均为1440点。基本数据如上例，又假定：==①借贷利率差delta r=0.5%;==
				==②期货合约买卖手续费双边为0.2个指数点，同时，市场冲击成本也是0.2个指数点;==
				==③股票买卖的双边手续费及市场冲击成本各为成交金额的0.6%，即合计为成交金额的1.2%，如以指数点表示，则为1.2%xS(t)。==4月1日、6月1日的无套利区间计算如下:
					这一部分是这样的：假设我现在买的是大豆现货，现在是400元/t，三个月之后理论期货价格为410（400+10）,到了3个月之后期货价格为420
					实际上在3个月之前，我可以==**做空大豆，买大豆现货**==
					然后3个月之后进行交割赚20元差价，同时支付持有成本，可以赚取10元利润
					现在到股指期货，我们会发现除了持有成本（红利，利息，占用成本）
					还有手续费，例如买卖期货、股票的手续费、借钱的利息
					那么如果这些"附加成本"加起来>10,那么你的利润就会被吞噬
					那么无套利区间就在：[410,415]上面，其中415=410+x
				现在，假设是相反情况
					现在是400元/t，三个月之后理论期货价格为410（400+10）,到了3个月之后期货价格为404
					那么我可以==**做多大豆期货，做空大豆现货**==
					那么无套利区间是[410-x,410]
				回到上面1,2,3再讲什么
					1.==**代表的是借钱的成本，也就是借贷利率差**==
					2.代表的是买卖期货的成本+市场冲击成本，也就是双边0.2的手续费
					3.代表的是股票买卖成本+市场冲击成本
					合计为1.2个点
					4.1股票附带成本：1400* 1.2%=16.8、期货的附带成本：0.4个点
					借贷利率差成本：1400* 0.5%* 3/12 = 1.75点
					三者总成本=16.8+0.4+1.75=18.95
					那么用==**理论价格±这个总附加成本=无套利区间**==
				技巧
					可以通过无套利区间的上下界平均数算出答案
					==**考试中出四个选项，一个是上界，一个是下界，通过求平均算出理论价格。然后你用你算出来的理论价格来看就可以了**==
					模拟误差
						在套利交易中，实际交易的现货股票组合与指数的股票组合很少会完全一致。这时，就可能导致两者未来的走势或回报不一致，从而导致一定的误差。这种误差，通常称为模拟误差。
						模拟误差来自两方面。一方面是因为组成指数的成分股太多，另一方面，即使组成指数的成分股并不太多，但由于指数大多以市值为比例构造，以及交易最小单位的限制，严格按比例复制很可能根本就难以实现
							其实就是==你不可能将沪深300指数的所有成分股按照公布的比例一个个买回来==
	股指期货的跨期套利
		和之前的跨期套利一样，利用不同月的价差关系做多某个月份的股指期货；同时做空另一个月的股指期货
		案例
			假定利率比股票分红率高3%，即r-d=3%。5月1日上午10点，沪深300指数为3000点，沪深300指数期货9月合约为3100点，6月合约价格为3050点，9月期货合约与6月期货合约之间的实际价差为50点，而理论价差为:
			==**3000+3000 * 3% * 3/12-(3000+3000* 3% * 1/12) = 3000* 3% * 3/12=22.8**==
			50> 22.8价差太大了，==**认为未来价差缩小，需要用卖出套利**==
			9月份价格高，那么做空9月；做多6月
			5月1日下午2点，9月合约涨至3150点，6月合约涨至3120点，9月期货合约与6月期货合约之间的实际价差缩小为30点。在不考虑交易成本的情况下，投资者平仓后每手合约获利为:

|          | 6月    | 9月    |
| -------- | ----- | ----- |
| 5.1 10am | 多3050 | 空3100 |
| 5.1 2pm  | 空3120 | 多3150 |
| 盈亏       | +70   | -50   |
			+20* 300 =6000元
	股指期货期限套利操作
		期价高估的正向套利
			理论价格±额外成本=无套利区间的上下限
			==**期价高估指的是>无套利区间上限**==
			那么应该做空股指期货，做多股指现货
		期价低估的反向套利
			期价<无套利区间下限就是期价低估
			那么做多期货，做空现货
