在[[套期保值的概念、原理、种类]]中，我们知道实现完全的套期保值是不可能的，但是为什么会这样呢？
	1.==**期货价格与现货价格变动幅度并不完全一致**==
	2.由于期货合约标的物可能与套期保值者在现货市场交易的商品等级存在差异，==**当不同等级的商品在供求关系上出现差异**==时，虽然两个市场价格变动趋势相近，但在变动程度上会出现差异性
	3.期货市场建立的==**头寸数量与被套期保值的现货数量之间存在差异**==时，即使两个市场价格变动幅度完全一致，也会出现两个市场盈亏不一致的情况
	4.因缺少对应的期货品种，一些加工企业无法直接对其所加工的产成品进行套期保值，==**只能利用其使用的初级产品的期货品种**==进行套期保值时，由于初级产品和产成品之间在价格变化上存在一定的差异性，从而导致不完全套期保值
既然变动不一致，那么我们需要一个概念来量化它
	基差
		某一特定地点某种商品/资产的现货价格，与相同商品/资产的某一特定期货合约价格间的价差
		公式
			基差＝现货价格－期货价格
		基差走强和基差走弱
			基差走强指的就是基差扩大，当现货价格的涨幅>期货价格涨幅/现货价格跌幅<期货价格跌幅时，基差数值变大
				就例如现货5元－期货7元＝－2
				现在到了现货7元－期货8元＝－1，那么基差走强
				又例如现货10元－期货12元＝》现货8元－期货9元＝》基差走强
			期差走弱指的是基差变小，当现货价涨幅<期货价格跌幅/现货价跌幅>期货价跌幅
		一，案例1（基差与卖出套保）
			5月初某糖厂与饮料厂签订销售合同，约定将在8月初销售100吨白糖，价格按交易时的市价计算。目前，白糖现货价格为5500元/吨。该糖厂担心未来糖价会下跌，于是卖出10手(每手10吨)9月份白糖期货合约，成交价格为5800元/吨。至8月初交易时，现货价跌至5000元/吨，与此同时，期货价格跌至5200元/吨。该糖厂按照现货价格出售100吨白糖，同时按照期货价格将9月份白糖期货合约对冲平仓。

|     | 现货    | 期货    | 基差      |
| --- | ----- | ----- | ------- |
| 5月  | 多5500 | 空5800 | －300    |
| 8月  | 空5000 | 多5200 | －200    |
| 盈亏  | －500  | ＋600  | 基差走强100 |
			那么在卖出套保中，基差走强，可以在弥补亏损的情况下盈利
			实际售价5600
		二，案例2
			5月初某地钢材价格为4380元/吨。某经销商目前持有5000吨钢材存货尚未出售。为了防范钢材价格下跌风险，该经销商卖出500手(每手10吨)11月份螺纹钢期货合约进行套期保值，成交价格为4800元/吨。到了8月初，钢材价格出现上涨，该经销商按4850元/吨的价格将该批现货出售，与此同时，将期货合约对冲平仓，成交价格为5330元/吨。

|     | 现货    | 期货    | 基差     |
| --- | ----- | ----- | ------ |
| 5月  | 多4380 | 空4800 | －420   |
| 8月  | 4850  | 5330  | －480   |
| 盈亏  | ＋470  | －530  | 基差走弱60 |
			实际售价：4850-530=4320
			==**结论：卖出套保＋基差走弱＝出现亏损**==
		一，案例1（基差与买入套保）
			5月初，某饲料公司预计3个月后需要购入3000吨豆粕。为了防止豆粕价格上涨，该饲料公司买入9月份豆粕期货合约300手(每手10吨)，成交价格为2910元/吨。当时现货市场豆粕价格为3160元/吨。至8月份，豆粕现货价格上涨至3600元/吨，该饲料公司按此价格采购3000吨豆粕，与此同时，将豆粕期货合约对冲平仓，成交价格为3280元/吨。

|     | 现货   | 期货   | 基差     |
| --- | ---- | ---- | ------ |
| 5月  | 3160 | 2910 | +250   |
| 8月  | 3600 | 3280 | +320   |
| 盈亏  | -440 | +370 | 基差走强70 |
			实际成本3600-370=3230
			==**结论：买入套保＋基差走强＝套保亏损**==
		二，案例2
			3月初，某轮胎企业为了防止天然橡胶原料价格进一步上涨，买入7月份天然橡胶期货合约100手(每手10吨),成交价格为24000元/吨，对其未来生产所需要的1000吨天然橡胶进行套期保值。当时现货市场天然橡胶价格为23000元/吨。之后天然橡胶价格未涨反跌，至6月初，天然橡胶现货价格跌至20000元/吨。该企业按此价格购入天然橡胶现货1000吨，与此同时，将天然橡胶期货合约对冲平仓，成交价格为21200元/吨。

|     | 现货     | 期货     | 基差      |
| --- | ------ | ------ | ------- |
| 3月  | 空23000 | 多24000 | －1000   |
| 6月  | 多20000 | 空21200 | －1200   |
| 盈亏  | ＋3000  | －2800  | 基差走弱200 |
			实际成本：20000＋2800= 22800
			==**结论：买入套保＋基差走弱＝套保盈利**==
		最终结论（==**很重要**==）：==**买走弱盈利，卖走强盈利，完全套保基差不变**==
			怎么直观理解：==**基差是现货-期货，基差走强，那么现货价就增长，那么这不是买入套保要看到的；但是是卖方希望看到的，因为卖方想高价卖出**==
		我们将基差变动带给套期保值的不确定性称为基差风险
			套保的实质是==**用较小的基差风险代替较大的现货价格波动风险**==
		那么为什么会发生基差呢？（影响因素）
			==**基差的大小和持仓费有关**==
			持仓费：又称持仓成本，指的是==**拥有/保留某种商品、资产等而支付的仓储费（商品期货的成本）、保险费（对应商品期货的成本）和利息（对于金融期货的成本）等费用总和**==（注意，在这里没有交易的手续费）
			假设我现在有一个现货价3000的大豆，3个月以后，假设3个月的仓储成本是300元
			那么3300就是期货价，因为300的仓储成本是必须要支付的
			==**持仓费的高低与距期货合约到期时间长短有关**==
				距离交割时间越近，持仓费越低；
					就例如时间越长，仓储成本越高
				理论上，期货合约到期，持仓费可以减小到0，基差也将变为0
		正向市场和反向市场
			在上面我们提到，==**现货价+持仓成本＝期货价，持仓成本>0，那么现货价<期货价**==
				我们定义==**正向市场是现货价<期货价的市场，此时基差<0**==
				反向市场就相反，现货价>期货价，基差>0
					也就是即便没有持仓成本，现货价还是>期货价，那么这个市场不正常，于是称作反向市场
			正向市场
				期货价>现货价/远期期货合约价格>近期期货合约价格
				假设5月是现在，9月是未来的一个合约，11月是更远的一个合约
					==**更远的合约，仓储成本更大，那么合约的价格也更大**==
				正向市场主要反映了持仓费
			反向市场
				现货价>期货价/远期期货合约价格<近期期货合约价格
				![[mmexport1776136086475.jpg]]
				第一种情况：==**短期内出现现货供给不足，需求疯涨，那么现货价格急剧上升；最终超越期货价格形成反向市场（但是趋势还是相同的）**==
				第二种情况：==**预计将来现货供过于求，现货价格急剧下跌；最终期货价格跌过现货价格形成反向市场（但是趋势仍然相同）**==
				而且即便是反向市场，==**也不意味着现货持有者没有持仓费支出**==
		规避基差风险的操作方式
			点价交易：就是买方和卖方协商未来进行现货交易
				那么价格怎么确定？＝》以某5月的期货合约在4.20的结算价的基础上±升贴水
				假设在4.22交货付款
				注意，==**点价交易并没有参与期货的买卖**==，本质上==**点价交易是一种定价方式**==
				分类
					确定交易时间的权利属于买方，那么叫做==**买方叫价交易**==
					如果权利属于卖方，那么叫做==**卖方叫价交易**==
			基差交易
				点价交易和套保的结合
				既然点价交易以期货合约的结算价为基础，那么我可以顺着期货合约进行套期保值
				实施点价之前，双方所约定的期货基准价不断变化，那么意味着==**交易者仍面临的价格变动风险**==，这就需要套利了
				(二)案例
					10月20日，中国某榨油厂与美国某贸易商签订进口合同，约定进口大豆的到岸价为"CBOT的1月大豆期货合约+CNF100美分"，即在1月份==**CBOT大豆期货价格的基础上加上100美分/蒲式耳的升水**==，以此作为进口到岸价格，并约定由该榨油厂在12月15日装船前根据CBOT期货盘面价格自行点价确定。合同确立后，大豆的进口到岸价格实际上并未确定下来，如果在榨油厂实施点价之前，==**1月份CBOT大豆期货价格上涨，该榨油厂就要接受此高价**==。为了规避这一风险，该榨油厂在签订进口合同同时，在CBOT上买入等数量的1月份大豆期货合约进行套期保值。
					假设订立合约的价格是800美分/蒲式耳，那么到岸价格为900美分/蒲式耳
					到了12月15日，该贸易商完成大豆装船，并通知该榨油厂点价。该油厂在1月份CBOT大豆期货上分批完成点价，均价为1030美分/蒲式耳。该批大豆的进口到岸价也相应确定下来，为1030+100=1130(美分/蒲式耳)。该榨油厂按该价格向贸易商结清货款。与此同时，该榨油厂将套期保值头寸卖出平仓，结束交易。

|     | 现货（美分/蒲式耳） | 期货（美分/蒲式耳） | 基差   |
| --- | ---------- | ---------- | ---- |
| 10月 | 900        | 800        | 100  |
| 12月 | 1130       | 1030       | 100  |
| 盈亏  | －230       | ＋230       | 基差不变 |
			企业开展套期保值业务需注意的事项(了解)
				一，企业在参与期货套期保值之前，需要结合自身情况进行评估
				二，企业应完善套期保值的机构设置
				三，企业需要具备健全的内部控制制度和风险管理制度
				四，加强对套期保值交易中现金流风险，流动性风险和操作风险等的管理

	20.【综合题(单选)】7月11日，某铝厂与某铝贸易商签订合约，约定以9月份铝期货合约为基准，==**以＞铝期货价格100元/吨的价格交收**==。同时铝厂进行套期保值。以16600元/吨的价格卖出9月份铝期货合约。此时铝现货价格为16350元/吨。8月中旬，该铝厂实施点价，以16800元/吨的期货价格作为基准价，进行实物交收，同时按该价格将期货合约对冲平仓，此时铝现货价格为16600元/吨。通过套期保值交易，该铝厂铝的售价相当于()元/吨。
	==**A.16700**==

|     | 现货        | 期货    |
| --- | --------- | ----- |
| 7月  | 16350     | 16600 |
| 8月  | 16800+100 | 16800 |
| 盈亏  | +550      | -200  |
假设铝厂在 7 月卖出 9 月期货，价格是16600
到了 8 月，铝厂卖现货，并把期货平仓。若那时现货价是 S1​，期货平仓价是 F1​，那么铝厂的实际售价就是：
S1+（F0-F1）整理一下得到： F0+(S1-F1),其中S1-F1就是基差
我们可以看到基差参与了实际售价的运算，那么基差的变动是有风险的，我们应该怎样来降低基差变动风险呢？
	通过点价交易+套保
	这个题里，铝厂和贸易商先签了一个基差合同：
		交收价 = 9月铝期货价格+100的升水
		然后铝厂做卖出套保，做空9月期货
		到8月中旬，铝厂选定8月某一天的16800为价格
		于是现货合同的交收价就不是看当日现货市场随行就市，而是按合同公式算：
		16800+100；
		与此同时，期货平仓亏损：16600−16800=−200。所以最终实际售价是16900−200=16700

52.【多项选择题】下列情形中，可能导致铜期货市场呈反向市场的有(
==**A.智利大型铜矿工人罢工**==
B.铜现货库存大量增加
==**C.某铜消费大国突然大量买入铜**==

A 选项：智利是全球重要的铜生产国，大型铜矿工人罢工，会导致铜的产量在短期内大幅减少，进而使铜的现货供应紧张。而期货市场上，由于期货合约对应的是未来的交割，其供应预期相对充足，所以在短期内会出现现货价格高于期货价格的情况，导致铜期货市场呈反向市场。

B 选项：铜现货库存大量增加，意味着现货市场上铜的供给较为充裕，在需求稳定的情况下，现货价格会受到打压。而期货市场因为对未来的预期可能相对较好，或者库存压力没有现货库存大，会出现期货价格高于现货价格的情况，这是正向市场的表现，而不是反向市场。

C 选项：某铜消费大国突然大量买入铜，会使铜的现货需求在短期内急剧增加，而现货市场的供应难以立刻跟上，导致现货价格快速上涨。期货市场上的交割是在未来，短期内的需求冲击相对较小，所以会出现现货价格高于期货价格的反向市场情况。

D 选项：铜需求疲软，会使得现货市场上铜的销售困难，价格下跌。在这种情况下，期货市场由于对未来有一定的预期，如果预期未来需求也不会有明显改善，期货价格也会下跌，且可能由于仓储成本等因素，期货价格会高于现货价格，呈现正向市场，而不是反向市场。
