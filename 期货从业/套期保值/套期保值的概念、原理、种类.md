套期保值的定义
	一般定义
		套期保值又称避险、风险对冲等。
	广义上的定义
		是指企业利用一个或一个以上的工具进行交易，预期全部或部分对冲其生产经营中所面临的价格风险的方式。
		也就是说可能做到完全的套期保值，也可能只能弥补一部分亏损
	选取的工具
		期货、期权、远期、互换
套期保值的原理
	套期保值的核心是==**“风险对冲”**==，
	其根本的原理是
		==**期货价格与现货价格受到相似的供求等因素的影响，两者的变动趋势趋同**==
		因为期货本质上是人们对未来现货价格的预期，那么现货涨，期货也会涨；现货跌，期货也会跌
	这样的话，通过套期保值，无论价格是涨还是跌，==**总会出现一个市场盈利而另一个市场亏损的情形**==，盈亏相抵，就可以规避因为价格波动而带来的风险
		这里的一个市场盈利，一个市场亏损，可以是现货市场，也可以是期货市场
	![[Pasted image 20260413144444.png]]
	这是套期保值矩阵
	![[Pasted image 20260413144627.png]]
	情况二，例如我是买入方，现货市场跌价，期货市场因为我是做多，进而亏损
	那么买入现货，因为现货跌价而节省的成本去弥补期货市场上的做多而造成的亏损
	做套保应该满足的条件
		(一)在套期保值数量选择上，要使期货与现货市场的价值变动大体相当
			1.套保中，==**期货合约所代表的数量与被套期保值的现货数量的比率，即套期保值比率通常为1**==
				==也就是3000t大豆现货，如果要做套保就需要做10t* 300手的大豆期货合约==
			2.若不存在与被套期保值商品或资产相同的期货合约，==企业可以选择其他的相关期货合约进行套期保值==，这种套期保值被称为交叉套期保值
				找到能和现货同涨同跌的就行
			那么，==**从原理上来说，因为现货价和期货价同方向变动。那么我在两个市场上分别做相反的操作就可以了**==，那么就做到一个市场盈利，另一个市场亏损
		(二)在期货头寸方向的选择上，==**应与现货头寸相反**==或作为现货未来交易的替代物
			简单来说，你未来要买入现货，你现在就做多期货；
			你未来要卖出现货，你现在就做空期货
		(三)期货头寸持有时间段与现货承担风险的时间段对应
			就相当于你三个月之后买大豆，那么你需要买一个远期合约至少也要三个月以后能覆盖的
			比如你真正要在 9 月买现货，但你只买了 6 月合约，那么 6 月合约平掉以后，到 9 月之间这段时间，你又重新暴露在价格上涨风险里了。这个空档期没有被保护住，套保就不完整。
	正确理解套保
		套保的本质就是风险对冲
		现货和期货冲抵的程度越大，规避风险的效果越好；就相当于一个-198，一个+200
		取绝对值，那么两者比值接近于1
套保的种类
	卖出套保
		假设你是一个都有生产商，担心三个月以后生产出来的豆油售价下跌
		那么我现在做空期货，假设现在现货价3000，期货价3200
		三个月之后，售价下跌到2800；期货价跌到2990
		那么期货上盈利210，现货市场上亏200
		口诀：怕涨就在期货做多，怕跌就在期货做空
	(二)案例1
		10月初，某地玉米现货价格为1710元/吨，当地某农场年产玉米5000吨。该农场对当前价格比较满意，==**担心待新玉米上市后，销售价格可能会下跌**==，该农场决定进行套期保值交易。10月5日该农场卖出500手(每手10吨)第二年1月份交割的玉米期货合约进行套期保值，成交价格为1680元/吨。到了11月，随着新玉米的大量上市，以及养殖业对玉米需求疲软，玉米价格开始大幅下滑。11月5日，该农场将收获的5000吨玉米进行销售，平均价格为1450元/吨，与此同时将期货合约买入平仓，平仓价格为1420元/吨。

|      | 现货价  | 期货价（做空） |
| ---- | ---- | ------- |
| 10月初 | 1710 | 1680    |
| 11月  | 1450 | 1420    |
| 盈亏   | -260 | +260    |
		下面我们要计算实际售价，卖出套保的实际售价就是：卖出价+期货盈亏
		假设我们没有平仓，然后经过一个月，现货价涨到1950，期货价涨到1920

|      | 现货价  | 期货价（做空） |
| ---- | ---- | ------- |
| 10月初 | 1710 | 1680    |
| 11月  | 1950 | 1920    |
| 盈亏   | 240  | -240    |
		农场参与套期保值操作的目的是规避价格不利变化的风险，而非获取投机性收益。
		==**事实上，套期保值操作在规避风险的同时，也放弃了获取投机性收益的机会。**==
	买入套保
		某一铝型材厂的主要原料是铝锭，某年3月初铝锭的现货价格为16430元/吨。该厂计划5月份需使用500吨铝锭。由于目前库存已满且能满足当前生产使用，如果现在购入，要承担仓储费和资金占用成本，而如果等到5月份购买又可能面临==**价格上涨风险**==。于是该厂决定进行铝的买入套期保值操作。3月1日，该厂以17310元/吨的价格买入100手(每手5吨)6月份到期的铝期货合约。到了5月1日，现货市场铝锭价格上涨至17030元/吨，期货价格涨至17910元/吨。该铝型材厂按照当前的现货价格购入500吨铝锭，同时将期货多头头寸对冲平仓，结束套期保值。
		直接将题目简化成表格

|     | 现货    | 期货    |
| --- | ----- | ----- |
| 3月  | 16430 | 17310 |
| 5月  | 17030 | 17910 |
| 盈亏  | -600  | +600  |
		实际成本：17030-600
		如果5月初铝锭价格不涨反跌呢？假设现货价格跌到15830，期货价格跌到16710那么：

|     | 现     | 期     |
| --- | ----- | ----- |
| 3月  | 16430 | 17310 |
| 5月  | 15830 | 16710 |
| 盈亏  | +600  | -600  |
		实际成本是15830-(-600)

一、什么叫现货多头
最典型的情况就是：**你已经持有商品或资产**。
比如你仓库里有100吨大豆。  
大豆价格上涨，你这100吨更值钱了，你受益。  
所以你在现货上是多头。
更一般地说，只要你站在“价格上涨对我有利”的一边，你就是现货多头。

二、什么叫现货空头
现货空头最典型的情况不是“我现在把货卖空了”，而是：
**我已经答应将来按固定价格==卖出==某商品，但我现在手里还没有这批货。**
