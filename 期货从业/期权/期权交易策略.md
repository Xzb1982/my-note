期权的基本交易策略
	买进看涨期权（看涨期权多头）
		假设行权价40￥（作为买价），权利金为1元
		如果标的市场价格≤40，多头不行权，因为标的市场价-行权价<0
		那么此时多头的损失一直都是这个权利金
		假设市场价格>40，买方行权；在[40,41]盈亏为负但是收益曲线有斜率
		![[Pasted image 20260505222524.png]]
		总的收益曲线如图所示，注：图上的那个1是-1，写错了
		综合分析
			平仓损益（单独买卖期权而不行权）：卖出价-买入价
			行权损益：==**标的资产价格-行权价格-权利金**==
			当内含价值=0的时候，最大损益=权利金
			==**到期损益平衡点=权利金+行权价格**==
			适用情形
				1.牛市/预期未来上涨：因为我锁定现在的低价，然后低价行权买入之后因为牛市而高价在市场卖出
				2.价格见底，市场波动率大/正在扩大，或隐含波动率低
					隐含波动率指的是==**期权价格反映的标的资产价波动率<标的资产价历史波动率/预期波动率**==（即现在的标的资产价波动率<历史的波动率）
					价格见底的时候锁定现在低价，未来涨了之后行权并以高价在市场卖出
					市场波动率大也意味着时间价值扩大
					==**隐含波动率低意味着未来波动率大**==
					而且==**前面提到权利金和波动率呈正比**==
			运用
				赚价差收益（即行权损益）
				以低成本持有标的，同时锁定最大损失
				规避标的资产上涨风险
					限制卖出标的资产的风险
						持有某资产多头的交易者，还想继续享受价格上涨带来的好处
						但是==**担心资产跌价，又担心现在卖资产之后价格又涨了**==
						假设有个标的资产价格3000，现在涨到3700；我现在想==**暂时**==卖掉这个标的资产，但是未来我还需要它；于是我先卖掉拿到700的收益
						于是我可以买一个看涨期权，将价格锁在3700。
						那么==如果这个资产价格涨了，我就可以锁定成本；如果价格跌了，我就又赚了==
					锁定未来购买标的资产的成本
	卖出看涨期权（看涨期权空头）
		仍设行权价40￥（作为卖价），权利金为1元
		如果市场价<40,卖方盈利，盈利金额=权利金=1
		>40时，卖方逐渐亏损，收益曲线如下图所示：
		![[Pasted image 20260505223345.png]]
		其中买方赚得越多；卖方亏损的越多；其幅度是一样的
		综合分析
			平仓损益一样
			行权损益=执行价格-标的资产价格+权利金
				因为卖方是先进货，再以执行价格卖出资产
				你也可以这样理解：买方赚的=卖方亏的
				或者：卖方行权损益=-买方行权损益=-(标的资产价格-执行价格-权利金)
			==**损益平衡点是一样的，都是权利金+行权价格**==
				因此==**损益平衡点和持有头寸方向无关，只和看涨还是看跌有关**==
			要交保证金（卖方都要交）
			履约方式和持仓
				卖出标的资产（因为是==**卖标的给买方接盘**==）
				履约持仓：==**标的资产空头**==
			适用情形
				==**横盘整理/小幅下跌**==
					因为我可以赚权利金的同时，买方不行权
				==**市场的波动率低/波动率收窄**==/隐含波动率高
			基本运用
				赚权利金/期权价差收益
				履约平仓标的资产的多头头寸
					其实就是==**通过期权履约的方式，将手上的标的物卖掉**==
	可以注意到，在上面例子中，41是买方和卖方的盈亏平衡点
	==**看涨期权的盈亏平衡点(无论买卖)=权利金+行权价格**==
	买进看跌期权
		仍然假设权利金=1，执行价是40
			注意，此时的执行价是一个卖价，执行价-标的市场价格；买方是从市场进货，卖标的资产给卖方
			当标的市场价>40时，买方不会行权，就例如42元，直接去市场上卖而不是行权价卖出；于是买方亏的是权利金
			当标的市场价<40时，在[39,40]之间盈亏为负但是收益曲线有斜率；
			![[Pasted image 20260506171808.png]]
			==**盈亏平衡点：执行价格-权利金**==
		综合分析
			看跌是==**锁定卖价**==
			标的资产价格状况
				熊市
				预测未来价格下跌
					如果价格真的下跌，那么我的==**看跌期权锁定的是高价；于是我可以在市场上买标的，然后通过行权价卖标的**==
				价格见顶，波动率扩大/隐含波动率低
					==**波动率增加对买方都有好处，无论是看涨/看跌的买方**==
			损益
				平仓损益和之前一样
				行权损益 = 行权价格-标的资产价格-权利金
					==**对于买方来讲，权利金一定是负的，但是要和下面的口诀区分，因为损益平衡点是看涨用+权利金，看跌用-权利金**==
			基本运用
				赚取价差收益
				实现做空标的资产的目的，同时锁定最大损失
				规避标的资产价下跌的风险
					保护标的资产多头
						持有标的资产多头，==**认为未来标的资产价上涨；但是担心标的资产下跌的损失**==
						在此情形下，投资者可买进看跌期权
					锁定现货持仓价值/未来出售标的资产的收入
						未来要卖标的的生产商，为了规避价格下跌导致的出售收入降低
						可以通过买看跌期权以锁定售货收入
	卖出看跌期权
		同理，卖出看跌如图：![[Pasted image 20260506171929.png]]
		行权损益
			标的资产价格-行权价+权利金（卖方肯定是赚权利金）
			注意和看涨期权多头的行权损益：标的资产价-行权价-权利金
		要交保证金（卖方都要交）
		期权头寸：看跌期权空头
		履约方式和持仓
			==**用行权价买标的资产，然后在市场上卖出**==
			履约持仓：标的资产多头
				只有买方不行权的时候我才能赚，而且最多赚权利金
		适用情形
			横盘整理/小幅上涨
			波动率低/波动率收窄/隐含波动率高
		基本应用
			赚权利金/期权价差收益
			履约平仓标的资产空头
			履约买进标的资产
		例如
			张三是买方，李四是卖方；看跌期权权利金是1；现在大豆市场价为3000
			张三要给李四大豆，李四给张三钱；这就是履约空头，变成多头
	盈亏平衡点口诀
		看涨用+，看跌用-

价差期权策略
	由==**类型相同**==的两个/多个期权头寸，采用==**不同的交易方向**==但是==**执行价格不一样**==的期权来构建的策略
	权利金总收支=卖出期权-买进期权
		例如卖出一个期权得到2￥/t,买同个期权1￥/t，那么权利金总收支=1￥/t
	价差期权的损益特点是==**潜在收益和潜在损失都比较小**==
	价差期权策略分类
		牛市价差、熊市价差、蝶式价差
	牛市价差
		和之前[[期货套利的基本策略]]中的价差套利一样
		买低价，卖高价
		接下来画图，盈利用实线；亏损用虚线
		首先看涨期权的买价是行权价，卖价是标的资产价
			假设现在市场上有个两个看涨期权，价格分别是5￥、7￥；标的资产价格为9￥
			那么5￥期权的内在价值是4￥；7￥期权的内在价值是2￥
			进而我们可知：==**行权价（即买价）越低的看涨期权，内在价值越高，进而期权费越大**==；那么在这里的==**期权费是净亏损**==，因为==**买在更下面，内在价值大的同时期权费也高**==
			当市场价格上升时，![[Pasted image 20260506221713.png]]
			在低价买的看涨期权盈利>在高价卖的看涨期权的亏损
			![[Pasted image 20260506222028.png]]
			我期望的市场价格是到卖看涨期权的那个价格之后就打住，不要再亏了
			因为我的盈利如图，是有限的，而且还要==**减去权利金净支出**==
			如果价格==跌到在低价买的期权行权价下面==了，那么==**只有期权费净支出**==
		现在我们来看看跌期权的情况
			假设现在市场上有两个看跌期权，价格分别为5￥、7￥；标的资产价格为3￥
			那么5￥期权的内在价值是2￥；7￥期权的内在价值是4￥；卖的价格越高，权利金（期权费）越大。
			而且方向始终是买低行权价的，卖高行权价的
			那么你在==**卖的时候得到的期权费>买的低价的要支付的期权费**==
			所以==**在期权费上是净收益**==
			注意和==**看涨期权价差套利的期权费的净支出**==区分开来
			![[Pasted image 20260506225725.png]]
			==**买入看跌，跌了就赚了；但是卖出看跌，跌了就亏了，因为买方盈利=卖方亏损**==
			于是行权价之间的亏损+权利金净收入
			==如果价格涨了，涨到高价的卖出的期权行权价的位置时；两个期权都不会行权==
		总结
			牛市价差期权策略一定是==**卖高行权价，买低行权价**==
			只不过==看涨期权的期权费是净亏损，但是看涨期权的行权价差值是盈利==
			==看跌期权的期权费是净盈利，行权价差值是亏损==
			同时，==**无论是看涨还是看跌，其盈利和亏损都是有限的**==
			==**看涨期权跌了不行权，就只有权利金净亏损==**
			**==看跌期权涨了不行权，就只有权利金净收益**==
	熊市价差期权
		一定是==**买高行权价，卖低行权价**==
		看涨期权
			例如现在市场上有两个看涨期权，行权价分别为5￥、7￥；标的资产价格为3￥
			因为==**看涨期权中，越高行权价，内在价值越低，权利金越低**==
				可以这样理解：==**你作为买方，如果真的涨了，行权价越低越有利，那么权利金就越多**==
				不过，==**现在是熊市，那么如果真的跌了，买高行权价的期权岂不是亏死**==
			那么现在是买高行权价的，卖低行权价的，那么==在看涨期权中低行权价=》高权利金==。因此==**权利金是净收入**==
			![[Pasted image 20260506235356.png]]
			如果价格上涨，高行权价买的盈利，而低价卖的亏了，这个多出来的亏损+权利金净收入
			如果价格下跌，那么==**两个期权都不行权，那么就只有净收入**==
		看跌期权
			买行权价高的，卖行权价低的
			看跌期权行权价越高，权利金越高
				因为如果真的跌了，我可以从市场上低价进货，以高行权价售出
			买意味着我付权利金，卖是收权利金
				在这里，我是==权利金净亏损，因为高行权价买的，付出的权利金更高==
			![[Pasted image 20260507225032.png]]
			盈亏如图，如果价格上涨，两个期权都不行权，价格下跌，只有权利金净亏损
			如果价格下跌，那么我的盈利>亏损
		总结
			熊市价差看涨期权：不行权就权利金净收益；行权时浮动亏损+权利金净收益就是我的总收益
	蝶式价差期权（了解即可）
		构建方法
			三个执行价格相邻的看涨/看跌期权，其价格分别是K1<K2< K3
			==**买行权价高和低的期权各1份，同时卖出2份的中间行权价的期权**==
			![[Pasted image 20260507231516.png]]
			由一个牛市价差和熊市价差期权策略组合
		目前只考构建方法
		空头蝶式价差期权是卖头，买中
		口诀
			==**蒸熊掌——正|熊涨==** + ==**买低卖高是牛市**==；
			**==熊市价差，结果价格上涨，那么只能得到权利金净收入**==
			负|牛涨
			牛市价差，价格上涨，权利金净支出+浮动盈利
			正|牛跌
			牛市价差，价格下跌，权利金净收入+浮动亏损
			负|熊跌
			熊市价差，价格下跌，权利金净支出+浮动盈利
			==**权利金首先看买卖，其次看内在价值**==
			同是买入，==**买看涨就是以行权价买标的，在市场上卖标的**==
			盈亏平衡点口诀
				==**看涨用+，看跌用-**==
	补充
		==卖出看跌=》目的是对冲标的资产空头；==
			==通过行权价买入资产，然后在市场上用这个多头对冲标的空头==
		==卖出看涨=》目的是对冲标的资产多头==
			==在市场上先进货，对冲标的空头，然后行权价卖出资产==

组合期权策略
	采用==**相同交易方向**==，但是一个是看涨期权，一个是看跌期权
	特点：潜在收益和潜在损失都很大
	类型：跨式期权、Strips期权、straps期权、宽跨式期权
	跨式期权
		相同交易方向且相同行权价
		多头跨式
			同时买进看涨期权和看跌期权
			因为都是买方，所以是权利金总收支为支出
			因为==买入一个看涨，一个看跌，相当于"什么话都给你讲完了"==
			总收支=浮动盈利-权利金支出
			![[Pasted image 20260508090558.png]]
			如图，当价格走到这两个中间区间之外的时候才能盈利，否则覆盖不了权利金支出
			当市场出现大幅波动的时候，才可能让价格走到两个区间之外才有盈利的机会
			假设权利金是4元，行权价为30元，那么在[26,34]里面是覆盖不了成本的
			![[Pasted image 20260508091706.png]]
			图像记忆方法
				指天为买，指地为卖；
				右手看涨，左手看跌（==**“手”指的是有斜率的线**==）
				![[Pasted image 20260508092140.png]]
		空头跨式
			同时卖出看涨期权和看跌期权
			因为都是卖方，权利金总收支为收入
			![[Pasted image 20260508093037.png]]
			如图，只有标的资产价在中间的时候，才能盈利
		损益特征
			多头跨式期权在标的资产价格大幅波动时盈利，价格小幅波动/价格不变时亏损
				其最大亏损是建仓时权利金总支出
			空头跨式期权在标的资产价格大幅波动时亏损，价格小幅波动/价格不变时盈利
				其最大盈利是建仓权利金总收入
		适用情形
			对应地，多头跨式期权又被称为做多波动率策略
			空头跨市期权又被称为做空波动率策略
	宽跨式期权
		相同交易方向但不同行权价
		但是==**看跌期权的行权价一定要<看涨期权的执行价**==，否则无法宽跨
		多头宽跨式期权
			买入看涨和看跌
			![[Pasted image 20260508094103.png]]
			假设期权费是4￥，那么上面中间的三个打叉的区间我都不能盈利，只有标的市场价到了
			假设权利金为4￥,买看涨的行权价为40￥，买看跌的行权价为38￥，那么在[34,44]里面是不会盈利的
		空头宽跨式期权
			卖出看涨和看跌
			![[Pasted image 20260508094836.png]]
			因为都是卖，所以在中间才能盈利

案例
1.【单项选择题】某投资者以100元/吨的权利金买入玉米期货看涨期权，执行价格为2200元/吨，期权到期日的玉米期货合约价格为2500元/吨，此时该投资者的理性选择和收益状况为()。
A.不执行期权，收益为0
B.不执行期权，亏损为100元/吨
C.执行期权，收益为300元/吨
==**D.执行期权，收益为200元/吨**==

买=》以行权价2200+100买标的；2500卖出

2.【单项选择题】买进看涨期权的损益平衡点是(
).
A.期权价格
B.执行价格-期权价格
C.执行价格
==**D.执行价格+权利金**==

B.执行价格-标的资产买价-权利金

==**D.执行价格-标的资产买价+权利金**==
