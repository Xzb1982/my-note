动词的重要性
	==**动词会影响句子的主干结构**==
		例如及物和不及物动词
		如果主谓结构已经可以完整地表意了，就不需要再多加一个宾语
		那么如果不能完整表意，那么从==主谓结构就衍生出了主谓宾，主谓宾宾补==
		==**主系表中的表语是解释主语的**==，系动词在英语里面意思为：linked verb，仅仅充当把后面的表语连住主语，从而进一步解释主语
	但是这还是没能解决一些问题
		例如the reading room , the running shoes , the flying birds
			这里的ing是动名词还是现在分词？
		又例如The performance was amazing.以及I was amazed at the performance.
			到底为啥事件用ing，人用ed
		再例如If it rains tomorrow, the competition will be canceled.
			为啥主将从现
		例如The train will leave in 3 minutes.
		The train is leaving in 3 minutes.

		Please remember to give me a call tomorrow. /I remember calling her last night
动词的类型
	实义动词
		表示动作、状态、行为
		一般来说==**词义完整**==，可==**单独作谓语**==，例如work,know,promise
		实义动词的分类维度
			==**按v后面要不要接Object（宾语），分成及物和不及物**==
				及物动词是需要接Object的，因此推断其本义不完整，后面要跟n/pron来补全语义，而且可用于被动语态
					例如we know the truth<=>the truth are known by us
				不及物动词后面不接宾语，本义已经完整，如果后面要接上宾语就要用prep和adv，而且不能用被动语态
					不能用被动语态的原因是主宾颠倒
					例如 They will arrive in Beijing next Sunday
			但实际上有些动词既可以作及物动词，又可以作不及物动词，这是由语义来决定的。
			按动词是否延续，分为延续性动词和非延续性动词
				延续性动词表示一种可以持续的行为过程/状态，可以和表示时间段的状语连用
					例如live,stay,work,clean,drive,read
					You can keep the dictionary for a week
				非延续性动词，表示行为/过程是瞬间完成的，动作一经发生就结束了，不能和表示一段时间的时间状语连用
					例如borrow,lend,finish,go,die,buy,arrive,join,come,catch等
					his uncle died three years ago
					如果我们要表示已经死去三年的这种状态，我们就要将died形容词化变成dead，这个时候我们还需要一个助动词be来帮助它变形
						His uncle has been dead for three years注意这里不是现在完成时
	系动词
		分为系动词be和半系动词
			半系动词：在某些情况下具有系动词的特点，即：能链接主语和表语；但也能同时作为实义动词的一类词
				因为系动词特别是be，一般没有实际意思，就是让表语帮助解释主语而已
				而半系动词具有表述功能，但是表述不全，需要表语辅助，进而说明主语的身份、状态、特征
				进而，有：==**系动词+表语 = 谓语**==
			例如
				be：表示主语性质、状态
				keep,stay,remain,continue,rest,lie：表持续
				seem,appear：表示看起来像
				smell,sound,taste,feel,look：表感官
				become,get,go,grow,turn,fall：表变化
				I kept quiet while he was reading.在他阅读时，我保持安静。
				Everything seemed possible.一切似乎都有可能。
				She became a teacher after graduation.她毕业后成为一名教师。
	助动词
		本身没有词义，但是帮助实义动词附加一些时态、语态、疑问、否定、人的主观意愿、语气的一些信息的动词，和实义动词一起作谓语
		正因如此，助动词有人称、单复数、时态的变化
		例如
			be do have shall will等
			其中，==**情态动词是助动词的一种特殊形式，因为情态动词还附加了人的主观情感和意愿这些信息，这是一般的助动词所没有的**==
		系动词和助动词的区分
			系动词是和表语一同解释主语的动词，并充当主语表语的“桥”，说明主语的身份状态特征
			但是助动词是配合实义动词变换形态、附加时态、语态、否定、疑问、人的主观意愿、语气这些信息的
				例如I love you => I do love you do表示强调=》Do I love you?do表示疑问
				其实I love you的助动词没有显现，在表示强调/疑问的时候才会单独显现出来帮助谓语/实义动词完成这项功能
		一些助动词的功能
			be ：
				 现在时态： am is are 
				过去时态：was were
				否定态：+not
			do：
				现在时态：do does
				过去时态：did
				否定态：+not
			have：
				现在时态：have has
				过去时态：had
				否定态：+not
				have可以作为实义动词，但是在助动词的情况下我们不翻译
				==**have一般帮助实义动词变成完成时态**==
			He is studying in the classroom.他正在教室里学习。
			I have finished my homework. 我已经完成了作业。
			Do you know the girl in red?你认识那位穿红衣服的女孩吗
	情态动词
		表示说话人的情感、态度、语气
			==**所有情态动词都属于非事实语气**==
		情态动词因为是助动词的一种特殊的形式，和助动词一样只能和动词原形一起构成谓语
		这也是为什么will 既是情态动词，也是将来时的一个助动词
			将来时基本等同于不确定语气，是非事实表达
	按照能不能做谓语，区分谓语动词和非谓语动词
		谓语动词
			我们将从三个维度来理解谓语动词：时态、语态、语气
			时态
				汉语里面我们用时间状语来表达时态，而不用动词的变形来体现时态
					例如我今天吃了，我明天吃，我昨天吃过了
				而英语里面，用动词的形态变化来体现时态
					时态一般有9种，最多可以达到16种
				那么时态其实是=时间+状态
				![[Pasted image 20260703135801.png]]
					在一般、进行、完成里面状态-状态结合，完成进行态就出现了
					在时间里面进行时间-时间结合，过去将来时就出现了
					这样就有16种进阶时态
				那么形式的变化就有16种
					时态的形式变化分成时间的和状态的两种形式的变化
					状态词的变化分成两种：==**在一般态的基础上，有进行态doing（同时也包含主动的含义）和完成态done（同时也包含被动的含义）**==
						进一步地，==**doing在实义动词瞬间发生的时候，可以被翻译为正在，如果是长时间发生，就翻译成一直**==
						done就有三种翻译的可能性：被，已经，已经被
						总结一下，状态的信息由do来承接，只有do，doing，done三种变化
					这里我们还要解释一下，==**动词原形是一种概念，实际上需要助动词帮助实义动词去体现这种概念**==
					然后附加了时间信息的承载词，由do be have来完成==**（即在状态的基础上表示时态的时候，时间信息由这些承载词承载，状态信息由doing done和一般态的do来承载）**==而这些词都是==**助动词**==
						现在时：do/does ; am/ is /are ; has / have
						过去时：did; was /were ; had
						将来时： will do ; will be ; will have
				然后我们来记忆一下
					

|     | 一般态     | 进行态                 | 完成态            |
| --- | ------- | ------------------- | -------------- |
| 现在  | does/do | be(am/is/are) doing | have/has done  |
| 过去  | did     | be(was/were) doing  | had done       |
| 将来  | will do | will be doing       | will have done |
				现在我们更进一步，进行叠加态和叠加时
					把过去时和将来时结合，我们得到过去将来时
					那么把承接词did和will结合就会得到would
					把进行态和完成态结合，我们得到完成进行态
					也就是承接词have done和be doing结合
						其中have和be都没有搭载时间信息，但是为了把状态叠加，我们要写出来
						而且==**完成进行态最终是帮助一个动作来完成表述，而不是因为信息由两个助动词搭载就说是两个动作来完成表述**==
						done和be结合，既要体现状态也要体现时间，就用been
						==因为doing是实义动词只不过用了一个do这个占位符==
						==而have用于做时间承载词，于是done和be要作为doing和have二者的桥梁。==
					现在我们应用一下
						完成进行态的一般时：has/have been doing
							它的过去时：had been doing
							它的将来时：will have been doing
						过去将来时的一般态：would do
							它的进行态：would be doing
							它的完成态：would have done
						那么过去将来时的完成进行态就是：would have been doing
					==**总结：完成进行态的状态承载词+承受动词：been doing；have为完成进行态的时间承载词**==
			语态
				被动语态：单纯的形式变化
				过去时：was/were done
				一般时：am/is/are done
				将来时：will be done
				从这里我们可以发现，被动语态是一种以be为时间承载词，但是用done来变动词形式的一个状态。
				也就是说被动语态虽然不是一种特殊的状态，但是我们可以把它放进时态表里面

|       | 一般态+主动   | 进行态+主动              | 完成态+主动          | 完成进行态+主动              | 被动态            |
| ----- | -------- | ------------------- | --------------- | --------------------- | -------------- |
| 现在    | does/do  | be(am/is/are) doing | have/has done   | has/have been doing   | was/were done  |
| 过去    | did      | be(was/were) doing  | had done        | had been doing        | am/is/are done |
| 将来    | will do  | will be doing       | will have done  | will have been doing  | will be done   |
| 过去将来时 | would do | would be doing      | would have done | would have been doing | would be done  |
注意：①表格最后的do只是一个占位符，最后用来填你想填的动词
②would have和would be的后面的have be保留原形是因为would已经变了不需要再变

续语态
				现在我们结合被动语态和其他状态拆解各个语法
				==**being done正在被 = be doing + be done = be +being done==**
				**==例如：the movie is being shown now.==**
				**==been done已经被 = have done + be donw = have been done==**
				**==例如The company's assets have been frozen这家公司的资产已被冻结==**
				**==been doing已经正在 = have been+ be doing = have been doing==**
				**==例如 Severe winds have been battering the coast狂风一直在海岸肆虐==**
				**==being doing是不存在的**==
			练习：过去完成时：过去时+完成态=~~had been done~~ had done
			过去完成进行时：过去时+完成进行态= did + have been doing =had been doing
			将来完成进行时：将来时+完成进行态= will do+have been doing =will have been doing
			过去将来完成进行时的被动语态：过去将来时+完成进行态+被动语态=would have been doing + 过去将来时的被动语态 would be done=would have been being done![[Pasted image 20260705131639.png]]
