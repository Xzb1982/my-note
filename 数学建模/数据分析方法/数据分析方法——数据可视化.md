散点图（关联关系）
	展示两个变量之间的关系
	查看变量间的相关性、回归分析
	![[Pasted image 20250831210430.png]]

直方图（数据分布例如高斯/正态分布）
	直方图适用于第一次的数据可视化
	因为直方图便于我们进行缺失值判定、同时可以引入一系列检验

热力图可以用矩阵散点图来替代
	![[Pasted image 20250831211437.png]]
	矩阵散点图的对角线呈现的是数据集自身的分布方式
	而其余的呈现的是数据集之间的相关程度

箱线图（异常值判定）
	箱线图是通过中位数+上下四分位点+上下须线来画出总的"箱"，然后离群值在外
	![[Pasted image 20250831212036.png]]

堆积图是柱状图的升级版
	![[Pasted image 20250831212148.png]]
	多个产品的比例

雷达图（评价模型常用）
	对几个个体进行排序、比较
	![[Pasted image 20250831212337.png]]

密度图
	直方图+概率密度

气泡图可以实现多维度的数据展示（可以作为热力图的一个平替，ai最喜欢生成热力图）
	![[Pasted image 20250901091352.png]]
	例如气泡的大小、颜色表示第三个维度

桑基图（适用于带问卷数据的数据集）
	![[Pasted image 20250901091948.png]]
	展示数据和流量
	展示选项之间的关联性

气象图（展示地理信息需要）

violin图（箱线图的升级版）

堆叠面积图（是堆积图的升级版）

网络图（展示空间上的毗邻关系）
![[Pasted image 20250901092411.png]]


时序图
	时间序列可以不是时间，也可以是随时间变化的压力
	![[Pasted image 20250901092636.png]]


词云图（展示词频）
	在问卷类竞赛用得多

时间轴（表示研究现状、发展历史时常用）

多项式回归图（和散点图类似，也是表示回归关系的）
	![[Pasted image 20250901093120.png]]


人口金字塔图仅适用于人口结构（性别+x）

极坐标图（展示角度与变量之间的关系，例如风速、风向）

多维尺度图
	表示多维数据降维之后的分布情况
	![[Pasted image 20250901093804.png]]
	同时通过不同颜色进行分类


双Y图
	在同一个图表内，展示两个不同尺度/数据量级的数据
	例如全市GDP的变化趋势和我家GDP变化趋势的比较
		因为全市GDP和我家的GDP不是一个量级的
		但是通过再构建一个新的y轴来适应我家的GDP
		从而将我家的GDP更好地可视化
	![[Pasted image 20250901094408.png]]


百分比堆积图
	展示数据之间的占比变化
	![[Pasted image 20250901094717.png]]

玫瑰图
	适用于周期性数据的分析

旭日图（适用于AHP层次分析法）

误差条图（条形图+置信区间）

仪表盘图（单指标可视化方法中更稀缺、更令人眼前一亮的可视化方式）

岭线图/山脉图
	多组数据的分布对比
	例如近七年的降雨量数据
