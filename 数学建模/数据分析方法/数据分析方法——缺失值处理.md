缺失值处理的两种常见思路
	- **靠整体水平来补**：不看顺序、只看总体分布，用均值/中位数/众数去替换缺失。适合“横截面/表格型数据”的零星缺失。
	- **靠邻近走势来补**：利用前后相邻观测的时间或位置关系去“估出”缺口（真正意义的插值）。适合“时间序列/连续型数据”的缺失。
四种高阶思路
	- **捐献者/相似样本法**：找“最像Ta的人”，用对方的值来补（KNN、热卡/冷卡）。
	- **建模预测法**：用其余变量建一个模型来预测缺失（回归、随机森林、深度学习、矩阵分解、专用模型）。
	- **似然/概率法**：把“缺了的数据”当成未知变量，配合统计模型一并估（EM、贝叶斯）。
	- **不只补一次**：考虑不确定性，**多重插补**生成多份数据再合并（MICE 等）。

简单、短缺口、单变量平滑 → 常用插值法；
多变量/长缺口/要做推断或强非线性 → 高阶插值法。

常用插值方法
	1. **均值 / 中位数 / 众数填补**
		- **直观理解**：把空格用“全班同学的平均分/中位数/最常见值”抹平。
		- **什么时候用**：零散缺失、多为表格数据；数值型用均值/中位数，分类型用众数。中位数更抗极端值。
		- **容易踩坑**：会**缩小方差**、弱化变量间相关性；大量使用会“过度平滑”。
	2. **线性插值（Linear）**
		- **直观理解**：两点一线；缺口就按直线连起来。
		- **什么时候用**：时间序列里**短、稀疏**的缺失且趋势近似线性（短时间温度、价格缓慢变化）。
		- **容易踩坑**：真实走势弯来弯去、或有突变/季节性时，会**低估尖峰、填错拐点**。
	3. **多项式插值（Polynomial）**
		- **直观理解**：用一条“弯曲的高阶曲线”穿过所有已知点。
		- **什么时候用**：观测点少、整体确有“多项式趋势”（平滑的大弧线）。
		- **容易踩坑**：阶数一高就**过拟合、震荡（Runge 现象）**；点多时计算也重，实务中较少首选。
	4. **样条插值（Spline）**
		- **直观理解**：把长曲线切成很多小段，每段用低阶多项式，段与段“平滑拼接”。
		- **什么时候用**：需要**平滑且能跟随局部变化**的时间序列（如传感器数据、平稳曲线）。
		- **容易踩坑**：对**异常值敏感**；节点/阶数选不好会“过平/过抖”。先去异常、再插值更稳。
	5. **前向/后向填充（Forward/Backward Fill）**
		- **直观理解**：缺口就“沿用上一次（或下一次）的读数”，好比“信号断了先沿用上次状态”。
		- **什么时候用**：**状态型**序列（库存、开关状态、账户等级），或短暂掉点的实时数据。
		- **容易踩坑**：**重复拉长旧值**，会掩盖真实波动；不适合开头/结尾以外的**长缺口**或强趋势数据。

高阶插值方法
	多重插补（Multiple Imputation, MI）
		- **一句话**：不是补一次就定死，而是**补很多次**得到多份完整数据，分别做分析，最后**合并结论**（方差更真实）。
		- **何时用**：做推断/回归/显著性检验，想让标准误、置信区间**不要被“补值过于确定”所低估**。
		- **常见做法**：MICE（逐变量循环回归、分类、PMM等）。
		- **易踩坑**：插补模型要**与最终分析模型“同等丰富”**（含同样的重要自变量、交互、非线性项），否则会传导偏差。
	KNN 插补
		- **一句话**：对每个缺口，找距离最近的 K 个“相似样本”，用它们的值（均值/加权均值/众数）来补。
		- **何时用**：变量不算太多、样本规模中等、关系**非线性**但不复杂。
		- **要点**：先**标准化**；混合数据类型要选合适的距离（如 Gower）；K 3–10 常见。
		- **易踩坑**：维度一高就“距离都差不多”（维数灾难）；对**异常值敏感**。
	模型预测插补（Model-Based）
		- **一句话**：把缺失变量当目标，其余变量当特征，**直接建一个预测模型**来补。
		- **何时用**：有稳定可学的关系、且你愿意为插补单独建模（线/非线都可）。
		- **典型**：线/逻辑回归、树模型、梯度提升等。
		- **易踩坑**：**确定性预测会缩小方差**；可用**随机扰动/PMM**缓解“过平”。
	热卡 / 冷卡（Hot-Deck / Cold-Deck）
		- **一句话**：**热卡**=在同一数据里找“相似的完整个体”随机抽一个值；**冷卡**=在外部“字典表/经验规则”里按分组给固定值。
		- **何时用**：**问卷/调查数据**、大量分类变量、需要保持**真实取值分布**。
		- **易踩坑**：热卡依赖**匹配分层是否合理**；冷卡易**一刀切**、对个体差异不敏感。
	回归插补（Regression Imputation）
		- **一句话**：用线性/逻辑回归等**点估计**来补缺。
		- **优点**：快、易解释。
		- **缺点**：**低估变异**（大家都贴回归线）；可在预测上**加随机误差**或用**PMM**保持分布。
	EM（期望最大化）
		- **一句话**：把缺失当“隐变量”，在**E 步**估其期望，在**M 步**最大化对数似然，**反复迭代**直到稳定。
		- **何时用**：可接受“数据≈某个分布族（常见是多元正态）”的**统计建模**场景；做参数估计（均值/协方差）。
		- **易踩坑**：对分布假设敏感；可能**收敛慢/陷局部最优**；结果仍是**单次估计**（不含插补不确定性）。
	随机森林插补（MissForest 等）
		- **一句话**：对每个有缺的变量，训练**随机森林**用其余变量预测缺失值，循环多轮直到收敛。
		- **何时用**：**混合类型数据**（数值+类别）、非线性/交互明显、缺失不算极端。
		- **优点**：鲁棒、往往**精度高**；无需复杂特征工程。
		- **易踩坑**：**计算时间长**；小样本+高维时也可能过拟合；仍是**确定性**，变异度略缩。
	深度学习插补
		- **一句话**：用**自编码器/变分自编码器/时序网络**等学到潜在结构，再重建缺失。
		- **何时用**：**超高维**、强非线性、样本量**很大**（图像/时序传感/电子病历等）。
		- **易踩坑**：训练复杂、需要调参和正则；**可解释性弱**；小样本不划算。
	矩阵分解插补（Matrix Factorization / 低秩）
		- **一句话**：把数据矩阵≈低秩分解 X≈UV⊤ 的重建值补缺。
		- **何时用**：**大规模稀疏矩阵**（推荐系统、问卷大表、基因表达），存在**低秩结构**。
		- **易踩坑**：对**噪声/离群**敏感；秩选太小→欠拟合，太大→过拟合；对**整块缺失**仍然困难。
	贝叶斯插补
		- **一句话**：给参数和缺失值都设**先验**，用后验抽样（如 MCMC）获得**分布**而非单点，再可自然**做多重插补**。
		- **何时用**：你**有先验**（专家知识/历史数据），或需要**严格的不确定性量化**。
		- **易踩坑**：先验选择影响大；计算重；需要一定贝叶斯功底。
