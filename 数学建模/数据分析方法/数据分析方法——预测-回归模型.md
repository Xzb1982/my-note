需要回归的情形
	当你要预测的是**连续数值**（金额、销量、房价、交通流量、旅游花费、能耗、评分等），且希望得到“自变量变化→因变量变化”的定量关系或数值预测时，就用回归。
回归预测的一般思路
	问题分析=》数据预处理=》特征分析=》模型选择=》参数估计=》模型评价=》结果预测
	在线性回归模型里面，一般参数估计就是模型求解部分
	在决策树模型里面，参数估计最后会得到一棵“树”
	更具体地，回归预测的具体每个步骤的目标包括：明确预测目标和影响因素=》处理缺失值、异常值和量纲差异=》分析变量之间的相关性，特别是线性相关性（特征分析）
		注意点1：关于非线性回归但是有单调关系的，我们可以关注以下几类相关系数
		①Spearman秩相关系数
			不是直线，但是自变量-因变量之间满足单调增加/单调减少的关系的，特别是：y=In(x)
			公式:$$ \rho_{spearman} = Corr(rank(X),rank(Y)) $$
			- 接近 1：强单调递增；
			- 接近 −1：强单调递减；
			- 接近 0：没有明显单调关系。
		②Kendall秩相关系数
			核心思路：把所有样本两两配对，判断两个变量对这两个样本的排序是否一致
			具体做法：
				假设有两个样本i,j，对应的观测值是(xi,yi),(xj,yj)
				比较：(xi-xj)(yi-yj)
				如果(xi-xj)(yi-yj)>0,则XY在排序方向上一致
				例如xi<xj,yi<yj; xi>xj,yi>yj这两种情况都属于“一致对”
				相反，(xi-xj)(yi-yj)<0,XY在排序方向上相反
				例如xi>xj,yi<yj; xi<xj,yi>yj;这种情况叫做：“不一致对”
				还有一种情况是xi=xj,yi=yj,这种情况叫做“并列对”
				现在，假设没有任何并列对
					设 $$一共有 \frac{n(n-1)}{2} 个样本对$$
					则:
					$$Kendall秩相关系数 \tau= \frac{一致对数量-不一致对数量}{全部样本对数量}$$
			含义
				Kendall τ =1，代表所有样本对排序完全一致
				接近1，则表示有强正向单调关系
				0代表，一致对和不一致对数量大致相同
				接近-1就代表是负向单调关系
				-1就是所有样本对排序完全相反
		注意点2：对于既没有单调关系，又非线性的，我们可以关注以下相关系数
			①针对y=x^2距离特征的相关系数——距离相关系数dCor
			其取值通常在 0 到 1 之间：
				- 越接近 1，依赖关系越强；
				- 等于 0，在通常条件下意味着相互独立；
				- 能识别 U 型、周期型、环形等非线性关系。
	这里还需要提到共线性问题
		关于共线性问题一般指的是自变量之间有相关关系，这会影响到因变量和自变量之间的相关关系，例如x1,x2,x3对y有相关关系，其中x1,x2的相关关系会加重三个变量对y的相关关系程度
		这个时候，我们需要采用逐步回归/岭回归/Lasso回归
	在特征分析和选型之后，就是用数据做拟合
	拟合完之后，接下来就是评价这个回归预测的效果
		核心：模型对未知样本是否预测得准确、稳定、可靠
		综合评价预测误差、解释能力、稳定性和残差特征
			这里的解释能力就例如说一组数据，然后喂给模型，然后模型另一端给出结果，你知不知道这个模型是怎么从输入，到达输出的，具体的步骤是什么，怎么解释
			就例如神经网络模型的解释能力是比较差的，因为你很难解释这个网络做了什么，然后得出来这个结果
			有时候这种情况也被称为“黑箱”
		常用的回归评价指标
			设：$$真实值：y_{i} ;预测值: \hat{y}_{i},样本量为n$$
			①均方误差MSE:
			$$MSE = \frac{\sum^n_{i=1}(y_{i}-\hat{y}_{i})}{n} $$
				真实值-预测值的差为残差
			②均方根误差RMSE：
			$$RMSE = \sqrt{ MSE } = \sqrt{ \frac{\sum^n_{i=1}(y_{i}-\hat{y}_{i})}{n} }$$
			对于1,2，两者越小越好
			③平均绝对误差MAE：
			$$MAE = \frac{\sum^n_{i=1}|y_{i}-\hat{y}_{i}|}{n}$$
			④平均绝对百分比误差MAPE：
			$$MAPE = \frac{\sum^n_{i=1}|\frac{y_{i}-\hat{y}_{i}}{y_{i}}|}{n} * 100\%$$
			MAPE的优势在于不受量纲的影响
			⑤决定系数R^2
				决定系数反映的是模型对因变量的变异解释程度
		我们可以将相对误差可视化，也就是：
			$$e_{i} = \frac{y_{i}-\hat{y}_{i}}{y_{i}}$$
			然后绘制ei的散点图
		常见的回归模型
			除了线性回归之外，还有KNN近邻回归（一般用不上而且效果不好）、然后就是决策树(随机森林方法)、XGBoost、支持向量机(SVR)、神经网络等
	决策树回归
		案例
			假设我们研究学习时间 x 与考试成绩 y 的关系：
			|学习时间 (x)/小时|考试成绩 (y)/分|
			|1|10|
			|2|12|
			|3|14|
			|4|30|
			|5|32|
			|6|34|
			决策树可能会在 x=3.5x=3.5x=3.5 的位置切一刀：
				学习时间 x ≤ 3.5？
		          /       \
		        是         否
			    预测12分    预测32分
			为什么分别预测12分和32分？
				左侧叶节点包含：10,12,14，求均值得到12
				同理，右侧叶节点包括30 32 34，求均值得到32
				所以得到：$$\hat{y} = \begin{cases}
				12, x\leq 3.5 \\
				32, x>  3.5
				\end{cases}$$
			为什么偏要选择3.5?
				决策树会尝试多个候选切分点，例如：1.5 2.5 3.5 4.5 5.5
				每次切分后，计算左右两组内部的平方误差：
				$$SSE = \sum_{i \in Left} (y_{i}- \bar{ y }_{Left})^2 + \sum_{i \in Right} (y_{i}- \bar{ y }_{Right})^2$$
				==**决策树选择使 SSE 最小的切分点**==
			随后，决策树可以按上面继续切分：
						 x ≤ 3.5？
		                /          \
		              是            否
		          x ≤ 1.5？      x ≤ 4.5？
		          /      \        /      \
		        10       13      30       33
		    随着树不断加深，每个叶节点中的样本越来越少，训练误差通常越来越低。但如果每个样本最终单独成为一个叶节点，模型就会直接记住训练数据，产生过拟合。
		通过不断选择变量和切分点，把样本空间划分成若干个区域
		==**然后在每一个叶节点里面，使用该节点样本目标值的均值/中位数作为预测值**==
		适用范围
			因变量和自变量之间存在明显的非线性关系
			自变量之间存在复杂的交互作用
			同时存在多个分段的规律
			希望得到“如果-那么"的决策规则/自变量存在分类变量或离散变量时
				因为不需要对离散/分类变量进行处理
		优点
			能处理非线性关系
			能自动识别变量交互
			不需要对变量进行标准化
			对变量单调变化不敏感
			模型规则较为直观
		缺点
			单棵树容易过拟合
			数据稍有变化，树的结构就容易发生改变
			预测结果是分段常数，不够平滑
			对训练范围以外的情况缺乏可靠外推能力
			决策树深度过大会记住训练数据的噪声
	随机森林回归
		==**为了解决单棵决策树容易过拟合的问题，我们需要多棵树进行决策，这就是随机森林**==
		过程
			假设第b棵树的预测值为hat(y)_ b,则最终预测结果为：
			$$\hat{y} = \frac{1}{B} \sum^B_{i=1} \hat{y}_{b}$$
			多棵树的预测结果取平均，可以降低单颗决策树的方差，改善预测稳定性
		优点
			非线性拟合能力更强
			比单颗决策树更加稳定
			通常不容易因为一棵树而导致过拟合
			不需要进行特征标准化
			可以计算特征重要性
			对异常值和噪声有一定的稳健性
		缺点
			单棵树的解释性很好；但是随机森林不是，其解释性要弱于线性回归和单棵树
			占用内存多而且时间长
			预测出来的结果不容易写成单一公式
			特征重要性不能直接解释成为因果关系
			对训练范围以外的情况缺乏可靠外推能力

	SHAP
		SHAP的全称是 **SHapley Additive exPlanations**，即“基于夏普利值的加性解释方法”。
		对某一个具体样本，模型为什么会预测出这个结果？每个自变量分别把预测值提高或降低了多少？
		SHAP把模型的预测结果分解为：
		$$f(x) = E[f(X)]+ \sum^p_{j=1} \phi_{j}$$
		其中：$$\begin{align*}
		f(x):是模型对样本的预测值\\
		E[f(X)]:是模型在背景数据上的平均预测值，又称基准值\\
		\phi_{j} : 是第j个特征的SHAP值\\
		\phi_{j} >0时，代表这个特征将预测值向上推动；反之，当\phi_{j}<0时，这个特征将预测值向下拉\\
		|\phi_{j}|越大，这个特征对这个预测值的影响越强
		\end{align*}$$
		所以，我们可以将这个公式理解为：
			==**当前预测值 = 模型的平均预测值 + 每个特征的贡献**==
			在随机森林回归中，==**SHAP常用于解释每棵树共同形成的预测结果**==，并可以进一步形成全局特征重要性图。
		案例
			假设使用随机森林预测考试成绩，只有两个自变量：
			x1：学习时间；
			x2：是否参加辅导班。
			现在有一名学生：每天学习6小时；参加了辅导班。
			随机森林最终预测他的考试成绩为：f(x)=80
			模型对全部学生的预测平均成绩为：E[f(x)] = 60
			我现在想得到：从平均水平60分提高到80分，这20分应该怎样分配给“学习时间”和“辅导班”？
			假设我们现在只知道：
				|已知的信息|模型预测成绩|
				|什么信息都不知道|60|
				|只知道每天学习6小时|72|
				|只知道参加辅导班|65|
				|两个信息都知道|80|
			这里的“什么信息都不知道”，就是用模型的平均预测值作为基准：f(∅) = 60
			好了，我们现在先要引入：[[自变量之间的相关关系对因变量的影响]]

线性回归模型
	首先，线性回归模型必须满足四种对误差项的基本假设
		独立性
			有N个样本，每个样本都有一个误差，而这些误差都是相互独立的；也就是第i个样本的误差项不应当影响到第j个样本的误差项
			在时间序列中，这种性质很容易被破坏，比如用每日气温预测用电量：
				- 今天实际用电量高于预测值；
				- 明天也很可能继续高于预测值。
			检查方法：
				- 残差随时间变化图；
				- Durbin–Watson检验；
				- Breusch–Godfrey检验；
				- 残差自相关函数 ACF；
				- 面板数据中的组内相关检验。
			注意：“样本之间独立”和“每个样本的误差独立”密切相关
		同方差
			无论自变量X取何值，每个样本的误差项的波动程度都大致相同
			==**例如所有样本的误差都在 ±100 之间，属于同方差；==**
			**==即使误差都比较大，只要其波动程度稳定，仍可能满足同方差性。**==
			检查方法
				- 残差—拟合值散点图；
				- Breusch–Pagan检验；
				- White检验；
				- Goldfeld–Quandt检验。
			解决方案
				- 对变量取对数；
				- 使用异方差稳健标准误；
				- 加权最小二乘法；
				- 补充遗漏变量；
				- 使用适当的广义线性模型。
		正态性
			给定自变量Xi，则误差项应当近似服从于均值0，方差为σ^2的正态分布
			这里包含三层意思
				- 误差大多数集中在0附近；
				- 较大的正误差和负误差都比较少；
				- 正负误差大体对称。
			在这里，正态性的作用主要在于：
				- 小样本下构造精确的 t 检验（你可以把t检验理解为小样本情况下，因为用不了正态分布，所以只能近似用t分布拟合正态分布）；
				- 小样本下进行 F 检验；
				- 构造准确的置信区间和预测区间。
			样本量较大时，根据中心极限定理，即使误差本身不完全正态，回归系数的抽样分布通常也会逐渐接近正态。
			检查正态性的方法
				- 残差直方图；
				- 残差密度图；
				- Q-Q图；
				- Shapiro–Wilk检验；
				- Jarque–Bera检验。
			注意：大样本时，极小的分布偏差也可能导致显著拒绝正态性。此时通常更应关注Q-Q图、异常值和推断方法是否稳健。
		0条件均值
			在给定自变量Xi以后，误差项的平均值为0；
			即误差项中不能含有与 X 系统相关的遗漏因素。也就是你找的和Y有关的自变量必须穷尽
			因为误差项是自变量“搞剩下的”，那么如果还有遗漏的因素，那么这里的误差项就包括了这个遗漏因素
			此时会产生==**内生性和遗漏变量偏误**==。这个问题会直接导致回归系数有偏，通常比异方差或非正态更严重。
	线性回归的参数估计过程
		使用最小二乘法，使得残差平方和达到最小
	线性回归的假设检验
		单变量检验——针对每个变量，检验其影响，而且具体化为是怎么样的影响
		F检验——从整体来看，我们的模型有没有效果
	之后，我们还需要看拟合优度
		主要看决定系数R^2和调整后的R^2
	模型诊断
		残差图（看同方差，自相关）
		QQ图（看正态性）
		Cook距离图（样本有无异常值）
	共线性问题的处理
		诊断方式：方差膨胀因子VIF，对于一个自变量来说，VIF值>10说明有共线性问题
		在0到10之间说明共线性问题不大
		一般来说，如果某一个变量Xi对Y没有影响（也就是系数为0）理论上我们会剔除掉他，但是实际上我们不会这样做
			在有共线性问题，而且单变量检验不显著的情况下，我们会考虑去做逐步回归/岭回归/主成分回归
			


回归类型判断
	1. **因变量类型/分布**
		- 连续且近似正态 → 往普通线性/树/集成方向想
		- 计数/比例/偏态很强 → **GLM**（Poisson、Binomial 等）或先做变换（log、Box–Cox）
	2. **你更看重“可解释”还是“预测精度”**
		- 要解释清楚 → **线性回归/GLM**（加 **Ridge/Lasso/ElasticNet**）或**浅层决策树**
		- 追求精度 → **树模型家族**（Random Forest / GBDT / XGBoost / LightGBM）
	3. **样本量 n 与特征数 p 的关系**
		- p ≪ n（样本多、特征不算多）→ 线性、树、集成都行
		- p ≈ n 或 p > n（高维/稀疏）→ **Lasso/ElasticNet、线性核 SVR**（做特征选择/收缩）
	4. **关系是否明显非线性、存在强交互**
		- 线性为主、轻微弯曲 → **多项式回归**或在线性里加交互项
		- 非线性强/交互复杂 → **树/GBDT/XGB/LightGBM、核 SVR、GPR**
	5. **异常点/噪声多吗**
		- 多噪声/多异常 → **Random Forest、GBDT** 更稳；Ridge 比 OLS 稳
		- 噪声少且边界清晰 → **SVR、kNN** 也可
	6. **是否需要“不确定性估计”**
		- 需要区间/置信度、样本不大 → **Bayesian Regression、GPR**
		- 不强求 → 常规方法 + 交叉验证/Bootstrap 评估
	7. **算力与时延**
		- 训练/推理都要快 → **线性、Ridge、浅树、LightGBM（小树+早停）**
		- 数据巨量/高维表格 → **LightGBM/XGBoost** 优先
		- 很小数据、想要光滑函数近似 → **GPR**（但扩展性差）

非线性回归
	**目标还是回归**：给出数值型 y，用特征 x预测其期望 E[y∣x]  
	**“非线性” **指：y 与 x 的关系不是一条直线/超平面**，或模型对参数是非线性的。一般写作：
		y=f(x;θ)+ε
	训练的本质仍是**最小化某种损失**（常用平方误差）并**控制复杂度**（正则化/先验），得到能泛化的 f^\hat。
	通用回归方法
		- 选函数空间 F （允许弯曲：基函数展开、核方法、树、邻域平均、GP等）
		- 选损失 L（平方误差、ε-不敏感损失、负对数似然…）
		- 加正则/先验（防过拟合）
		- 求解（梯度法、二次规划、贪心分割、线性代数）
		- 用 f^(x)\hat （有的还给不确定度）
	- 多项式：**线性解正规方程** ⇒ y^=ϕ(x)⊤β^
	- SVR：**解二次规划（核技巧）** ⇒ y^=∑(α−α∗)k(xi,x)+b
	- 决策树：**最小化叶内 SSE 的贪心切分** ⇒ 预测=叶均值
	- kNN：**邻域加权平均** ⇒ 预测=近邻的平均
	- GPR：**贝叶斯核法，后验闭式** ⇒ 输出均值+方差

正则化回归
	**正则化（regularization）**= 在“拟合误差”之外，再加一个**复杂度惩罚**，限制参数自由度，让模型更稳定、可泛化。  
	统一写法（以线性模型为例）：
		![[Pasted image 20250829161108.png]]
	- **正则化是一种“控制复杂度”的原则**，和“线性/非线性”是**正交**概念。
    - 线性模型可正则化（岭/Lasso/弹性网）。
    - 非线性模型也在用正则化
	- **共同规律**：目标基本都是：数据拟合+λ×复杂度

集成回归
	**把很多个弱回归器（通常是小决策树）加起来，得到一个更强、更稳的预测函数**。
	![[Pasted image 20250829161558.png]]
	随机森林
		Bagging（装袋法）— 随机森林的做法
			目标：**降方差，抗过拟合**。  
			步骤：
			1. 对训练集做 **bootstrap** 抽样得到 MMM 份样本。
			2. 每份样本上训练一棵回归树；每个结点再**随机抽取部分特征**做最优切分。
			3. 预测时对所有树的输出**取平均**
			树多、深度适中；可用 **OOB（袋外）误差**做验证；特征重要性来自分裂带来的误差下降。
	Boosting（提升法）— GBDT / XGBoost / LightGBM / AdaBoost
		目标：**降偏差，逐步修正残差**。  
		加法模型 + 函数空间的梯度下降：
	它与线性/非线性/正则化的关系
	- **函数形状**：集成树学到的是**分段常数/分段线性**的非线性函数，能自动捕捉特征**交互**和**非线性**。
