把一堆高维特征整理、压缩到更少的“轴/部件/主题”上：信息尽量不丢、计算更快、可视化更清。

这里的**“有/无监督”说的是**你在**学习降维映射的那一步**有没有用到 y（标签）

用到y的数据降维线性方法
	**LDA（线性判别分析）—找“分得最开的投影”**（监督式）
		- 想法：同时“拉大类间距离、压小类内距离”的投影方向。
		- 输出：可用于降维到 ≤(类别数-1) 维的轴；常作分类前的投影。
		- 何时用：已知标签，想要更好地区分/可视化各类。
		- 注意：假设较强（各类协方差相近、近似高斯）；别和“LDA主题模型”混淆。

统计里说“**非高斯**”（non-Gaussian）通常就是指**不服从高斯/正态分布**（单变量不正态、多变量不多元正态）

没有用到y的数据降维线性方法
	PCA（主成分分析）—找“变化最大的方向”
	**ICA（独立成分分析）—把“混在一起的信号”拆开**
		- 想法：把观测分解成若干**相互独立且非高斯**的成分（盲源分离）。
		- 例子：鸡尾酒会里分离不同人声。
		- 何时用：信号分离、脑电/语音等；也可做特征提取。
		- 注意：对噪声和独立性假设较敏感，结果可交换/符号不定。
	**NMF（非负矩阵分解）—把“整体”拆成“可加的部件”**
		- 想法：X≈W×H，两个矩阵都非负；得到“加法式的部件表示”。
		- 例子：人脸=眼睛+鼻子+嘴巴的加和；文本=若干主题的加权。
		- 何时用：数据天然非负（计数、像素、金额），且追求“可解释的部件”。
		- 注意：需要非负；解依赖初始化，常需多次运行取稳健结果。
	**因子分析（FA）—建模“潜在因素+噪声”**
		- 想法：观测=少数潜在因子的线性组合+特定误差；强调“共同度”。
		- 适用：心理/问卷、经济指标等“想解释背后的潜变量”。
		- 对比PCA：PCA是**方差压缩**（描述性），FA是**生成式噪声模型**（解释性更强）。
		- 注意：模型假设强，需要做适配性检验（KMO、Bartlett等）。
	**随机投影（RP）—“掷筛子”也能大致保距离**
		- 想法：用随机矩阵把高维投影到低维；依据 Johnson–Lindenstrauss 引理，距离大致保真。
		- 何时用：超高维（文本、稀疏向量）快速粗降维，做基线或预处理。
		- 注意：不可解释、结果有随机性，但**快**且常“够用”。
	**Kernel PCA（核PCA）—先“弯到高维”，再做PCA**
		- 想法：用核函数把数据隐式映到高维，在线性PCA中捕捉**非线性**结构。
		- 何时用：明显的弯曲流形（环/螺旋），线性PCA分不开时。
		- 注意：核与参数选择敏感；样本多时计算/存储昂贵。

非线性数据降维方法
	**t-SNE**（最适合做“图很漂亮”的聚类可视化）
		- 想法：让近邻在低维仍然靠近，远处的点“尽量别挤在一起”。
		- 优点：簇分得开、看点清楚。
		- 限制：**全局结构不可靠**（簇与簇之间距离别解读）；难嵌入新样本；对参数/随机种子敏感。
		- 小招：先用 PCA 降到 30–50 维，再做 t-SNE；`perplexity` 常在 5–50 试几档。
	**UMAP**（更快、更能兼顾一点全局）
		- 想法：把高维“拓扑结构”尽量在低维复原。
		- 优点：速度快、可 `transform` 新样本；常比 t-SNE 更保留大结构。
		- 关键参：`n_neighbors`（越小越强调局部，越大越看全局），`min_dist`（簇的“挤”或“松”）。
	**MDS**（让**成对距离**在低维尽量不变）
		- 想法：找一个低维配置，使得低维距离≈高维距离（或秩次）。
		- 优点：当你有“距离/相似度矩阵”时通用、直观。
		- 限制：对噪声/大规模不友好；“度量/非度量”两类要分清。
		- 旁注：**经典 MDS**在欧氏距离下等价于对距离矩阵做一次“PCA”。
	**Isomap**（保留**测地线/弯曲**距离）
		- 想法：邻居图上算最短路径当作“弯曲距离”，再做 MDS。
		- 优点：能揭开“弯曲流形”（像瑞士卷）。
		- 限制：图不连通或有噪点会崩；`k` 选太小断、太大又直线化。
	**LLE**（保局部线性重构关系）
		- 想法：每个点都可由邻居线性重构；在低维也要保持同样的重构权重。
		- 优点：对“局部线性”的流形很灵。
		- 限制：对噪声/参数敏感；非流形或稀疏边界会出问题。
		- **Hessian LLE**：加入二阶信息，曲率更准，但更脆弱更慢。
	**Spectral Embedding / Laplacian Eigenmaps**（图拉普拉斯特征向量）
		- 想法：最小化图上相邻点的“拉伸”，等价求图拉普拉斯的前几个特征向量。
		- 擅长：**图/网络数据**或天然以图形式给出的相似度。
		- 限制：需要解特征分解，内存 O(n2)。
	**Diffusion Maps**（看“扩散距离”，多尺度结构）
		- 想法：在邻居图上做随机游走，若两点在很多路径上“互相可达”，在低维就近。
		- 优点：稳、能反映多尺度几何。
		- 限制：核宽度/时间尺度要调；计算偏重。
	**Autoencoder（自动编码器）**（用神经网络学非线性压缩）
		- 想法：神经网络把 XXX 压到瓶颈层（低维表示），再重构回来。
		- 优点：可扩展到大数据；能自然 `encode` 新样本；可做**去噪/稀疏/变分**等扩展。
		- 限制：训练与调参成本高；结果受网络结构/正则化影响。


- **通用基线**：标准化 → **PCA**。满意就停。
    
- **已知标签要分得更开**：**LDA**（或 PLS / supervised PCA）。
    
- **非负且求“部件化”解释**：**NMF**。
    
- **信号分离/独立性**：**ICA**。
    
- **可视化看簇（小样本）**：**t-SNE**；（中大样本或需 transform）**UMAP**。
    
- **保弯曲几何距离**：**Isomap**；**局部线性**：**LLE/Hessian LLE**。
    
- **图/网络相似度**：**Spectral Embedding / Diffusion Maps**。
    
- **深度表征+可嵌入新样本**：**Autoencoder/Variational AE**。
    
- **线性不够但仍想可变换新样本、参数可控**：**Kernel PCA**（注意算力与参数）。
