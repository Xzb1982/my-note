无监督学习
	- **目标**：在**没有标签 y** 的情况下，只凭特征 **X** 去**发现结构**：哪些样本彼此像、数据可能躲着哪些“形状/规律/主题”、有没有离群点等。
	- **产物**：分组/簇、低维表示、异常分数、主题/规则、密度模型……而不是“已知标签或具体数值的预测”。
	**没有已知的因变量 y**，只拿到特征 X，目标是**从数据里自己发现结构/模式**，而不是去预测一个给定的 y。
	**聚类**：把样本自动分组
	    - **K-Means**（球状簇、需定 K）、**GMM**（高斯混合，可软分配，AIC/BIC 选 K）、**层次聚类**（可画树状图）、**DBSCAN/HDBSCAN**（按密度找簇，能识别噪声与非球形簇，不用定 K）。
	**降维 / 表征学习**：压缩到少数“主轴”，便于可视化/建模
		- **PCA**（线性、解释性好）、**NMF**（非负、部件化解释）、**t-SNE/UMAP**（非线性可视化）、**自编码器**（非线性重构）。
	**异常检测**：找“不像其他人”的点
		- **Isolation Forest**、**One-Class SVM**、**LOF/KNN-based**、**KDE**（密度估计）。
	**主题/规则**：从离散事件或文本里挖语义或共现
		- **LDA 主题模型（Latent Dirichlet Allocation）**、**Apriori/FP-Growth** 关联规则。
	**协同过滤/矩阵分解**（推荐）：**SVD/NMF** 等。
	用无监督学习的情形
		- 没有标签但想要**洞察**或**先分群再运营**（客户细分、用户画像）。
		- 作为**监督学习的前置**：降维、去噪、造特征、去异常。
		- 你怀疑数据存在**结构变化/制度分段**（疫情前后、季节/区域模式）。
		- 需要**可视化**高维数据，看分布/重叠/离群。

降维和无特征提取
	PCA（[[评价模型重述]]）
		**PCA不看 y，只研究 X 的相关结构**，因此本质是**无监督**；但你可以把它产出的“主成分/得分/权重”拿去做排序、定权，或接到任何监督模型前面用。
	t-SNE
		把“谁和谁是邻居”的关系搬到 2D/3D，**专注可视化**局部簇。
		**典型用途**：看高维样本有没有自然分群/异常点；演示图最漂亮。
		**常见坑**
			- **全局距离不可信**：簇与簇之间远近不能做定量解读，簇大小也不代表样本多少。
			- **难复现**：对随机种子和参数敏感；大样本计算量高。
			- **不适合当特征**：降到 2D 的坐标喂给下游模型，稳定性通常差。
		**搭配**：仅做**探索/展示**；聚类请改用 **UMAP + HDBSCAN/DBSCAN**
	UMAP
		基于流形/拓扑的嵌入，**兼顾局部+一定全局**，既能看图，也可把低维表示当特征。
		**典型用途**：可视化 / 前置降维；配 **HDBSCAN** 做稳健聚类。
		**常见坑**
			- 仍有随机性；`n_neighbors` 太小会把连续结构撕裂成碎簇。
			- 维度太低（2D）用于建模可能损失可分性；下游用 **5–20 维**更稳。
	自编码器
		用神经网络学“压缩后还能还原”的低维表示；**非线性降维/去噪/表征**。
		**典型用途**：数据复杂、非线性强、特征很多时的**无监督特征学习**；**重构误差**做异常检测；变体有 **Denoising AE、VAE**。
		**常见坑**
			- 数据少/噪声大→容易学到“记忆体”，需强正则+早停。
			- 可解释性比 PCA/因子分析弱；超参较多，需验证集调参。
	ICA独立成分分析
		把混合信号拆成**统计独立、非高斯**的“源”；更像“分离”而不是“压缩”。
		**典型用途**：EEG/音频/图像通道分离；经济金融里也用于找“独立冲击”。
		**常见坑**
			- 对噪声和异常值敏感；**独立性**假设不满足会失效。
			- 不是按“重要性”排序，解释和筛选要靠业务含义。
		**搭配**：**ICA → 回归/分类**（把独立源当驱动因子）；或与 PCA 先后配合（PCA 去噪 → ICA 分离）。
	FA因子分析
		假设观测变量由少数**潜在因子**线性组合 + 各自**独特误差**构成，用来**找潜在维度并命名**。
		**典型用途**：构建“发展因子、收入因子、可达性因子”等可解释维度；比 PCA 更像“统计模型”。
		**常见坑**
			- 对样本量有要求（经验：**每个变量≥5–10 个样本，总样本≥100**）。
			- 假设较强：近似正态、线性、误差独立；强偏态要先变换。
			- 别把“特征值>1”机械当作唯一选因子依据，结合碎石图/并行分析。
		**搭配**：因子得分做**评价模型**；因子载荷回分配**指标权重**；也可与 **SEM** 做验证性因子分析（CFA）。
