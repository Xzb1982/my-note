你可以把“函数名”先当成“变量名”来看。  
因为在 Python 里，函数名称本质上也是一个标识符。  
标识符只要满足命名规则，这个函数名在语法上就是合法的。

最核心的规则只有这几条：

函数名只能由字母、数字、下划线组成。  
函数名不能出现运算符号。  
函数名不能以数字开头。  
函数名不能用关键字。
