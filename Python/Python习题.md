第44题怎么从常识推出来

题目要画“边长为200的正菱形，4个内角均为90度”。

你从常识看，这其实就是一个旋转了 45 度的正方形。  
因为菱形四边相等，而“4 个内角都 90 度”的菱形，本质上就是正方形，只是题图里把它斜着放了。

题目给的代码是：

import turtle  
turtle.pensize(2)  
d = ____(1)____  
for i in range(4):  
    turtle.seth(d)  
    d += ____(2)____  
    turtle.fd(____(3)____)

先看每一行在干什么。

turtle.pensize(2)

意思是把画笔变粗一点，这和形状本身没关系。

关键是这三行：

d = ____(1)____  
for i in range(4):  
    turtle.seth(d)  
    d += ____(2)____  
    turtle.fd(____(3)____)

你可以把它翻译成人话：

先给一个初始方向 d。  
一共画 4 次。  
每次先把方向设置成 d。  
然后把 d 改成下一个方向。  
再向前走一条边。

现在来推空。

第一空 `d = ____(1)____`

题目给的参考思路里已经有一个类似版本：

d = -45  
for i in range(4):  
    turtle.seth(d)  
    d += 90  
    turtle.fd(200)

为什么是 `-45`？

因为普通正方形如果第一条边朝右，是水平的。  
但题图要求箭头和图形是斜着的，所以第一条边应该先朝右下方 45 度。  
在 turtle 里，这个方向可以写成 `-45`。

所以第一空是：

-45

第二空 `d += ____(2)____`

你每画完一条边，下一个方向应该转多少度？

正方形每一条边和下一条边互相垂直，所以方向每次变化 90 度。  
因此这里应该加 90。

所以第二空是：

90

第三空 `turtle.fd(____(3)____)`

题目已经说边长为 200，所以前进距离就是 200。

所以第三空是：

200

完整答案：

import turtle  
turtle.pensize(2)  
d = -45  
for i in range(4):  
    turtle.seth(d)  
    d += 90  
    turtle.fd(200)

题目说，输入 4 个数字，分别是：x0, y0, x1, y1
这表示两个点：(x0,y0), (x1,y1)
题目要你求这两个点之间的距离。
距离公式是：

$$r=\sqrt{(x_1-x_0)^2+(y_1-y_0)^2}$$

先把题目翻成程序动作:
	第一步，输入一行内容，比如：0 1 3 5
	第二步，把这一行按空格拆开，得到 4 个数。
	第三步，代入距离公式计算。
	第四步，保留 2 位小数输出。

你看参考代码里有一句：
ntxt = input("请输入4个数字(空格分隔):")
这说明输入进来的最开始其实是一个字符串，不是 4 个独立变量。  
所以要先拆分,于是用split函数：nls = ntxt.split()
`split()` 的作用就是“按空格切开”。
例如："0 1 3 5".split()
会得到：['0', '1', '3', '5']

分开来还是不够，因为还是字符串的数据类型，所以接下来要把它们变成数字：
x0 = eval(nls[0])  
y0 = eval(nls[1])  
x1 = eval(nls[2])  
y1 = eval(nls[3])

然后代入公式：
r = pow(pow(x1-x0, 2) + pow(y1-y0, 2), 0.5)

最后保留 2 位小数输出：

print("{:.2f}".format(r))

这里的 `:.2f` 意思是保留 2 位小数。

y1 = eval(nls[3])

我的答案：n = eval(input("请输入一个整数："))
for i in range(1,n):
    for j in range(1,n):
        if j>=i:（注意是j>=i不是i>=j）

- ==**`i` 表示第几行==**
- **==`j` 表示这一行里当前检查到哪个数字**==

第41题逐行理解
第一行：
n = int(input())

第二行：
for i in range(1, n):
这一行表示：
让 `i` 从 1 取到 `n-1`。
如果 `n = 8`，那么 `i` 会依次取：
1, 2, 3, 4, 5, 6, 7
这里的 `i` 可以理解成“第几行”。
也就是：
- `i = 1` 表示第1行
- `i = 2` 表示第2行
- ...
- `i = 7` 表示第7行

第三行：
for j in range(1, n):
这一行表示：
每一行里，再让 `j` 从 1 取到 `n-1`。
如果 `n = 8`，那么每一行里，`j` 都会依次尝试：
1, 2, 3, 4, 5, 6, 7
这里的 `j` 可以理解成“这一行里当前准备打印哪个数字”。


第四行：
if j >= i:
这一句是关键。
它的意思是：
只有当 `j` 大于等于当前行号 `i` 时，才打印它。
比如第1行时，`i = 1`：
- `j = 1`，满足 `j >= i`
- `j = 2`，满足
- `j = 3`，满足
- ...
- `j = 7`，满足
    

所以第1行打印：
1 2 3 4 5 6 7
再比如第3行时，`i = 3`：
- `j = 1`，不满足
- `j = 2`，不满足
- `j = 3`，满足
- `j = 4`，满足
- ...
- `j = 7`，满足
所以第3行打印：3 4 5 6 7

第五行：
print(j, end=' ')
这一句表示：
把 `j` 打印出来，但打印完不要立刻换行，而是在后面接一个空格。
这里最重要的是 `end=' '`。
默认情况下，`print()` 打印完会换行。  
但这里我们不想每打印一个数字就换行，我们想让同一行多个数字连在一起，所以改成空格。
比如这一行会形成：
1 2 3 4 5 6 7
而不是：
1  
2  
3  
4  
5  
6  
7

第六行：
print()
这一句什么都不打印，只负责换行。
它放在外层循环里面的意思是：
每一行的数字打印完以后，程序换到下一行。
所以整个图形才能一行一行排出来。

第2行

print("{0:{1}^30}".format("PYTHON", a))

这一行是格式化输出，乍一看很吓人，但拆开就不难。
重点在：
"{0:{1}^30}".format("PYTHON", a)
这个意思是：
把第 0 个参数 `"PYTHON"` 按照“第 1 个参数 `a` 作为填充字符、宽度 30、居中”的规则格式化。

这个格式串怎么拆
{0:{1}^30}
你把它拆成三层看。
第一层，`0`  
==**表示要格式化的是第 0 个参数，也就是 `"PYTHON"`。**==
第二层，`{1}`  
==**表示填充字符取第 1 个参数，也就是 `a`。**==
第三层，`^30`  
`^` 表示居中对齐，`30` 表示总宽度是 30。
所以整句连起来就是：
“把 `PYTHON` 用 `a` 这个字符填充，放在宽度 30 的中间。”
