编程语言的执行方式
	程序设计，又称为编程
	源代码
		人类可读的，采用某种语言编写的程序
	目标代码
		计算机可以直接执行的，人类难以理解的代码（例如机器码）
	将人类通过某种语言编写的代码，转换成机器可以读取的方式有两种
		①编译
			将源代码一次性转化成目标代码的过程
			![[Pasted image 20250925153853.png]]
			源代码通过编译器软件变成机器可读码
		②解释
			将源代码逐条转化且同时运行为目标代码的过程
			![[Pasted image 20250925154034.png]]
	而根据这两种形式衍生出了两种不同的编程语言
		通过编译来执行的编程语言是静态语言
			C/C++,Java
			C语言
				学习的是：指针、内存、数据类型（例如整数数据类型）
				C语言的本质是帮助我们理解计算机的系统结构
				在C语言诞生的时候，计算机不是很发达，内存很小，在编写语言时需要照顾到内存，不然小内存支撑不了大程序
			Java
				解决一台计算机上的程序在其他计算机上也能运行的问题
					采用“面向对象”和跨平台的方式来解决这个问题
				Java可以帮助我们理解程序设计过程中的主客体关系
				其学习内容是：对象、跨平台、运行时
			C++
				集C语言和Java的特点，又能跨平台，又可以帮助我们理解计算机的本质
				内容：对象、多态、继承
			Python
				面向编程逻辑，使用第三方库的语言
				根据我需要解决的问题，借助计算机来求解
			HTML/JS/CSS
				针对网络前端开发
		通过解释来执行的编程语言是脚本语言
			Python、JavaScript、PHP
		静态语言的优点是优化更充分
			因为编译器通过对整个程序的一次性翻译转化成机器语言
			而在这之中，我们可以进行优化，让程序运行速度更快
		脚本语言的优点是维护更灵活
			因为脚本语言是边用源代码边翻译，维护程序更灵活
			源代码可以跨多个操作系统运行
	程序的编写方法称作IPO（输入，进程，输出）
		输入包括程序内部参数的输入
		输出也包括程序内部参数输出的结果
		通用的处理方法简称为算法
		计算机只能解决计算问题，但是一个问题可以从不同角度来理解，从而产生不同的计算部分；而每个问题的计算部分都有从I到P再到O的过程
