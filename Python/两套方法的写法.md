`split` 不是一个独立的全局函数，它是**字符串对象自带的方法**。

所以要写成：ntxt.split()
而不是：split(ntxt)

你可以先把这两种写法理解成两套不同的语法。
	第一种是“函数”的语法：
	函数名(数据)
	例如：
		len(ntxt)  
		print(ntxt)  
		int("123")
	==**这里的 `len`、`print`、`int` 都是 Python 直接提供的函数名，所以它们可以直接写在前面。**==

第二种是“对象的方法”的语法：
	对象.方法名()
	例如：
	ntxt.split()  
	s.upper()  
	name.strip()
	==**这里的 `split`、`upper`、`strip` 都不是独立函数，而是字符串这个类型自带的功能，所以要通过“点”调用。**==
