在Python中，有许多内置的异常，当其中的某些错误出现时，它们会强制您的程序输出错误。

异常
	在 Python 中，异常是程序执行过程中出现的错误事件，它会打断程序的正常流程。
	例如，当你尝试除以零、访问不存在的文件或使用未定义的变量时，Python 会抛出相应的异常。

当发生这些异常时，==**它将导致当前进程停止并将其传递给调用进程，直到被处理为止**==。如果不处理，我们的程序将崩溃。
	例如，如果函数A调用函数B，后者又调用函数C，并且在函数C中发生异常。如果不在C中处理该异常，则该异常将传递给B，然后传递给A。

基本内容
	`try` 块：==**包含可能会抛出异常的代码**==。Python 会尝试执行 `try` 块中的代码。
	`except` 块：用于==捕获和处理 `try` 块中抛出的异常==。当 `try` 块中的代码抛出异常时，==程序会跳转到 `except` 块中执行相应的处理代码。==
	代码示例
		`try: # 可能会抛出异常的代码` 
			`result = 10 / 0 # 这里会抛出 ZeroDivisionError 异常`
		`except ZeroDivisionError: `
			`# 处理 ZeroDivisionError 异常的代码` 
			`print("除数不能为零！")`
