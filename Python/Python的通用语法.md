		②通过==缩进==表达代码的格式框架
			缩进可以表达所属关系，这形成了代码间的包含和层次关系

			==**单行注释以井号#开头，从#到这一行结束的所有内容都是注释。==**
			**==多行注释可以使用三个连续的单引号'''或三个连续的双引号"""来实现**==
	命名与保留字
		变量
			是程序中用于保存和表示数据的占位符
			给变量命名或关联标识符（关联两个变量）的过程叫命名
			用“＝”向变量赋值或修改当前值
			变量命名规则
				Aa，12，1_2,这是门Python_好课程
				即大小写字母、汉字、下划线、数字及其组合都可以命名变量
				注意：==①大小写敏感；②首字符不能是数字；③不能和保留字相符==
			变量的首字符不能是数字的原因
				编程语言在“读代码”时，通常先要把一串字符切分成不同类别：数字、变量名、运算符、关键字。
				这个过程叫词法分析。
				大多数语言规定“变量名不能以数字开头”，本质上是为了让这一步的判定简单、稳定
				你可以直接看一个例子：
				123abc
				编译器看到这一串时，如果变量允许数字开头，那它就会困惑：
				==它到底是一个变量名 123abc==
				==还是数字 123 后面跟着标识符 abc==
		保留字
			被编程语言内部定义并保留使用的标识符
			例如if,elif,else,in(一共33个)
			==**保留字也是大小写敏感的**==，if是保留字，但是If可以用做变量
			又例如True、False、None，这三个保留字是首字母大写的
			Python 3.x有35个关键字，分别为：and，as,
			==assert==, ==async==, ==await==, break, class,
			continue, def, del, elif, else, ==except==,
			==**F**==alse, ==finally==, for, from, ==global==, if,
			import, in, ==is==, ==lambda==, ==**N**==one, ==nonlocal==,
			not, or, pass, raise, return, ==**T**==rue, ==try==,
			while, ==with==, ==yield==。
			在 if 里，else 表示“条件不成立时执行”
			在循环里，else 表示“循环没有被 break 打断，而是正常结束时执行”
	数据类型
		一个信息，有好几种从不同角度中解释的方法
		我们需要让机器识别到，就要区分好数据类型
		那么：程序设计语言通过一定方式向计算机表达数据的形式就是数据类型
			例如用"123"表示123这个文本字符串，123则表示数字的123
		数据类型有
			数字类型
				整数类型
					包括正负数
				浮点数类型（相当于数学中的实数）
			字符串类型
				用"或'表示的一段字符的有序序列
				既然是有序的，那么我们可以对其中的字符进行索引
					例如"这是门Python好课程"，的“是”是该字符串的第1个字符
					注意，在计算机中，基本上都是从0开始编号
				字符串的序号
					正向递增序号
					反向递减序号（反向递减序号从-1开始）
					![[25bf7738b8cbc4c76ec8db27f19eaca8.jpg]]
				字符串的使用（使用[]来获取字符串中的一个或多个字符）
					索引
						返回字符串中的单个字符
						格式
							"字符串"[对应序号]，或变量名[对应序号]
					切片
						返回字符串中一段字符串的子串
						格式
							"字符串"[第一个对应序号:最后一个对应序号之前的一个字符]，或变量名[第一个对应序号:最后一个对应序号之前的一个字符]
							注意：一定是从第一个对应序号到最后一个对应序号之前的一个字符
							例如:“请输入带有符号的温度值：”[1:3]返回的是序号为1到2的字符
							又例如：TempStr[0:-1]从序号0到最后一个字符之前的一个字符
			列表类型
				由0个或多个数据组成的有序序列
				列表中用","来分隔各元素
				例如["F","f"]表示两个元素："F","f"
				使用保留字in判断一个元素是否在列表中
					例如：TempStr[-1]in["C","c"]来判断前者是否与列表中某个元素相同
					也就是TempStr最后一个字符是否是"C"或者"c"
					如果有，那么就返回True，没有就返回False
	语句与函数
		赋值符号是"="
		赋值语句
			由赋值符号构成的一段代码
			①其作用是用来==**给变量赋予新的数据值**==
			例如C = (eval(TempStr[0:-1])-32)/1.8 ，这个赋值语句的意思就是将右侧的运算结果赋给变量C
			②赋值语句的==**右侧的数据类型同时一并赋给变量**==
				例如：TempStr=input("")
				这段代码的意思是input返回一个字符串，TempStr顺便也变成了字符串类型
		分支语句
			由判断条件决定程序运行哪些代码的语句
			一般我们使用==if elif else构成条件判断的分支结构==
			例如if TempStr[-1] in ["F","f"]:
				如果条件成立，那么执行冒号后缩进的语句
				若不成立，那么冒号后缩进的语句将被跳过
			其中“：”是语法的一部分
			冒号以及后续缩进用来表示后续语句与条件的所属关系
		函数
			根据输入参数产生不同输出的功能过程
			例如input(),eval(),print()
			就像数学中的函数一样，也是根据括号中的内容返回一定的结果
			格式
				函数名(参数)，例如eval(TempStr[0:-1])，其中TempStr[0:-1]为参数
	输入与输出
		输入是程序获得外部信息的一个过程
		input()函数
			从控制台获得用户输入的函数
			格式
				变量 ＝ input(提示信息字符串)
				这样，我们就可以==将输入的信息以字符串类型保存在变量中==
		print()函数
			以字符形式向控制台输出结果的函数
			格式
				print(要输出的字符串或字符串变量)
			print函数的格式化
				例如print("转换后的温度是{:.2f}C".format(C))
				其中{:.2f}就是print的格式化方法
				"{}"表示槽，后续变量填充到槽中
				在这里，{:.2f}表示==**将变量C填充到这个位置时，取小数点后2位**==
		eval()函数（评估函数）
			去掉参数最外侧引号并执行余下语句的函数
			例如eval("1") => 1
			eval("1+3")=> 4
			eval(' "1+2" ') => '1+2'(注意，""和''作用完全一致，因此直接可以用单引号表示)
			eval (TempStr[0:-1])
				如果TempStr输出的值是“12.3”，那么通过eval输出的是12.3
			a ='100'
			print(eval(a+"1+2"))
				首先字符串'100'+"1+2"=1001+2'，然后将字符串'1001+2'通过eval()函数转化得到1001+2=1003

补充点：
	注释可以在一行的任意位置开始，但只有==**在#后的部分才会被作为注释不被执行;**==
	Python语言有严格的格式要求，==**不能随意缩进**==，只有在需要的时候才在代码前加空格;
	Python语言==**允许在一行的末尾加分号**==，但一般不建议这么做，会影响代码的美观度。
	在Python语言中，赋值不需要两侧数据类型一致，在赋值操作之后，**==左侧变量将会自动变成右侧对应的数据类型==**
