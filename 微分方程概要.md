微分方程
	学微分方程的意义
		==**从微积分的分析，到一个更大的框架，也就是将各种运算联系起来，这就是微分方程的功用**==
	凡含有未知函数的导数或微分的方程叫微分方程
		==**其实质在于联系自变量,未知函数以及未知函数的某些导数(或微分)之间的关系式**==
	分类：
		常微分方程：未知函数为一元函数的微分方程.
		偏微分方程：未知函数为多元函数的微分方程
	微分方程的阶数=微分方程中未知函数的最高阶导数的阶数

	微分方程的解
		代入微分方程能使方程成为恒等式的解，就是微分方程的解

		微分方程解的分类
			通解
				微分方程的解中含有任意常数C,
				且（相互独立的，如C1,C2）任意常数的个数与微分方程的阶数相同
					==**相互独立是指不能通过合并而使得通解中的任意常数的个数减少**==
					![[Pasted image 20250616211235.png]]
					其中第二个微分方程的C1,C2就是相互独立的
			特解
				确定了通解中任意常数C以后的解
				通常要确定某个特解，需要给出给特解应满足的附加条件，称之为定解条件
				一般来说，n阶微分方程，应有n个定解条件
				![[Pasted image 20250616211654.png]]
			普通解
				既非特解也非通解

				但是这个解确实是微分方程的解
				而这一题没有两个定解条件
				因此，这个解是一个普通解

	一阶线性微分方程
		可以化为：
		$$\begin{align*}
\frac{dy}{dx} + P(x)y= Q(x)形式,则原方程是一阶线性微分方程
\end{align*}$$
		当 Q(x) 恒等于0 ，则为一阶线性微分齐次方程
		否则就是一阶线性微分非齐次方程
		![[Pasted image 20250617141942.png]]
		上面的是齐次的
		下面的是非齐次的
		对于一阶线性齐次微分方程，我们可以通过分离变量进行求解：
			$$\begin{align*}
\frac{dy}{dx} + P(x)y = 0 \implies \frac{1}{y} dy = -P(x) dx\\
\int \frac{1}{y} dy = \int -P(x) dx 
\end{align*}$$
		对于齐次，我们可用公式法
		原方程不需要都移到等号一边：
		$$\begin{align*}
		原方程 ： \frac{dy}{dx} + P(x)y = Q(x)\\
y=  e^{-\int P(x)dx }\left[ \int Q(x)e^{\int P(x)dx }dx  + C\right],其中括号里面的是 u(x)
\end{align*}$$
	对于一个微分方程，先判断是多少阶，后判断是齐次还是非齐次
	对于一阶齐次，直接分离变量，对于一阶非齐次，用公式
	$$\begin{align*}
求 y' + \frac{1}{x}y = \frac{\sin x}{x} 的通解\\
这是一阶非齐次线性微分方程， 用公式 e^{-\int  \frac{1}{x} dx}\left[  \int \frac{\sin x}{x} e^{\int \frac{1}{x} dx}dx +C  \right] \\
原式 =  e^{-\ln x} \left[  \int \frac{\sin x}{x} e^{\ln x}dx +C  \right] = \frac{1}{x} \left[ \int \frac{\sin x}{x}*x dx+C \right] = \frac{1}{x} ( -\cos x +C )
\end {align*}$$
	$$\begin{align*}
求 \frac{dy}{dx} - \frac{y}{x} = x^2 的通解\\
一阶非齐次，用公式法： P(x) = - \frac{1}{x} ; Q(x) = x^2 \\
e^{-\int - \frac{1}{x} dx } \left[  \int x^2 e^{ -\int \frac{1}{x} dx } +C  \right] = x \left[  \int x dx +C  \right]= x\left(  \frac{1}{2} x^2 +C  \right) = \frac{1}{2} x^3 +Cx
\end{align*}$$

求解\frac{dy}{dx} = 2xy的通解

齐次方程
	 $$\begin{align*}
形如 \frac{dy}{dx} = f\left(  \frac{y}{x}  \right) 的微分方程称为齐次方程
\end{align*}$$
	其解法的核心在于用变量代换，化成可分离变量：
	$$\begin{align*}
设 u= \frac{y}{x} 即 y= x u,∴ \frac{dy}{dx} = u+ x \frac{du}{dx}\\
代入原式： u+ x \frac{du}{dx} = f(u),\\
即 \frac{du}{dx } = \frac{f(u)-u}{x} 
\end{align*}$$
	$$\begin{align*}
求\frac{dy}{dx} = \frac{y}{x} + \tan \frac{y}{x},满足条件：y|_{x=1 } =  \frac{\pi}{6} 的特解\\
设 u= \frac{y}{x} ， y= ux , \frac{dy}{dx} = \frac{du}{dx} x+ u\\
\frac{du}{dx} * x = \tan u \implies \int \cot udu =\int  \frac{1}{x} dx\\
\int \frac{1}{\sin u} du = \ln x+ C\\
\ln |\sin u|= \ln |x| + C \implies |\sin u| = Cx \implies C= \frac{1}{2}\\
最后回代 : \sin \frac{y}{x} = \frac{1}{2} x
\end{align*}$$
	例题2：
	$$\begin{align*}
y^2+ x^2 \frac{dy}{dx} = xy \frac{dy}{dx}\\
设 u= \frac{y}{x} ， y= ux , \frac{dy}{dx} = \frac{du}{dx} x+ u\\
\frac{u^2}{u-1} = \frac{dy}{dx } = \frac{du}{dx} x+ u \implies \frac{u}{u-1} = x \frac{du}{dx} \\ 
\left(  1- \frac{1}{u}  \right)du = \frac{1}{x} dx \implies u- \ln|u| +C = \ln|x| \\
最后 y= C e^{ \frac{y}{x}} 
\end{align*}$$
	例题3：$$\begin{align*}
x \frac{dy}{dx } - 4x \tan \frac{y}{x} -y =0\\
设 u= \frac{y}{x} ， y= ux , \frac{dy}{dx} = \frac{du}{dx} x+ u\\
\frac{dy}{dx} = \frac{y}{x} +  4\tan \frac{y}{x} \implies \frac{dy}{dx} = u + 4\tan u\\
4 \tan u= x \frac{du}{dx} \implies \frac{1}{4} \frac{1}{\tan u} du= \frac{1}{x} dx\\
\frac{1}{4} \ln |\sin u| = \ln|x| +C \implies Cx = \sin \left(  \frac{y}{x}  \right)^ {\frac{1}{4} }
\end{align*}$$
	例题4：
	$$\begin{align*}
求(xy + 3x^2)dy -y ^2 dx=0,满足 y|_{x=1}=3的特解\\
\frac{dy}{dx} = \frac{u^2}{u+3} = \frac{du}{dx} x+ u \implies - \frac{1}{3} \left(  1+ \frac{3}{u}  \right) du = \frac{1}{x} dx\\
- \frac{1}{3} u - \ln|u| = \ln|x| +C\\
解得：  Ce^{ -\frac{1}{3} * \frac{y}{x} } =y \implies C= 3e \\
3e^{ -\frac{1}{3} * \frac{y}{x} +1 } =y
\end{align*}$$
